# Task 3.1: Staging Deploy

## Goal

Deploy to staging and run data migration.

## Pre-Deploy

Must complete Task 2.5 (Local Testing) first!

## Steps

1. **Deploy to staging**:
   ```bash
   convex deploy
   # Deploy www to staging
   ```

2. **Run data migration**:
   ```bash
   convex run migrations:migrationCopyAuthData
   ```

3. **Verify migration**:
   - Check users copied
   - Check API keys copied
   - Check no errors

4. **Test auth flows**:
   - Registration
   - Login
   - Session persistence
   - API keys

## Verification

- [ ] Deployed to staging
- [ ] Migration run successfully
- [ ] All auth flows work
- [ ] No errors
