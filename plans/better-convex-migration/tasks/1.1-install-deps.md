# Task 1.1: Install better-convex and Update Dependencies

## Goal

Install better-convex and update auth.config.ts with JWKS.

## Steps

1. Install better-convex:
   ```bash
   cd packages/backend && bun add better-convex
   cd apps/www && bun add better-convex
   ```

2. Update `packages/backend/convex/auth.config.ts`:
   ```typescript
   import { getAuthConfigProvider } from "@convex-dev/better-auth/auth-config";
   import type { AuthConfig } from "convex/server";

   export default {
     providers: [getAuthConfigProvider({ jwks: process.env.JWKS })],
   } satisfies AuthConfig;
   ```

3. Generate JWKS:
   ```bash
   npx better-convex env sync --auth
   ```

4. Verify environment:
   ```bash
   convex env get JWKS
   ```

## Verification

- [ ] better-convex installed in both packages
- [ ] auth.config.ts updated
- [ ] JWKS environment variable set
- [ ] TypeScript compiles
