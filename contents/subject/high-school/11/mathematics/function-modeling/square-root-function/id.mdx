## Pengertian Fungsi Akar

Fungsi akar merupakan salah satu jenis fungsi yang melibatkan operasi akar kuadrat. Fungsi ini memiliki bentuk umum <InlineMath math="f(x) = \sqrt{g(x)}" /> di mana <InlineMath math="g(x)" /> adalah fungsi di dalam tanda akar.

Bentuk paling sederhana dari fungsi akar adalah <InlineMath math="f(x) = \sqrt{x}" />. Fungsi ini mengambil nilai input <InlineMath math="x" /> dan menghasilkan akar kuadrat dari nilai tersebut.

### Karakteristik Fungsi Akar

Fungsi akar memiliki beberapa karakteristik khusus yang membedakannya dari fungsi lainnya:

1. **Domain terbatas**: Karena akar kuadrat dari bilangan negatif tidak terdefinisi dalam bilangan real, domain fungsi akar terbatas pada nilai-nilai yang membuat ekspresi di dalam akar bernilai non-negatif.

2. **Grafik berbentuk kurva**: Grafik fungsi akar berbentuk kurva yang dimulai dari suatu titik dan terus naik dengan kecepatan yang semakin lambat.

3. **Nilai selalu non-negatif**: Hasil dari fungsi akar selalu bernilai non-negatif (≥ 0).

## Domain dan Range Fungsi Akar

Untuk memahami fungsi akar dengan baik, penting untuk menentukan domain dan range-nya.

### Menentukan Domain

Domain fungsi akar <InlineMath math="f(x) = \sqrt{g(x)}" /> adalah semua nilai <InlineMath math="x" /> yang membuat <InlineMath math="g(x) \geq 0" />.

**Langkah-langkah menentukan domain:**

| Langkah | Penjelasan                                            | Contoh: <InlineMath math="f(x) = \sqrt{x-2}" />                   |
| ------- | ----------------------------------------------------- | ----------------------------------------------------------------- |
| 1       | Identifikasi ekspresi dalam akar                      | <InlineMath math="g(x) = x-2" />                                  |
| 2       | Buat pertidaksamaan <InlineMath math="g(x) \geq 0" /> | <InlineMath math="x-2 \geq 0" />                                  |
| 3       | Selesaikan pertidaksamaan                             | <InlineMath math="x \geq 2" />                                    |
| 4       | Tuliskan domain                                       | <InlineMath math="D_f = \{x \mid x \geq 2, x \in \mathbb{R}\}" /> |

### Menentukan Range

Range fungsi akar adalah semua nilai output yang mungkin dihasilkan oleh fungsi tersebut.

Untuk fungsi <InlineMath math="f(x) = \sqrt{x-2}" />, karena akar kuadrat selalu menghasilkan nilai non-negatif, maka:

<BlockMath math="R_f = \{y \mid y \geq 0, y \in \mathbb{R}\}" />

### Translasi Vertikal

Fungsi <InlineMath math="f(x) = \sqrt{x} + k" /> menggeser grafik fungsi akar dasar sejauh <InlineMath math="k" /> satuan ke atas (jika <InlineMath math="k > 0" />) atau ke bawah (jika <InlineMath math="k < 0" />).

| Fungsi                                    | Transformasi            | Domain                         | Range                           |
| ----------------------------------------- | ----------------------- | ------------------------------ | ------------------------------- |
| <InlineMath math="f(x) = \sqrt{x}" />     | Fungsi dasar            | <InlineMath math="x \geq 0" /> | <InlineMath math="y \geq 0" />  |
| <InlineMath math="f(x) = \sqrt{x} + 2" /> | Geser 2 satuan ke atas  | <InlineMath math="x \geq 0" /> | <InlineMath math="y \geq 2" />  |
| <InlineMath math="f(x) = \sqrt{x} - 3" /> | Geser 3 satuan ke bawah | <InlineMath math="x \geq 0" /> | <InlineMath math="y \geq -3" /> |

## Fungsi Akar Bentuk Umum

Bentuk umum fungsi akar yang mengalami transformasi adalah:

<BlockMath math="f(x) = a\sqrt{b(x-h)} + k" />

Di mana:

- <InlineMath math="a" /> menentukan dilatasi vertikal dan refleksi (jika <InlineMath math="a < 0" />
  )
- <InlineMath math="b" /> menentukan dilatasi horizontal
- <InlineMath math="h" /> menentukan translasi horizontal
- <InlineMath math="k" /> menentukan translasi vertikal

**Langkah-langkah menggambar grafik:**

| Langkah | Tindakan            | Contoh: <InlineMath math="f(x) = 2\sqrt{x-1} + 3" />                                           |
| ------- | ------------------- | ---------------------------------------------------------------------------------------------- |
| 1       | Tentukan titik awal | <InlineMath math="x-1 = 0 \Rightarrow x = 1" />, <InlineMath math="f(1) = 3" />, Titik: (1, 3) |
| 2       | Tentukan domain     | <InlineMath math="x-1 \geq 0 \Rightarrow x \geq 1" />                                          |
| 3       | Buat tabel nilai    | Pilih beberapa nilai <InlineMath math="x \geq 1" />                                            |
| 4       | Hitung nilai fungsi | Untuk <InlineMath math="x = 2" />: <InlineMath math="f(2) = 2\sqrt{1} + 3 = 5" />              |
| 5       | Plot titik-titik    | Plot (1,3), (2,5), (5,7), dst.                                                                 |
| 6       | Hubungkan titik     | Buat kurva mulus melalui titik-titik                                                           |

## Persamaan Fungsi Akar

Untuk menyelesaikan persamaan yang melibatkan fungsi akar, ikuti langkah-langkah berikut:

| Langkah | Penjelasan            | Contoh: <InlineMath math="\sqrt{2x+3} = 5" />          |
| ------- | --------------------- | ------------------------------------------------------ |
| 1       | Isolasi akar          | Sudah terisolasi                                       |
| 2       | Kuadratkan kedua ruas | <InlineMath math="(\sqrt{2x+3})^2 = 5^2" />            |
| 3       | Sederhanakan          | <InlineMath math="2x+3 = 25" />                        |
| 4       | Selesaikan            | <InlineMath math="2x = 22 \Rightarrow x = 11" />       |
| 5       | Verifikasi            | <InlineMath math="\sqrt{2(11)+3} = \sqrt{25} = 5" /> ✓ |

## Pertidaksamaan Fungsi Akar

Untuk menyelesaikan pertidaksamaan fungsi akar, perhatikan domain dan sifat fungsi akar.

**Contoh:** Selesaikan <InlineMath math="\sqrt{x-1} < 3" />

<div className="flex flex-col gap-4">
  <BlockMath math="\text{Syarat: } x-1 \geq 0 \Rightarrow x \geq 1" />
  <BlockMath math="\sqrt{x-1} < 3" />
  <BlockMath math="x-1 < 9" />
  <BlockMath math="x < 10" />
</div>

Dengan menggabungkan syarat domain: <InlineMath math="1 \leq x < 10" />
