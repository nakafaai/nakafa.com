## Menggambar Grafik Fungsi Polinomial

Grafik fungsi polinomial memberikan gambaran visual tentang bagaimana nilai fungsi berubah seiring perubahan nilai input <InlineMath math="x" />. Bentuk grafik ini bisa sangat bervariasi tergantung pada derajat dan koefisien fungsinya.

## Metode Plotting Titik

Cara paling mendasar untuk menggambar grafik adalah dengan menentukan beberapa pasangan titik <InlineMath math="(x, y)" /> yang memenuhi fungsi tersebut, lalu menghubungkannya dengan kurva yang mulus.

**Langkah-langkah:**

1.  Pilih beberapa nilai <InlineMath math="x" /> yang berbeda.
2.  Hitung nilai <InlineMath math="y = P(x)" /> untuk setiap nilai <InlineMath math="x" /> yang dipilih.
3.  Buat tabel pasangan nilai <InlineMath math="(x, y)" />.
4.  Plot titik-titik tersebut pada bidang koordinat.
5.  Hubungkan titik-titik tersebut dengan kurva yang halus dan kontinu.

### Fungsi Linear (Derajat 1)

Gambarkan grafik fungsi <InlineMath math="f(x) = 2x + 5" />.

Kita pilih beberapa nilai <InlineMath math="x" /> dan hitung <InlineMath math="y" />:

| <InlineMath math="x" /> | <InlineMath math="y = f(x)" /> | <InlineMath math="(x, y)" />   |
| :---------------------- | :----------------------------- | :----------------------------- |
| -3                      | -1                             | <InlineMath math="(-3, -1)" /> |
| -2                      | 1                              | <InlineMath math="(-2, 1)" />  |
| -1                      | 3                              | <InlineMath math="(-1, 3)" />  |
| 0                       | 5                              | <InlineMath math="(0, 5)" />   |
| 1                       | 7                              | <InlineMath math="(1, 7)" />   |
| 2                       | 9                              | <InlineMath math="(2, 9)" />   |
| 3                       | 11                             | <InlineMath math="(3, 11)" />  |

Plot titik-titik dan hubungkan:

<LineEquation
  title={
    <>
      Grafik <InlineMath math="f(x) = 2x + 5" />
    </>
  }
  description="Grafik fungsi linear derajat 1."
  showZAxis={false}
  cameraPosition={[0, 0, 15]}
  data={[
    {
      points: Array.from({ length: 7 }, (_, i) => {
        const x = -3 + i;
        return { x, y: 2 * x + 5, z: 0 };
      }),
      color: getColor("YELLOW"),
    },
  ]}
/>

## Karakteristik Umum Grafik Polinomial

Grafik fungsi polinomial selalu **mulus** (tidak ada sudut tajam) dan **kontinu** (tidak ada lompatan atau jeda). Bentuk umumnya sangat dipengaruhi oleh **derajat** polinomialnya.

- **Derajat 0:** <InlineMath math="P(x) = c" />. Grafiknya berupa garis horizontal.
- **Derajat 1:** <InlineMath math="P(x) = ax + b" />. Grafiknya berupa garis lurus (miring).
- **Derajat 2:** <InlineMath math="P(x) = ax^2 + bx + c" />. Grafiknya berupa parabola.
- **Derajat 3:** <InlineMath math="P(x) = ax^3 + \dots" />. Grafiknya memiliki bentuk seperti huruf 'S' atau 'S' terbalik, bisa memiliki hingga dua 'puncak' atau 'lembah'.
- **Derajat 4:** Grafiknya bisa memiliki hingga tiga 'puncak' atau 'lembah'.
- **Derajat 5:** Grafiknya bisa memiliki hingga empat 'puncak' atau 'lembah'.

Secara umum, grafik fungsi polinomial berderajat <InlineMath math="n" /> dapat memotong sumbu <InlineMath math="x" /> **maksimal** <InlineMath math="n" /> kali dan memiliki **maksimal** <InlineMath math="n-1" /> titik balik (puncak atau lembah).

## Perilaku Ujung (End Behavior)

Salah satu karakteristik penting grafik fungsi polinomial adalah **perilaku ujung**-nya, yaitu arah grafik saat <InlineMath math="x" /> menuju tak hingga positif (<InlineMath math="x \to \infty" />) atau tak hingga negatif (<InlineMath math="x \to -\infty" />).

Perilaku ujung ini ditentukan **hanya** oleh **suku utama** <InlineMath math="a_n x^n" />:

1.  **Derajat <InlineMath math="n" /> (Genap atau Ganjil)**
2.  **Koefisien utama <InlineMath math="a_n" /> (Positif atau Negatif)**

Ada 4 kemungkinan kombinasi:

1.  **<InlineMath math="n" /> Genap, <InlineMath math="a_n > 0" /> (Positif):**

    - Saat <InlineMath math="x \to \infty" />, <InlineMath math="y \to \infty" /> (kanan atas <InlineMath math="\nearrow" />)
    - Saat <InlineMath math="x \to -\infty" />, <InlineMath math="y \to \infty" /> (kiri atas <InlineMath math="\nwarrow" />)
    - Contoh: <InlineMath math="y = x^2" />, <InlineMath math="y = x^4" />

      <LineEquation
        title={
          <>
            Grafik <InlineMath math="y = x^2" /> (Genap, Positif)
          </>
        }
        description="Grafik naik ke kiri dan ke kanan."
        showZAxis={false}
        cameraPosition={[0, 0, 15]}
        data={[
          {
            points: Array.from({ length: 7 }, (_, i) => {
              const x = -3 + i;
              return { x, y: x * x, z: 0 };
            }),
            color: getColor("LIME"),
          },
        ]}
      />

2.  **<InlineMath math="n" /> Genap, <InlineMath math="a_n < 0" /> (Negatif):**

    - Saat <InlineMath math="x \to \infty" />, <InlineMath math="y \to -\infty" /> (kanan bawah <InlineMath math="\searrow" />)
    - Saat <InlineMath math="x \to -\infty" />, <InlineMath math="y \to -\infty" /> (kiri bawah <InlineMath math="\swarrow" />)
    - Contoh: <InlineMath math="y = -x^2" />, <InlineMath math="y = -x^6" />

      <LineEquation
        title={
          <>
            Grafik <InlineMath math="y = -x^2" /> (Genap, Negatif)
          </>
        }
        description="Grafik turun ke kiri dan ke kanan."
        showZAxis={false}
        cameraPosition={[0, 0, 15]}
        data={[
          {
            points: Array.from({ length: 7 }, (_, i) => {
              const x = -3 + i;
              return { x, y: -(x * x), z: 0 };
            }),
            color: getColor("ROSE"),
          },
        ]}
      />

3.  **<InlineMath math="n" /> Ganjil, <InlineMath math="a_n > 0" /> (Positif):**

    - Saat <InlineMath math="x \to \infty" />, <InlineMath math="y \to \infty" /> (kanan atas <InlineMath math="\nearrow" />)
    - Saat <InlineMath math="x \to -\infty" />, <InlineMath math="y \to -\infty" /> (kiri bawah <InlineMath math="\swarrow" />)
    - Contoh: <InlineMath math="y = x" />, <InlineMath math="y = x^3" />, <InlineMath math="y = x^5" />

      <LineEquation
        title={
          <>
            Grafik <InlineMath math="y = x^3" /> (Ganjil, Positif)
          </>
        }
        description="Grafik turun ke kiri dan naik ke kanan."
        showZAxis={false}
        cameraPosition={[0, 0, 15]}
        data={[
          {
            points: Array.from({ length: 41 }, (_, i) => {
              const x = -3 + i * 0.15;
              return { x, y: x * x * x, z: 0 };
            }),
            color: getColor("SKY"),
            showPoints: false,
          },
        ]}
      />

4.  **<InlineMath math="n" /> Ganjil, <InlineMath math="a_n < 0" /> (Negatif):**

    - Saat <InlineMath math="x \to \infty" />, <InlineMath math="y \to -\infty" /> (kanan bawah <InlineMath math="\searrow" />)
    - Saat <InlineMath math="x \to -\infty" />, <InlineMath math="y \to \infty" /> (kiri atas <InlineMath math="\nwarrow" />)
    - Contoh: <InlineMath math="y = -x" />, <InlineMath math="y = -x^3" />

      <LineEquation
        title={
          <>
            Grafik <InlineMath math="y = -x^3" /> (Ganjil, Negatif)
          </>
        }
        description="Grafik naik ke kiri dan turun ke kanan."
        showZAxis={false}
        cameraPosition={[0, 0, 15]}
        data={[
          {
            points: Array.from({ length: 41 }, (_, i) => {
              const x = -3 + i * 0.15;
              return { x, y: -(x * x * x), z: 0 };
            }),
            color: getColor("AMBER"),
            showPoints: false,

### Menggunakan Perilaku Ujung

Mengetahui perilaku ujung sangat membantu dalam mengidentifikasi grafik fungsi polinomial tanpa harus menggambarnya secara detail.

**Contoh Aplikasi:**

Cocokkan fungsi berikut dengan kemungkinan perilaku ujungnya:

1. <InlineMath math="f(x) = x^4 + 2x^3 - 2x - 3" />

   - Suku utama: <InlineMath math="x^4" />
   - Derajat <InlineMath math="n=4" /> (Genap)
   - Koefisien utama <InlineMath math="a_n=1" /> (Positif)
   - Perilaku ujung: Kiri atas (<InlineMath math="\nwarrow" />), Kanan atas (<InlineMath math="\nearrow" />)

   <div className="mt-4">
     <LineEquation
       title={
         <>
           Grafik <InlineMath math="f(x) = x^4 + 2x^3 - 2x - 3" />
         </>
       }
       description={
         <>
           Perilaku Ujung: <InlineMath math="\nwarrow" />{" "}
           <InlineMath math="\nearrow" />
         </>
       }
       showZAxis={false}
       cameraPosition={[0, 0, 15]}
       data={[
         {
           points: Array.from({ length: 51 }, (_, i) => {
             const x = -3 + i * 0.12;
             const y = x ** 4 + 2 * x ** 3 - 2 * x - 3;
             return { x, y, z: 0 };
           }),
           color: getColor("TEAL"),
           showPoints: false,
         },
       ]}
     />
   </div>

2. <InlineMath math="g(x) = -x^3 + 2x^2 - x + 1" />

   - Suku utama: <InlineMath math="-x^3" />
   - Derajat <InlineMath math="n=3" /> (Ganjil)
   - Koefisien utama <InlineMath math="a_n=-1" /> (Negatif)
   - Perilaku ujung: Kiri atas (<InlineMath math="\nwarrow" />), Kanan bawah (<InlineMath math="\searrow" />)

   <div className="mt-4">
     <LineEquation

3. <InlineMath math="h(x) = -x^6 - \frac{11}{4}x^4 + x + 5x^3 + 2" />

   - Suku utama: <InlineMath math="-x^6" />

   - Koefisien utama <InlineMath math="a_n=-1" /> (Negatif)
