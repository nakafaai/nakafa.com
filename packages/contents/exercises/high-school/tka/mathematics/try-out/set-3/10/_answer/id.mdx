Suku banyak <InlineMath math="x^3 - 12x^2 + (p + 4)x - (p + 8) = 0" /> dengan <InlineMath math="a = 1" />, <InlineMath math="b = -12" />, <InlineMath math="c = p + 4" />, dan <InlineMath math="d = -(p + 8)" />. Akar-akar <InlineMath math="x_1, x_2, x_3" /> membentuk deret aritmatika dengan beda <InlineMath math="2" />. Maka <InlineMath math="x_1 = k" />, <InlineMath math="x_2 = k + 2" />, <InlineMath math="x_3 = k + 4" />.

Menentukan nilai <InlineMath math="k" /> dengan penjumlahan akar-akar

<MathContainer>
<BlockMath math="x_1 + x_2 + x_3 = -\frac{b}{a}" />

<BlockMath math="k + (k + 2) + (k + 4) = -\frac{(-12)}{1}" />

<BlockMath math="3k + 6 = 12" />

<BlockMath math="3k = 6" />

<BlockMath math="k = 2" />
</MathContainer>

Sehingga akar-akarnya adalah <InlineMath math="x_1 = 2" />, <InlineMath math="x_2 = 4" />, <InlineMath math="x_3 = 6" />. Kemudian menentukan nilai <InlineMath math="p" /> dari hasil kali akar-akarnya

<MathContainer>
<BlockMath math="x_1 \cdot x_2 \cdot x_3 = -\frac{d}{a}" />

<BlockMath math="2(4)(6) = -\frac{[-(p + 8)]}{1}" />

<BlockMath math="48 = p + 8" />

<BlockMath math="p = 40" />
</MathContainer>

Sehingga nilai dari <InlineMath math="p - 36 = 40 - 36 = 4" />.
