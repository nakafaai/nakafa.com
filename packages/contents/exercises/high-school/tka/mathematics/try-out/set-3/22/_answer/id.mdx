Definisi dari mutlak <InlineMath math="|x - 2|" /> adalah

<BlockMath math="|x - 2| = \begin{cases} x - 2, & \text{untuk } x \geq 2 \\ -(x - 2), & \text{untuk } x < 2 \end{cases}" />

Untuk memudahkan dalam menemukan penyelesaiannya, kita membagi menjadi 3 daerah berdasarkan definisi dari nilai mutlak di atas

<NumberLine
  title={<>Pembagian Daerah</>}
  description={<>Tiga daerah berdasarkan definisi nilai mutlak: <InlineMath math="x < 0" />, <InlineMath math="0 \leq x < 2" />, dan <InlineMath math="x \geq 2" />.</>}
  segments={[
    {
      start: Number.NEGATIVE_INFINITY,
      end: 0,
      endLabel: <InlineMath math="0" />,
      shaded: false,
      label: <InlineMath math="\text{Daerah I}" />,
    },
    {
      start: 0,
      end: 2,
      startLabel: <InlineMath math="0" />,
      endLabel: <InlineMath math="2" />,
      startInclusive: true,
      shaded: false,
      label: <InlineMath math="\text{Daerah II}" />,
    },
    {
      start: 2,
      end: Number.POSITIVE_INFINITY,
      startLabel: <InlineMath math="2" />,
      startInclusive: true,
      shaded: false,
      label: <InlineMath math="\text{Daerah III}" />,
    },
  ]}
/>

<InlineMath math="\text{Daerah I:}" /> <InlineMath math="(x < 0)" />

<MathContainer>
<BlockMath math="|x| = -x" />

<BlockMath math="|x - 2| = -(x - 2) = -x + 2" />

<BlockMath math="|x| + |x - 2| > 3" />

<BlockMath math="-x + (-x + 2) > 3" />

<BlockMath math="-2x + 2 > 3" />

<BlockMath math="-2x > 1" />

<BlockMath math="x < -\frac{1}{2}" />
</MathContainer>

Maka himpunan penyelesaiannya adalah <InlineMath math="\{x < 0\} \cap \{x < -\frac{1}{2}\} = \{x < -\frac{1}{2}\}" />.

<InlineMath math="\text{Daerah II:}" /> <InlineMath math="(0 \leq x < 2)" />

<MathContainer>
<BlockMath math="|x| = x" />

<BlockMath math="|x - 2| = -(x - 2) = -x + 2" />

<BlockMath math="|x| + |x - 2| > 3" />

<BlockMath math="x + (-x + 2) > 3" />

<BlockMath math="2 > 3" />
</MathContainer>

Artinya tidak ada nilai <InlineMath math="x" /> yang memenuhi daerah II.

<InlineMath math="\text{Daerah III:}" /> <InlineMath math="(x \geq 2)" />

<MathContainer>
<BlockMath math="|x| = x" />

<BlockMath math="|x - 2| = x - 2" />

<BlockMath math="|x| + |x - 2| > 3" />

<BlockMath math="x + (x - 2) > 3" />

<BlockMath math="2x > 5" />

<BlockMath math="x > \frac{5}{2}" />
</MathContainer>

Maka himpunan penyelesaiannya adalah <InlineMath math="\{x \geq 2\} \cap \{x > \frac{5}{2}\} = \{x > \frac{5}{2}\}" />.

Maka solusi totalnya adalah gabungan dari ketiga daerah

<BlockMath math="\{x < -\frac{1}{2}\} \cup \{x > \frac{5}{2}\}" />

Jadi solusinya adalah <InlineMath math="\{x < -\frac{1}{2} \cup x > \frac{5}{2}\}" />.
