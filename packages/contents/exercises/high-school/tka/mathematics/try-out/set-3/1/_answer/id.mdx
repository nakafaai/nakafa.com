Titik <InlineMath math="(a, b)" /> terletak pada kurva <InlineMath math="y = x^2 + 2" />. Substitusikan <InlineMath math="a" /> dan <InlineMath math="b" /> ke fungsi kurva sehingga menjadi

<BlockMath math="b = a^2 + 2" />

Titik menjadi <InlineMath math="(a, a^2 + 2)" />. Kemudian tentukan jarak titik <InlineMath math="(a, a^2 + 2)" /> ke garis <InlineMath math="y = x" />

<MathContainer>
<BlockMath math="\text{jarak} = \frac{|pm + qn + c|}{\sqrt{p^2 + q^2}}" />

<BlockMath math="f(a) = \frac{|a - (a^2 + 2)|}{\sqrt{1^2 + (-1)^2}}" />

<BlockMath math="f(a) = \frac{a^2 - a + 2}{\sqrt{2}}" />

<BlockMath math="f'(a) = \frac{1}{\sqrt{2}}(2a - 1)" />
</MathContainer>

Jarak terdekat adalah jarak minimum. Jarak minimumnya adalah

<MathContainer>
<BlockMath math="f'(a) = 0" />

<BlockMath math="\frac{1}{\sqrt{2}}(2a - 1) = 0" />

<BlockMath math="a = \frac{1}{2}" />
</MathContainer>

Jarak minimum saat <InlineMath math="a = \frac{1}{2}" />, substitusikan sehingga didapat nilai <InlineMath math="b" />

<MathContainer>
<BlockMath math="b = a^2 + 2" />

<BlockMath math="b = \left(\frac{1}{2}\right)^2 + 2" />

<BlockMath math="b = \frac{9}{4}" />
</MathContainer>

Penjumlahan <InlineMath math="a + b" />

<BlockMath math="\frac{1}{2} + \frac{9}{4} = \frac{11}{4} = 2\frac{3}{4}" />
