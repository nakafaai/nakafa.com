Saat dibagi <InlineMath math="(x - 1)" /> sisanya <InlineMath math="3" />, artinya

<BlockMath math="f(1) = 3" />

Saat dibagi <InlineMath math="(x - 2)" /> sisanya <InlineMath math="4" />, artinya

<BlockMath math="f(2) = 4" />

Fungsi <InlineMath math="f(x)" /> dibagi <InlineMath math="x^2 - 3x + 2 = (x - 1)(x - 2)" /> bersisa <InlineMath math="f(1)" /> dan <InlineMath math="f(2)" />. Kita bisa asumsikan misalnya sisa <InlineMath math="s(x) = ax + b" />.

Kita bisa menyusun persamaan

Saat sisa <InlineMath math="= f(1)" />

<MathContainer>
<BlockMath math="s(1) = f(1)" />

<BlockMath math="a(1) + b = 3" />

<BlockMath math="a + b = 3 \quad \ldots (1)" />
</MathContainer>

Saat sisa <InlineMath math="= f(2)" />

<MathContainer>
<BlockMath math="s(2) = f(2)" />

<BlockMath math="a(2) + b = 4" />

<BlockMath math="2a + b = 4 \quad \ldots (2)" />
</MathContainer>

Eliminasi kedua persamaan

<BlockMath math="\begin{aligned}
2a + b &= 4 \quad \ldots (2) \\
a + b &= 3 \quad \ldots (1) \\
\hline
(2) - (1): \quad (2a + b) - (a + b) &= 4 - 3 \\
a &= 1
\end{aligned}" />

Nilai <InlineMath math="a + b = 3" />, maka <InlineMath math="b = 3 - a = 3 - 1 = 2" />.

Sehingga sisanya adalah

<MathContainer>
<BlockMath math="s(x) = ax + b" />

<BlockMath math="s(x) = x + 2" />
</MathContainer>
