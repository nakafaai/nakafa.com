Kita ingat konsep persamaan kuadrat. Untuk persamaan kuadrat <InlineMath math="ax^2 + bx + c = 0" /> dengan akar-akar <InlineMath math="x_1" /> dan <InlineMath math="x_2" />, berlaku

<MathContainer>
<BlockMath math="x_1 + x_2 = -\frac{b}{a}" />

<BlockMath math="x_1 \cdot x_2 = \frac{c}{a}" />
</MathContainer>

Kita misalkan akar-akar adalah <InlineMath math="a" /> dan <InlineMath math="b" />. Karena salah satu akarnya tiga kali dari akar yang lain, maka akar-akar adalah <InlineMath math="a" /> dan <InlineMath math="3a" /> (karena <InlineMath math="b = 3a" />).

Penjumlahan akar-akar

<MathContainer>
<BlockMath math="a + 3a = -\frac{-4(k + 1)}{1}" />

<BlockMath math="4a = 4(k + 1)" />

<BlockMath math="a = k + 1 \quad \ldots (1)" />
</MathContainer>

Perkalian akar-akar

<MathContainer>
<BlockMath math="a \cdot 3a = \frac{k^2 - k + 7}{1}" />

<BlockMath math="3a^2 = k^2 - k + 7 \quad \ldots (2)" />
</MathContainer>

Substitusi persamaan (1) ke (2)

<MathContainer>
<BlockMath math="3(k + 1)^2 = k^2 - k + 7" />

<BlockMath math="3(k^2 + 2k + 1) = k^2 - k + 7" />

<BlockMath math="3k^2 + 6k + 3 = k^2 - k + 7" />

<BlockMath math="3k^2 + 6k + 3 - k^2 + k - 7 = 0" />

<BlockMath math="2k^2 + 7k - 4 = 0" />

<BlockMath math="(k + 4)(2k - 1) = 0" />
</MathContainer>

Maka <InlineMath math="k = -4" /> atau <InlineMath math="k = \frac{1}{2}" />.

Sekarang kita cek apakah kedua nilai <InlineMath math="k" /> memenuhi syarat semua akar lebih dari <InlineMath math="2" />.

Untuk <InlineMath math="k = -4" />

<BlockMath math="a = k + 1 = -4 + 1 = -3" />

Karena <InlineMath math="a = -3 < 2" />, maka <InlineMath math="k = -4" /> tidak memenuhi syarat.

Untuk <InlineMath math="k = \frac{1}{2}" />

<BlockMath math="a = k + 1 = \frac{1}{2} + 1 = \frac{3}{2}" />

Karena <InlineMath math="a = \frac{3}{2} < 2" />, maka <InlineMath math="k = \frac{1}{2}" /> juga tidak memenuhi syarat.

Kedua nilai <InlineMath math="k" /> tidak memenuhi syarat karena akarnya <InlineMath math="a = k + 1" /> nilainya kurang dari <InlineMath math="2" />. Jadi tidak ada nilai yang memenuhi.
