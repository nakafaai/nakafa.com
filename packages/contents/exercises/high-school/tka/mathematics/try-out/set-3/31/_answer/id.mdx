Suku banyak pertama memiliki pembagi <InlineMath math="x^2 + x - 2" />. Faktorkan pembaginya

<BlockMath math="x^2 + x - 2 = (x - 1)(x + 2) \Rightarrow x = 1 \text{ atau } x = -2" />

Sisanya adalah <InlineMath math="s(x) = ax + b" />.

Saat <InlineMath math="x = 1" />

<MathContainer>
<BlockMath math="f(1) = s(1)" />

<BlockMath math="f(1) = a(1) + b" />

<BlockMath math="f(1) = a + b \quad \ldots (1)" />
</MathContainer>

Saat <InlineMath math="x = -2" />

<MathContainer>
<BlockMath math="f(-2) = s(-2)" />

<BlockMath math="f(-2) = a(-2) + b" />

<BlockMath math="f(-2) = -2a + b \quad \ldots (2)" />
</MathContainer>

Suku banyak kedua memiliki pembagi <InlineMath math="x^2 - 4x + 3" />. Faktorkan pembaginya

<BlockMath math="x^2 - 4x + 3 = (x - 1)(x - 3) \Rightarrow x = 1 \text{ atau } x = 3" />

Sisanya adalah <InlineMath math="s(x) = 2bx + a - 1" />.

Saat <InlineMath math="x = 1" />

<MathContainer>
<BlockMath math="f(1) = s(1)" />

<BlockMath math="f(1) = 2b(1) + a - 1" />

<BlockMath math="f(1) = a + 2b - 1 \quad \ldots (3)" />
</MathContainer>

Saat <InlineMath math="x = 3" />

<MathContainer>
<BlockMath math="f(3) = s(3)" />

<BlockMath math="f(3) = 2b(3) + a - 1" />

<BlockMath math="f(3) = 6b + a - 1 \quad \ldots (4)" />
</MathContainer>

Dari persamaan (1) dan (3) akan didapat

<MathContainer>
<BlockMath math="f(1) = f(1)" />

<BlockMath math="a + b = a + 2b - 1" />

<BlockMath math="b = 1" />
</MathContainer>

Dari persamaan (2) dan diketahui <InlineMath math="f(-2) = 7" />, maka

<MathContainer>
<BlockMath math="f(-2) = 7" />

<BlockMath math="-2a + b = 7 \quad \ldots (5)" />
</MathContainer>

Substitusikan <InlineMath math="b = 1" /> ke persamaan (5)

<MathContainer>
<BlockMath math="-2a + 1 = 7" />

<BlockMath math="-2a = 6" />

<BlockMath math="a = -3" />
</MathContainer>
