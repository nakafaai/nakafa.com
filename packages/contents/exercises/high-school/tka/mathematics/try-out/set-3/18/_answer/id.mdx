Dimisalkan <InlineMath math="a = \log_3 x" /> dan <InlineMath math="b = \log_2 y" />. Akan didapat persamaan pertama

<MathContainer>
<BlockMath math="\log_3 x + \log_4 y^2 = 5" />

<BlockMath math="\log_3 x + \frac{2}{2} \log_2 y = 5" />

<BlockMath math="\log_3 x + \log_2 y = 5" />

<BlockMath math="a + b = 5" />

<BlockMath math="a = 5 - b \quad ...(1)" />

Bentuk <InlineMath math="\log_3 x \cdot \log_2 y = a \cdot b" />

<MathContainer>
<BlockMath math="ab = (5 - b)b" />

<BlockMath math="f(b) = 5b - b^2" />
</MathContainer>

Artinya nilai maksimum <InlineMath math="\log_3 x \cdot \log_2 y" /> sama dengan nilai maksimum dari fungsi <InlineMath math="f(b) = 5b - b^2" />. Syarat maksimum fungsi adalah <InlineMath math="f'(b) = 0" />

<MathContainer>
<BlockMath math="f'(b) = 0" />

<BlockMath math="5 - 2b = 0" />

<BlockMath math="2b = 5" />

<BlockMath math="b = \frac{5}{2}" />
</MathContainer>

Dengan begitu nilai <InlineMath math="a" />

<MathContainer>
<BlockMath math="a = 5 - b" />

<BlockMath math="a = 5 - \frac{5}{2}" />

<BlockMath math="a = \frac{10}{2} - \frac{5}{2}" />

<BlockMath math="a = \frac{5}{2}" />
</MathContainer>

Maka nilai <InlineMath math="ab" /> akan maksimum saat <InlineMath math="a = \frac{5}{2}" /> dan <InlineMath math="b = \frac{5}{2}" />. Hasil kalinya adalah

<BlockMath math="ab = \frac{5}{2} \cdot \frac{5}{2} = \frac{25}{4}" />
