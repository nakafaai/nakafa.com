Definisi dari mutlak <InlineMath math="|x + 1|" /> adalah

<BlockMath math="|x + 1| = \begin{cases} x + 1, & \text{untuk } x + 1 \geq 0 \text{ atau } x \geq -1 \\ -(x + 1), & \text{untuk } x + 1 < 0 \text{ atau } x < -1 \end{cases}" />

Untuk <InlineMath math="x \geq -1" />

<MathContainer>
<BlockMath math="|x + 1| = x + 1" />

<BlockMath math="|x + 1| > x + 3" />

<BlockMath math="x + 1 > x + 3" />

<BlockMath math="1 > 3" />
</MathContainer>

Tidak ada nilai <InlineMath math="x" /> yang memenuhi untuk <InlineMath math="x \geq -1" />.

Untuk <InlineMath math="x < -1" />

<MathContainer>
<BlockMath math="|x + 1| = -(x + 1)" />

<BlockMath math="|x + 1| > x + 3" />

<BlockMath math="-(x + 1) > x + 3" />

<BlockMath math="-x - 1 > x + 3" />

<BlockMath math="-2x > 4" />

<BlockMath math="x < -2" />
</MathContainer>

Nilai <InlineMath math="x" /> yang memenuhi adalah <InlineMath math="x < -2" />. Sehingga himpunan penyelesaiannya adalah <InlineMath math="\{x < -2\}" />.

Selesaikan bentuk berikutnya <InlineMath math="|x + 2| < 3" />

<MathContainer>
<BlockMath math="|x + 2| < 3" />

<BlockMath math="-3 < x + 2 < 3" />

<BlockMath math="-3 - 2 < x < 3 - 2" />

<BlockMath math="-5 < x < 1" />
</MathContainer>

Sehingga himpunan penyelesaiannya adalah <InlineMath math="\{-5 < x < 1\}" />.

Karena nilai <InlineMath math="x" /> haruslah memenuhi bentuk dari <InlineMath math="|x + 1| > x + 3" /> dan <InlineMath math="|x + 2| < 3" />, maka kita buat irisan dari keduanya yaitu

<BlockMath math="\{x < -2\} \cap \{-5 < x < 1\} = \{-5 < x < -2\}" />
