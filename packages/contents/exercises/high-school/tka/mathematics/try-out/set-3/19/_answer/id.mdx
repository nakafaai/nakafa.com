Pertidaksamaan logaritma memiliki solusi khusus untuk syaratnya yaitu

<MathContainer>
<BlockMath math="2x - 1 > 0 \rightarrow x > \frac{1}{2}" />

<BlockMath math="2 - x > 0 \rightarrow x < 2" />

<BlockMath math="x > 0" />
</MathContainer>

Solusi khususnya adalah <InlineMath math="\{\frac{1}{2} < x < 2\}" />. Solusi umum dari pertidaksamannya adalah

<MathContainer>
<BlockMath math="\log_{\frac{1}{2}}(2x - 1) + \log_{\frac{1}{2}}(2 - x) \geq 2 \log_{\frac{1}{2}} x" />

<BlockMath math="\log_{\frac{1}{2}}((2x - 1)(2 - x)) \geq \log_{\frac{1}{2}} x^2" />

<BlockMath math="\log_{\frac{1}{2}}(-2x^2 + 5x - 2) \geq \log_{\frac{1}{2}} x^2" />

<BlockMath math="-2x^2 + 5x - 2 \leq x^2" />

<BlockMath math="-3x^2 + 5x - 2 \leq 0" />
</MathContainer>

Dikali <InlineMath math="-1" /> maka tanda dibalik

<MathContainer>
<BlockMath math="3x^2 - 5x + 2 \geq 0" />

<BlockMath math="(3x - 2)(x - 1) \geq 0" />

<BlockMath math="x = \frac{2}{3} \cup x = 1" />
</MathContainer>
