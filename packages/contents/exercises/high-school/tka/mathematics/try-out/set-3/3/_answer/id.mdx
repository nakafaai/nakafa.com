Garis singgung memiliki persamaan umum <InlineMath math="y = mx + c" /> yang melalui titik <InlineMath math="(-8, 0)" />. Substitusikan titiknya sehingga menjadi

<MathContainer>
<BlockMath math="y = mx + c" />

<BlockMath math="0 = -8m + c" />

<BlockMath math="c = 8m" />
</MathContainer>

Dengan begitu garis singgungnya menjadi

<MathContainer>
<BlockMath math="y = mx + 8m" />

<BlockMath math="x = \frac{y - 8m}{m}" />
</MathContainer>

Ubah fungsi parabolanya menjadi bentuk <InlineMath math="x" />

<MathContainer>
<BlockMath math="y = \sqrt{x} + 1" />

<BlockMath math="\sqrt{x} = y - 1" />

<BlockMath math="x = (y - 1)^2" />
</MathContainer>

Substitusikan garis ke parabola

<MathContainer>
<BlockMath math="\frac{y - 8m}{m} = (y - 1)^2" />

<BlockMath math="\frac{y - 8m}{m} = y^2 - 2y + 1" />

<BlockMath math="y - 8m = my^2 - 2my + m" />

<BlockMath math="0 = my^2 - 2my - y + m + 8m" />

<BlockMath math="my^2 - (2m + 1)y + 9m = 0" />
</MathContainer>

Menjadi persamaan kuadrat dengan <InlineMath math="a = m" />, <InlineMath math="b = -(2m + 1)" />, <InlineMath math="c = 9m" />. Syarat menyinggung adalah <InlineMath math="D = 0" />

<MathContainer>
<BlockMath math="b^2 - 4ac = 0" />

<BlockMath math="\{-(2m + 1)\}^2 - 4m(9m) = 0" />

<BlockMath math="4m^2 + 4m + 1 - 36m^2 = 0" />

<BlockMath math="-32m^2 + 4m + 1 = 0" />

<BlockMath math="32m^2 - 4m - 1 = 0" />

<BlockMath math="(8m + 1)(4m - 1) = 0" />

<BlockMath math="m = -\frac{1}{8} \cup m = \frac{1}{4}" />
</MathContainer>

Karena bentuk dari <InlineMath math="y = \sqrt{x} + 1" /> selalu fungsi naik, maka <InlineMath math="m = \frac{1}{4}" /> yang memenuhi. Maka persamaan garis singgungnya

<MathContainer>
<BlockMath math="y = mx + 8m" />

<BlockMath math="y = \frac{1}{4}x + 8 \cdot \frac{1}{4}" />

<BlockMath math="y = \frac{1}{4}x + 2" />

<BlockMath math="4y = x + 8" />

<BlockMath math="4y - x - 8 = 0" />
</MathContainer>
