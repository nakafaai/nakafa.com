Let <InlineMath math="\alpha" /> and <InlineMath math="\beta" /> be the roots of the equation <InlineMath math="x^2 - bx + 6 = 0" />, so we get

<MathContainer>
<BlockMath math="\alpha + \beta = \frac{-(-b)}{1} = b" />

<BlockMath math="\alpha\beta = \frac{6}{1} = 6" />
</MathContainer>

Let <InlineMath math="p" /> and <InlineMath math="q" /> be the roots of the equation <InlineMath math="x^2 - 4x + c = 0" />, then

<MathContainer>
<BlockMath math="p = \alpha + \beta \quad \text{and} \quad q = \alpha - \beta" />

<BlockMath math="p + q = 4" />

<BlockMath math="(\alpha + \beta) + (\alpha - \beta) = 4" />

<BlockMath math="2\alpha = 4" />

<BlockMath math="\alpha = 2" />
</MathContainer>

Next, substitute the value <InlineMath math="\alpha = 2" /> into the first quadratic equation to obtain

<MathContainer>
<BlockMath math="x^2 - bx + 6 = 0" />

<BlockMath math="2^2 - b(2) + 6 = 0" />

<BlockMath math="4 - 2b + 6 = 0" />

<BlockMath math="-2b = -10" />

<BlockMath math="b = 5" />
</MathContainer>

So <InlineMath math="p = \alpha + \beta = b = 5" />, then substitute the value <InlineMath math="p = 5" /> into the second quadratic equation to obtain

<MathContainer>
<BlockMath math="x^2 - 4x + c = 0" />

<BlockMath math="5^2 - 4(5) + c = 0" />

<BlockMath math="25 - 20 + c = 0" />

<BlockMath math="c = -5" />
</MathContainer>

The quadratic equation that has roots <InlineMath math="b = 5" /> and <InlineMath math="c = -5" /> is

<MathContainer>
<BlockMath math="x^2 - (b + c)x + bc = 0" />

<BlockMath math="x^2 - (5 + (-5))x + 5(-5) = 0" />

<BlockMath math="x^2 - 0x - 25 = 0" />

<BlockMath math="x^2 - 25 = 0" />

<BlockMath math="x^2 - 5^2 = 0" />
</MathContainer>
