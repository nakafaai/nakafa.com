Use the following trigonometric identities

<MathContainer>
<BlockMath math="\sin 2x = 2 \sin x \cos x" />

<BlockMath math="\cos 2x = 2 \cos^2 x - 1" />
</MathContainer>

Then

<MathContainer>
<BlockMath math="\sin 2x + \cos 2x = -16 \cos x + 8 \sin x + \cos^2 x" />

<BlockMath math="2 \sin x \cos x + 2 \cos^2 x - 1 = -16 \cos x + 8 \sin x + \cos^2 x" />

<BlockMath math="2 \sin x \cos x + 2 \cos^2 x - 1 + 16 \cos x - 8 \sin x - \cos^2 x = 0" />

<BlockMath math="2 \sin x \cos x + \cos^2 x + 16 \cos x - 8 \sin x - 1 = 0" />

<BlockMath math="2 \cos x (\sin x + 8) - \sin^2 x - 8 \sin x = 0" />

<BlockMath math="2 \cos x (\sin x + 8) - \sin x (\sin x + 8) = 0" />

<BlockMath math="(2 \cos x - \sin x)(\sin x + 8) = 0" />
</MathContainer>

So <InlineMath math="2 \cos x - \sin x = 0" /> or <InlineMath math="\sin x + 8 = 0" />.

Since <InlineMath math="0 \leq x \leq \frac{\pi}{2}" />, then <InlineMath math="\sin x \geq 0" />, so <InlineMath math="\sin x + 8 \neq 0" />.

Therefore <InlineMath math="2 \cos x - \sin x = 0" />

<MathContainer>
<BlockMath math="\sin x = 2 \cos x" />

<BlockMath math="\frac{\sin x}{\cos x} = 2" />

<BlockMath math="\tan x = 2 = \frac{2}{1}" />
</MathContainer>

The opposite side is <InlineMath math="2" />, and the adjacent side is <InlineMath math="1" />. Then the hypotenuse is

<BlockMath math="\text{Hyp} = \sqrt{2^2 + 1^2} = \sqrt{4 + 1} = \sqrt{5}" />

Thus

<MathContainer>
<BlockMath math="\sin x = \frac{\text{opp}}{\text{hyp}} = \frac{2}{\sqrt{5}}" />

<BlockMath math="\cos x = \frac{\text{adj}}{\text{hyp}} = \frac{1}{\sqrt{5}}" />
</MathContainer>

Then the value of <InlineMath math="\sin 2x" />

<MathContainer>
<BlockMath math="\sin 2x = 2 \sin x \cos x" />

<BlockMath math="\sin 2x = 2 \cdot \frac{2}{\sqrt{5}} \cdot \frac{1}{\sqrt{5}}" />

<BlockMath math="\sin 2x = \frac{4}{5}" />
</MathContainer>
