The basic concept of polynomial division is

<BlockMath math="P(x) = g(x) \cdot h(x) + s(x)" />

where <InlineMath math="P(x)" /> is the polynomial being divided, <InlineMath math="g(x)" /> is the divisor, <InlineMath math="h(x)" /> is the quotient, and <InlineMath math="s(x)" /> is the remainder.

Thus

<MathContainer>
<BlockMath math="x^3 + 2x^2 + mx + n = (x^2 - 4x + 3)h(x) + (3x + 2)" />

<BlockMath math="x^3 + 2x^2 + mx + n = (x - 1)(x - 3)h(x) + (3x + 2)" />
</MathContainer>

where <InlineMath math="P(x) = x^3 + 2x^2 + mx + n" />, <InlineMath math="g(x) = (x - 1)(x - 3)" />, and <InlineMath math="s(x) = 3x + 2" />.

Substitute <InlineMath math="x = 1" />

<MathContainer>
<BlockMath math="P(1) = (1 - 1)(1 - 3)h(1) + (3(1) + 2)" />

<BlockMath math="1^3 + 2(1)^2 + m(1) + n = 0 + 5" />

<BlockMath math="1 + 2 + m + n = 5" />

<BlockMath math="m + n = 2 \quad \ldots (1)" />
</MathContainer>

Substitute <InlineMath math="x = 3" />

<MathContainer>
<BlockMath math="P(3) = (3 - 1)(3 - 3)h(3) + (3(3) + 2)" />

<BlockMath math="3^3 + 2(3)^2 + m(3) + n = 0 + 11" />

<BlockMath math="27 + 18 + 3m + n = 11" />

<BlockMath math="45 + 3m + n = 11" />

<BlockMath math="3m + n = -34 \quad \ldots (2)" />
</MathContainer>

Eliminate both equations

<BlockMath math="\begin{aligned}
3m + n &= -34 \quad \ldots (2) \\
m + n &= 2 \quad \ldots (1) \\
\hline
(2) - (1): \quad (3m + n) - (m + n) &= -34 - 2 \\
-2m &= 36 \\
m &= -18
\end{aligned}" />

Substitute into equation (1)

<MathContainer>
<BlockMath math="m + n = 2" />

<BlockMath math="-18 + n = 2" />

<BlockMath math="n = 20" />
</MathContainer>
