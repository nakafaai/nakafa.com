Misalkan <InlineMath math="\alpha" /> dan <InlineMath math="\beta" /> akar-akar persamaan <InlineMath math="x^2 - bx + 6 = 0" />, sehingga diperoleh

<MathContainer>
<BlockMath math="\alpha + \beta = \frac{-(-b)}{1} = b" />

<BlockMath math="\alpha\beta = \frac{6}{1} = 6" />
</MathContainer>

Misalkan <InlineMath math="p" /> dan <InlineMath math="q" /> akar-akar persamaan <InlineMath math="x^2 - 4x + c = 0" />, maka

<MathContainer>
<BlockMath math="p = \alpha + \beta \quad \text{dan} \quad q = \alpha - \beta" />

<BlockMath math="p + q = 4" />

<BlockMath math="(\alpha + \beta) + (\alpha - \beta) = 4" />

<BlockMath math="2\alpha = 4" />

<BlockMath math="\alpha = 2" />
</MathContainer>

Selanjutnya nilai <InlineMath math="\alpha = 2" /> disubstitusi ke persamaan kuadrat pertama diperoleh

<MathContainer>
<BlockMath math="x^2 - bx + 6 = 0" />

<BlockMath math="2^2 - b(2) + 6 = 0" />

<BlockMath math="4 - 2b + 6 = 0" />

<BlockMath math="-2b = -10" />

<BlockMath math="b = 5" />
</MathContainer>

Jadi <InlineMath math="p = \alpha + \beta = b = 5" />, selanjutnya nilai <InlineMath math="p = 5" /> disubstitusi ke persamaan kuadrat yang kedua diperoleh

<MathContainer>
<BlockMath math="x^2 - 4x + c = 0" />

<BlockMath math="5^2 - 4(5) + c = 0" />

<BlockMath math="25 - 20 + c = 0" />

<BlockMath math="c = -5" />
</MathContainer>

Persamaan kuadrat yang mempunyai akar-akar <InlineMath math="b = 5" /> dan <InlineMath math="c = -5" /> adalah

<MathContainer>
<BlockMath math="x^2 - (b + c)x + bc = 0" />

<BlockMath math="x^2 - (5 + (-5))x + 5(-5) = 0" />

<BlockMath math="x^2 - 0x - 25 = 0" />

<BlockMath math="x^2 - 25 = 0" />

<BlockMath math="x^2 - 5^2 = 0" />
</MathContainer>
