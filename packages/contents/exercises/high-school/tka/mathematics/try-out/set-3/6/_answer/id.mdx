Diketahui <InlineMath math="u_3 = a" />, <InlineMath math="u_4 = \frac{1}{a}" /> dan <InlineMath math="u_5 = \frac{1}{a^2 + 2a}" /> dengan <InlineMath math="a \neq 0" /> dan rasio <InlineMath math="r \neq 1" />. Maka nilai satu suku dengan suku lainnya yang berurutan tidak boleh sama.

<MathContainer>
<BlockMath math="\frac{u_4}{u_3} = \frac{u_5}{u_4}" />

<BlockMath math="u_4 \cdot u_4 = u_3 \cdot u_5" />

<BlockMath math="\frac{1}{a} \cdot \frac{1}{a} = a \cdot \frac{1}{a^2 + 2a}" />

<BlockMath math="\frac{1}{a^2} = \frac{a}{a(a + 2)}" />

<BlockMath math="\frac{1}{a^2} = \frac{1}{a + 2}" />

<BlockMath math="a^2 = a + 2" />

<BlockMath math="a^2 - a - 2 = 0" />
</MathContainer>

Faktorkan

<BlockMath math="(a + 1)(a - 2) = 0" />

<BlockMath math="a = -1 \cup a = 2" />

Untuk <InlineMath math="a = -1" />

<MathContainer>
<BlockMath math="u_3 = a = -1" />

<BlockMath math="u_4 = \frac{1}{a} = \frac{1}{-1} = -1" />
</MathContainer>

Karena <InlineMath math="u_3 = u_4" /> maka rasionya <InlineMath math="1" />. Sedangkan syarat <InlineMath math="r \neq 1" />. Maka <InlineMath math="a = -1" /> tidak memenuhi.

Untuk <InlineMath math="a = 2" />

<MathContainer>
<BlockMath math="u_3 = kr^2 = 2 \quad ...(1)" />

</MathContainer>
