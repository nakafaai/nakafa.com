Misalkan suku banyak <InlineMath math="x^3 - 7x^2 + px + q = 0" /> memiliki akar-akar <InlineMath math="x_1, x_2, x_3" />. Barisan yang dibentuk adalah geometri dengan rasio <InlineMath math="2" />. Maka bisa dimisalkan <InlineMath math="x_1 = k" />, <InlineMath math="x_2 = 2k" />, <InlineMath math="x_3 = 4k" />. Atau nilai <InlineMath math="a = k" /> dan <InlineMath math="r = 2" />. Jika diurutkan <InlineMath math="x_1 = a = k" />, <InlineMath math="x_2 = ar = 2k" />, <InlineMath math="x_3 = ar^2 = k \cdot 2^2 = 4k" />.

Suku banyak berturut-turut memiliki nilai <InlineMath math="a = 1" />, <InlineMath math="b = -7" />, <InlineMath math="c = p" />, <InlineMath math="d = q" />. Maka hasil penjumlahan akar-akar suku banyak

<BlockMath math="k + 2k + 4k = -\frac{(-7)}{1}" />

<BlockMath math="7k = 7" />

<BlockMath math="k = 1" />
</MathContainer>

Diperoleh nilai <InlineMath math="x_1 = k = 1" />. Substitusikan <InlineMath math="x = 1" /> ke persamaan suku banyak
