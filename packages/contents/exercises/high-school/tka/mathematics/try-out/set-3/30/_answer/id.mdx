Diketahui persamaan kuadrat <InlineMath math="(m + 2)x^2 - (2m - 1)x + m + 1 = 0" /> dengan syarat harus <InlineMath math="x_1 < 0" /> dan <InlineMath math="x_2 < 0" />.

Agar persamaan tersebut merupakan persamaan kuadrat, maka koefisien <InlineMath math="x^2" /> tidak boleh nol, yaitu <InlineMath math="m + 2 \neq 0" /> atau <InlineMath math="m \neq -2" />.

Syarat diskriminan adalah <InlineMath math="D \geq 0" />, maka diperoleh

<MathContainer>
<BlockMath math="D = b^2 - 4ac \geq 0" />

<BlockMath math="[-(2m - 1)]^2 - 4(m + 2)(m + 1) \geq 0" />

<BlockMath math="(2m - 1)^2 - 4(m + 2)(m + 1) \geq 0" />

<BlockMath math="4m^2 - 4m + 1 - 4(m^2 + 3m + 2) \geq 0" />

<BlockMath math="4m^2 - 4m + 1 - 4m^2 - 12m - 8 \geq 0" />

<BlockMath math="-16m - 7 \geq 0" />

<BlockMath math="-16m \geq 7" />

<BlockMath math="m \leq -\frac{7}{16}" />
</MathContainer>

Syarat penjumlahan akar-akar: <InlineMath math="x_1 + x_2 < 0" />

<MathContainer>
<BlockMath math="x_1 + x_2 = -\frac{b}{a} = -\frac{-(2m - 1)}{m + 2} = \frac{2m - 1}{m + 2}" />

<BlockMath math="\frac{2m - 1}{m + 2} < 0" />
</MathContainer>

Pertidaksamaan <InlineMath math="\frac{2m - 1}{m + 2} < 0" /> terpenuhi jika pembilang dan penyebut berbeda tanda. Maka

<BlockMath math="(2m - 1)(m + 2) < 0" />

Pertidaksamaan ini terpenuhi untuk <InlineMath math="-2 < m < \frac{1}{2}" />.

Syarat perkalian akar-akar: <InlineMath math="x_1x_2 > 0" />

<MathContainer>
<BlockMath math="x_1x_2 = \frac{c}{a} = \frac{m + 1}{m + 2}" />

<BlockMath math="\frac{m + 1}{m + 2} > 0" />
</MathContainer>

Pertidaksamaan <InlineMath math="\frac{m + 1}{m + 2} > 0" /> terpenuhi jika pembilang dan penyebut sama tanda. Maka

<BlockMath math="(m + 1)(m + 2) > 0" />

Pertidaksamaan ini terpenuhi untuk <InlineMath math="m < -2" /> atau <InlineMath math="m > -1" />.

Sekarang kita gabungkan semua syarat:
- <InlineMath math="m \neq -2" /> (agar persamaan kuadrat)
- <InlineMath math="m \leq -\frac{7}{16}" /> (dari diskriminan)
- <InlineMath math="-2 < m < \frac{1}{2}" /> (dari penjumlahan akar)
- <InlineMath math="m < -2" /> atau <InlineMath math="m > -1" /> (dari perkalian akar)

Irisan dari semua syarat tersebut adalah <InlineMath math="-1 < m \leq -\frac{7}{16}" />.

Jadi, nilai <InlineMath math="m" /> yang memenuhi adalah <InlineMath math="-1 < m \leq -\frac{7}{16}" />.
