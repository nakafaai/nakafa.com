Permisalan bahwa <InlineMath math="|x - 5| = p" />, kita dapat menyelesaikan pertidaksamaan menjadi

<MathContainer>
<BlockMath math="|x - 5|^2 - 3|x - 5| + 2 < 0" />

<BlockMath math="p^2 - 3p + 2 < 0" />

<BlockMath math="(p - 1)(p - 2) < 0" />

<BlockMath math="p = 1 \cup p = 2" />
</MathContainer>

Kembalikan ke bentuk asal

<MathContainer>
<BlockMath math="p = 1 \rightarrow |x - 5| = 1 \rightarrow x = 6 \cup x = 4" />

<BlockMath math="p = 2 \rightarrow |x - 5| = 2 \rightarrow x = 7 \cup x = 3" />
</MathContainer>
