Ingat konsep penjumlahan sinus

<BlockMath math="\sin(A + B) = \sin A \cos B + \cos A \sin B" />

Dengan demikian

<MathContainer>
<BlockMath math="3 \cos \theta - \sin \theta = r \sin(\theta + \alpha)" />

<BlockMath math="3 \cos \theta - \sin \theta = r(\sin \theta \cos \alpha + \cos \theta \sin \alpha)" />

<BlockMath math="3 \cos \theta - \sin \theta = r \sin \alpha \cos \theta + r \cos \alpha \sin \theta" />
</MathContainer>

Dengan membandingkan koefisien <InlineMath math="\cos \theta" /> dan <InlineMath math="\sin \theta" />, diperoleh

<MathContainer>
<BlockMath math="r \sin \alpha = 3 \quad \ldots \text{(1)}" />

<BlockMath math="r \cos \alpha = -1 \quad \ldots \text{(2)}" />
</MathContainer>

Kuadratkan kedua persamaan

<MathContainer>
<BlockMath math="r^2 \sin^2 \alpha = 9" />

<BlockMath math="r^2 \cos^2 \alpha = 1" />
</MathContainer>

Dengan menjumlahkan kedua persamaan di atas akan didapat

<MathContainer>
<BlockMath math="r^2 \sin^2 \alpha + r^2 \cos^2 \alpha = 9 + 1" />

<BlockMath math="r^2(\sin^2 \alpha + \cos^2 \alpha) = 10" />

<BlockMath math="r^2 \cdot 1 = 10" />

<BlockMath math="r^2 = 10" />

<BlockMath math="r = \sqrt{10}" />
</MathContainer>

Untuk menentukan kuadran <InlineMath math="\alpha" />, bagi persamaan <InlineMath math="\text{(1)}" /> dengan persamaan <InlineMath math="\text{(2)}" />

<BlockMath math="\frac{\text{(1)}}{\text{(2)}}: \quad \frac{r \sin \alpha}{r \cos \alpha} = \frac{3}{-1}" />

<MathContainer>
<BlockMath math="\tan \alpha = -3" />
</MathContainer>

Dari persamaan <InlineMath math="\text{(1)}" />: <InlineMath math="r \sin \alpha = 3" />. Karena <InlineMath math="r > 0" />, maka <InlineMath math="\sin \alpha > 0" />.

Dari persamaan <InlineMath math="\text{(2)}" />: <InlineMath math="r \cos \alpha = -1" />. Karena <InlineMath math="r > 0" />, maka <InlineMath math="\cos \alpha < 0" />.

Maka <InlineMath math="\alpha" /> berada di kuadran <InlineMath math="\text{II}" />.
