Bentuk umum suku ke-<InlineMath math="n" /> barisan aritmatika adalah <InlineMath math="u_n = a + (n - 1)b" />. Maka

<MathContainer>
<BlockMath math="u_1 \times u_2 = a \times (a + (2 - 1)b) = 10" />

<BlockMath math="a(a + b) = 10 \quad ...(1)" />
</MathContainer>

Dan juga

<MathContainer>
<BlockMath math="u_1 \times u_3 = a \times (a + (3 - 1)b)" />

<BlockMath math="a(a + 2b) = 16 \quad ...(2)" />
</MathContainer>

Bagi persamaan <InlineMath math="(1)" /> dan <InlineMath math="(2)" />

<MathContainer>
<BlockMath math="\frac{a(a + b)}{a(a + 2b)} = \frac{10}{16}" />

<BlockMath math="\frac{a + b}{a + 2b} = \frac{5}{8}" />

<BlockMath math="8(a + b) = 5(a + 2b)" />

<BlockMath math="8a + 8b = 5a + 10b" />

<BlockMath math="3a = 2b" />

<BlockMath math="b = \frac{3}{2}a" />
</MathContainer>

Substitusikan <InlineMath math="2b = 3a" /> ke persamaan <InlineMath math="(2)" />

<MathContainer>
<BlockMath math="a(a + 2b) = 16" />

<BlockMath math="a(a + 3a) = 16" />

<BlockMath math="a(4a) = 16" />

<BlockMath math="4a^2 = 16" />

<BlockMath math="a^2 = 4" />

<BlockMath math="a = \pm 2" />
</MathContainer>

Karena suku-sukunya positif maka <InlineMath math="a > 0" /> atau yang memenuhi adalah <InlineMath math="a = 2" />. Sehingga <InlineMath math="b = \frac{3}{2}(2) = 3" />. Maka suku ke-10 barisan adalah

<MathContainer>
<BlockMath math="u_{10} = a + (10 - 1)b" />

<BlockMath math="u_{10} = 2 + (9)(3)" />

<BlockMath math="u_{10} = 2 + 27" />

<BlockMath math="u_{10} = 29" />
</MathContainer>
