The sum of an infinite geometric series has a value of <InlineMath math="\frac{9}{4}" /> or <InlineMath math="S_\infty = \frac{9}{4}" />. With <InlineMath math="u_1 = a" /> and <InlineMath math="r = -\frac{1}{a}" />.

<MathContainer>
<BlockMath math="S_\infty = \frac{u_1}{1 - r}" />

<BlockMath math="\frac{a}{1 - \left(-\frac{1}{a}\right)} = \frac{9}{4}" />

<BlockMath math="\frac{a}{1 + \frac{1}{a}} = \frac{9}{4}" />

<BlockMath math="\frac{a}{\frac{a + 1}{a}} = \frac{9}{4}" />

<BlockMath math="a \cdot \frac{a}{a + 1} = \frac{9}{4}" />

<BlockMath math="\frac{a^2}{a + 1} = \frac{9}{4}" />

<BlockMath math="4a^2 = 9(a + 1)" />

<BlockMath math="4a^2 = 9a + 9" />

<BlockMath math="4a^2 - 9a - 9 = 0" />
</MathContainer>

Factorize

<BlockMath math="(4a + 3)(a - 3) = 0" />

<BlockMath math="a = -\frac{3}{4} \cup a = 3" />

Because the value of <InlineMath math="a > 0" />, the one that satisfies is <InlineMath math="a = 3" />. So <InlineMath math="r = -\frac{1}{a} = -\frac{1}{3}" />. Determine the value of <InlineMath math="3u_6 - u_5" />

<MathContainer>
<BlockMath math="u_6 = ar^5" />

<BlockMath math="u_5 = ar^4" />

<BlockMath math="= 3ar^5 - ar^4" />

<BlockMath math="= ar^4(3r - 1)" />

<BlockMath math="= 3\left(-\frac{1}{3}\right)^4 \left(3\left(-\frac{1}{3}\right) - 1\right)" />

<BlockMath math="= 3 \cdot \frac{1}{81} (-1 - 1)" />

<BlockMath math="= 3 \cdot \frac{1}{81} (-2)" />

<BlockMath math="= \frac{1}{27} (-2)" />

<BlockMath math="= -\frac{2}{27}" />
</MathContainer>
