Let <InlineMath math="a = \log_3 x" /> and <InlineMath math="b = \log_2 y" />. The first equation will be obtained

<MathContainer>
<BlockMath math="\log_3 x + \log_4 y^2 = 5" />

<BlockMath math="\log_3 x + \log_{2^2} y^2 = 5" />

<BlockMath math="\log_3 x + \frac{2}{2} \log_2 y = 5" />

<BlockMath math="\log_3 x + \log_2 y = 5" />

<BlockMath math="a + b = 5" />

<BlockMath math="a = 5 - b \quad ...(1)" />
</MathContainer>

Form <InlineMath math="\log_3 x \cdot \log_2 y = a \cdot b" />

<MathContainer>
<BlockMath math="ab = (5 - b)b" />

<BlockMath math="f(b) = 5b - b^2" />
</MathContainer>

This means the maximum value of <InlineMath math="\log_3 x \cdot \log_2 y" /> is equal to the maximum value of the function <InlineMath math="f(b) = 5b - b^2" />. The condition for the maximum of the function is <InlineMath math="f'(b) = 0" />

<MathContainer>
<BlockMath math="f'(b) = 0" />

<BlockMath math="5 - 2b = 0" />

<BlockMath math="2b = 5" />

<BlockMath math="b = \frac{5}{2}" />
</MathContainer>

Thus the value of <InlineMath math="a" />

<MathContainer>
<BlockMath math="a = 5 - b" />

<BlockMath math="a = 5 - \frac{5}{2}" />

<BlockMath math="a = \frac{10}{2} - \frac{5}{2}" />

<BlockMath math="a = \frac{5}{2}" />
</MathContainer>

So the value of <InlineMath math="ab" /> will be maximum when <InlineMath math="a = \frac{5}{2}" /> and <InlineMath math="b = \frac{5}{2}" />. The result of their product is

<BlockMath math="ab = \frac{5}{2} \cdot \frac{5}{2} = \frac{25}{4}" />
