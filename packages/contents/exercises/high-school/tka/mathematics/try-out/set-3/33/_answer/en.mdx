When divided by <InlineMath math="(x - 1)" /> the remainder is <InlineMath math="3" />, meaning

<BlockMath math="f(1) = 3" />

When divided by <InlineMath math="(x - 2)" /> the remainder is <InlineMath math="4" />, meaning

<BlockMath math="f(2) = 4" />

Function <InlineMath math="f(x)" /> divided by <InlineMath math="x^2 - 3x + 2 = (x - 1)(x - 2)" /> has remainders <InlineMath math="f(1)" /> and <InlineMath math="f(2)" />. We can assume, for example, the remainder <InlineMath math="s(x) = ax + b" />.

We can form equations

When remainder <InlineMath math="= f(1)" />

<MathContainer>
<BlockMath math="s(1) = f(1)" />

<BlockMath math="a(1) + b = 3" />

<BlockMath math="a + b = 3 \quad \ldots (1)" />
</MathContainer>

When remainder <InlineMath math="= f(2)" />

<MathContainer>
<BlockMath math="s(2) = f(2)" />

<BlockMath math="a(2) + b = 4" />

<BlockMath math="2a + b = 4 \quad \ldots (2)" />
</MathContainer>

Eliminate both equations

<BlockMath math="\begin{aligned}
2a + b &= 4 \quad \ldots (2) \\
a + b &= 3 \quad \ldots (1) \\
\hline
(2) - (1): \quad (2a + b) - (a + b) &= 4 - 3 \\
a &= 1
\end{aligned}" />

The value <InlineMath math="a + b = 3" />, then <InlineMath math="b = 3 - a = 3 - 1 = 2" />.

Thus the remainder is

<MathContainer>
<BlockMath math="s(x) = ax + b" />

<BlockMath math="s(x) = x + 2" />
</MathContainer>
