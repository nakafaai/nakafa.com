The condition for the inequality to be satisfied is

<MathContainer>
<BlockMath math="x^2 - x > 0" />

<BlockMath math="x(x - 1) > 0" />

<BlockMath math="x = 0 \cup x = 1" />
</MathContainer>

So the solution is <InlineMath math="\{x < 0 \cup x > 1\}" />. Whereas the general solution of the inequality is

<MathContainer>
<BlockMath math="\log_2(x^2 - x) \leq 1" />

<BlockMath math="\log_2(x^2 - x) \leq \log_2 2" />

<BlockMath math="x^2 - x \leq 2" />

<BlockMath math="x^2 - x - 2 \leq 0" />

<BlockMath math="(x + 1)(x - 2) \leq 0" />

<BlockMath math="x = -1 \cup x = 2" />
</MathContainer>

<BlockMath math="\{x < 0 \cup x > 1\} \cap \{-1 \leq x \leq 2\} = \{-1 \leq x < 0 \cup 1 < x \leq 2\}" />
