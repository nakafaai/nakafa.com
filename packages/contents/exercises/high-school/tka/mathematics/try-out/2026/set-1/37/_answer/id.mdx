Untuk menghitung modus dari data histogram, gunakan rumus modus untuk data berkelompok.

#### Histogram Data

<HistogramChart
  title="Histogram Distribusi Nilai"
  description="Distribusi frekuensi nilai siswa dengan kelas modal ditandai."
  yAxisLabel="Frekuensi"
  data={[
    { name: "49,5-54,5", value: 5 },
    { name: "54,5-59,5", value: 9 },
    { name: "59,5-64,5", value: 14 },
    { name: "64,5-69,5", value: 11 },

  chartConfig={{
    value: {
      label: "Frekuensi",

      },

  }}
/>

#### Menentukan Kelas Modus

Kelas modus adalah interval dengan frekuensi terbesar, yaitu interval <InlineMath math="59{,}5 - 64{,}5" /> dengan frekuensi <InlineMath math="14" />

#### Menghitung Selisih Frekuensi

Selisih frekuensi kelas modus dengan kelas sebelumnya

<BlockMath math="d_1 = 14 - 9 = 5" />

Selisih frekuensi kelas modus dengan kelas sesudahnya

<BlockMath math="d_2 = 14 - 11 = 3" />

#### Menentukan Parameter

Batas bawah kelas modus

<BlockMath math="T_b = 59{,}5" />

Panjang kelas

<BlockMath math="L = 64{,}5 - 59{,}5 = 5" />

#### Menghitung Modus

Gunakan rumus modus untuk data berkelompok

<MathContainer>
<BlockMath math="M_o = T_b + \left(\frac{d_1}{d_1 + d_2}\right) \cdot L" />
<BlockMath math="= 59{,}5 + \left(\frac{5}{5 + 3}\right) \cdot 5" />
<BlockMath math="= 59{,}5 + \left(\frac{5}{8}\right) \cdot 5" />
<BlockMath math="= 59{,}5 + 3{,}125" />
<BlockMath math="= 62{,}625 \approx 62{,}6" />
</MathContainer>

Jadi, modus dari data yang ditunjukkan pada histogram adalah <InlineMath math="62{,}6" />
