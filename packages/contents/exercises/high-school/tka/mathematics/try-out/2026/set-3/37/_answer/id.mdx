Diketahui bentuk pertidaksamaannya

<BlockMath math="\sin x + \sin 2x > \sin 3x" />

Kedua ruas dikurang <InlineMath math="\sin x" />

<BlockMath math="\sin 2x > \sin 3x - \sin x" />

Gunakan identitas trigonometri <InlineMath math="\sin 2x = 2 \sin x \cos x" /> dan <InlineMath math="\sin A - \sin B = 2 \cos \frac{A + B}{2} \sin \frac{A - B}{2}" />

<MathContainer>
<BlockMath math="2 \sin x \cos x > 2 \cos \frac{3x + x}{2} \sin \frac{3x - x}{2}" />
<BlockMath math="2 \sin x \cos x > 2 \cos 2x \sin x" />
</MathContainer>

Kedua ruas dibagi <InlineMath math="2" /> dan gunakan identitas <InlineMath math="\cos 2x = 2 \cos^2 x - 1" />

<MathContainer>
<BlockMath math="\sin x \cos x > \cos 2x \sin x" />
<BlockMath math="\sin x \cos x > (2 \cos^2 x - 1) \sin x" />
<BlockMath math="\sin x \cos x - (2 \cos^2 x - 1) \sin x > 0" />
<BlockMath math="\sin x (\cos x - 2 \cos^2 x + 1) > 0" />
<BlockMath math="\sin x (2 \cos^2 x - \cos x - 1) < 0" />
</MathContainer>

Faktorkan <InlineMath math="2 \cos^2 x - \cos x - 1 = (2 \cos x + 1)(\cos x - 1)" />

<BlockMath math="\sin x (2 \cos x + 1)(\cos x - 1) < 0" />

Dengan begitu <InlineMath math="x" /> pembuat nolnya

<MathContainer>
<BlockMath math="\sin x = 0 \Rightarrow x = 0^\circ, 180^\circ, 360^\circ" />
<BlockMath math="2 \cos x + 1 = 0 \Rightarrow \cos x = -\frac{1}{2} \Rightarrow x = 120^\circ, 240^\circ" />
<BlockMath math="\cos x - 1 = 0 \Rightarrow \cos x = 1 \Rightarrow x = 0^\circ, 360^\circ" />
</MathContainer>

Titik-titik kritisnya adalah <InlineMath math="x = 0^\circ" />, <InlineMath math="x = 120^\circ" />, <InlineMath math="x = 180^\circ" />, <InlineMath math="x = 240^\circ" />, dan <InlineMath math="x = 360^\circ" />.
