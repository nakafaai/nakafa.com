Determine the derivative of the function

<MathContainer>
<BlockMath math="y = 4 \sin x \sin(x - 60^\circ)" />

<BlockMath math="y' = 4[\cos x \cdot \sin(x - 60^\circ) + \sin x \cdot \cos(x - 60^\circ)]" />
</MathContainer>

Use trigonometric identities <InlineMath math="\sin A \cos B + \cos A \sin B = \sin(A + B)" /> and <InlineMath math="\cos A \cos B - \sin A \sin B = \cos(A + B)" />

<MathContainer>
<BlockMath math="y' = 4[\sin(x - 60^\circ) \cos x + \cos(x - 60^\circ) \sin x]" />

<BlockMath math="y' = 4 \sin[(x - 60^\circ) + x]" />

<BlockMath math="y' = 4 \sin(2x - 60^\circ)" />
</MathContainer>

Or using the identity <InlineMath math="\sin A \sin B = \frac{1}{2}[\cos(A - B) - \cos(A + B)]" />

<MathContainer>
<BlockMath math="y = 4 \sin x \sin(x - 60^\circ)" />

<BlockMath math="y = 4 \cdot \frac{1}{2}[\cos(x - (x - 60^\circ)) - \cos(x + (x - 60^\circ))]" />

<BlockMath math="y = 2[\cos(60^\circ) - \cos(2x - 60^\circ)]" />

<BlockMath math="y = 2 \cos 60^\circ - 2 \cos(2x - 60^\circ)" />

<BlockMath math="y' = -2 \cdot (-\sin(2x - 60^\circ)) \cdot 2" />

<BlockMath math="y' = 4 \sin(2x - 60^\circ)" />
</MathContainer>

The maximum value occurs when <InlineMath math="y' = 0" /> or when <InlineMath math="\sin(2x - 60^\circ)" /> reaches its maximum value, which is <InlineMath math="\sin(2x - 60^\circ) = 1" />.

However, since <InlineMath math="y = 2 \cos 60^\circ - 2 \cos(2x - 60^\circ) = 1 - 2 \cos(2x - 60^\circ)" />, the maximum value occurs when <InlineMath math="\cos(2x - 60^\circ)" /> is minimum, which is <InlineMath math="\cos(2x - 60^\circ) = -1" />.

<MathContainer>

<BlockMath math="2x - 60^\circ = 180^\circ + k \cdot 360^\circ" />

<BlockMath math="2x = 240^\circ + k \cdot 360^\circ" />

<BlockMath math="x = 120^\circ + k \cdot 180^\circ" />
</MathContainer>

With <InlineMath math="k" /> an integer.
