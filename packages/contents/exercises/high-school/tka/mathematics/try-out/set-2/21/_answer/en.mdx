Solve the inequality

<BlockMath math="|x - 2a| \leq b - a" />

Add <InlineMath math="2a" /> to both sides

<MathContainer>
<BlockMath math="-b + a + 2a \leq x - 2a + 2a \leq b - a + 2a" />

<BlockMath math="-b + 3a \leq x \leq b + a" />
</MathContainer>

Therefore the solution is <InlineMath math="-b + 3a \leq x \leq b + a" />.
