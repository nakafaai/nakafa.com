Jika suatu fungsi <InlineMath math="y = \sqrt{x^2 - 7}" />, maka....

1. <InlineMath math="y = \frac{4}{3}x - \frac{7}{3}" /> merupakan persamaan garis singgung di <InlineMath math="x = 4" />
2. Kurva berbentuk lingkaran berpusat di <InlineMath math="(0,0)" />
3. Garis <InlineMath math="y = -\frac{3}{4}x + 6" /> memotong tegak lurus garis singgung di <InlineMath math="x = 4" />
4. <InlineMath math="y = \frac{4}{3}x - \frac{25}{3}" /> merupakan garis singgung kurva di <InlineMath math="(4, -3)" />
