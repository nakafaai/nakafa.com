Diketahui dua fungsi kuadrat

<MathContainer>
<BlockMath math="f(x) = (k - 1)x^2 + kx - 1, \quad k \neq 1" />

<BlockMath math="g(x) = (k - 2)x^2 + x + 2k, \quad k \neq 2" />
</MathContainer>

#### Mencari syarat kedua fungsi berpotongan

Eliminasi <InlineMath math="y" /> pada kedua fungsi tersebut dengan menyamakan <InlineMath math="f(x) = g(x)" />.

<MathContainer>
<BlockMath math="(k - 1)x^2 + kx - 1 = (k - 2)x^2 + x + 2k" />

<BlockMath math="x^2 + (k - 1)x - (2k + 1) = 0 \quad ...(1)" />
</MathContainer>

Syarat agar kedua fungsi berpotongan di dua titik yang berbeda adalah diskriminan harus positif, yaitu <InlineMath math="D > 0" />.

<MathContainer>
<BlockMath math="D > 0" />

<BlockMath math="(k - 1)^2 - 4(1)(-2k - 1) > 0" />

<BlockMath math="k^2 - 2k + 1 + 8k + 4 > 0" />

<BlockMath math="k^2 + 6k + 5 > 0" />

<BlockMath math="(k + 5)(k + 1) > 0" />
</MathContainer>

Dari pertidaksamaan <InlineMath math="(k + 5)(k + 1) > 0" />, diperoleh <InlineMath math="k < -5" /> atau <InlineMath math="k > -1" />.

Karena <InlineMath math="k \neq 1" /> dan <InlineMath math="k \neq 2" />, maka bilangan asli terkecil yang memenuhi adalah <InlineMath math="k = 3" />.

#### Substitusi nilai k

Substitusi nilai <InlineMath math="k = 3" /> ke persamaan pertama untuk mencari jumlah akar-akar <InlineMath math="x" />.

<MathContainer>
<BlockMath math="x^2 + (k - 1)x - (2k + 1) = 0" />

<BlockMath math="x^2 + (3 - 1)x - (2(3) + 1) = 0" />

<BlockMath math="x^2 + 2x - 7 = 0" />
</MathContainer>

Dari persamaan kuadrat tersebut, jumlah akar-akarnya adalah

<BlockMath math="x_1 + x_2 = -2" />

#### Mencari nilai y

Substitusi nilai <InlineMath math="k = 3" /> ke fungsi <InlineMath math="f(x)" /> dan <InlineMath math="g(x)" />.

Untuk <InlineMath math="f(x)" />

<MathContainer>
<BlockMath math="f(x) = (k - 1)x^2 + kx - 1" />

<BlockMath math="f(3) = (3 - 1)x^2 + 3x - 1" />

<BlockMath math="f(3) = 2x^2 + 3x - 1" />
</MathContainer>

Untuk <InlineMath math="g(x)" />

<MathContainer>
<BlockMath math="g(x) = (k - 2)x^2 + x + 2k" />

<BlockMath math="g(3) = (3 - 2)x^2 + x + 2(3)" />

<BlockMath math="g(3) = x^2 + x + 6" />
</MathContainer>

Kemudian cari nilai <InlineMath math="x" /> dengan menghilangkan <InlineMath math="x^2" />.

<MathContainer>
<BlockMath math="y = 2x^2 + 3x - 1 \quad |\times 1|" />

<BlockMath math="y = x^2 + x + 6 \quad |\times 2|" />

<BlockMath math="y = 2x^2 + 3x - 1" />

<BlockMath math="2y = 2x^2 + 2x + 12 \quad -" />

<BlockMath math="-y = x - 13" />

<BlockMath math="x = 13 - y" />
</MathContainer>

Substitusi <InlineMath math="x = 13 - y" /> ke <InlineMath math="g(x)" />.

<MathContainer>
<BlockMath math="y = x^2 + x + 6" />

<BlockMath math="y = (13 - y)^2 + (13 - y) + 6" />

<BlockMath math="y = 169 - 26y + y^2 + 13 - y + 6" />

<BlockMath math="y^2 - 28y + 176 = 0" />
</MathContainer>

Dari persamaan kuadrat tersebut, jumlah akar-akarnya adalah

<BlockMath math="y_1 + y_2 = 28" />

#### Membentuk persamaan kuadrat baru

Persamaan kuadrat baru yang akar-akarnya <InlineMath math="x_1 + x_2" /> dan <InlineMath math="y_1 + y_2" /> adalah

<MathContainer>
<BlockMath math="x^2 - ((x_1 + x_2) + (y_1 + y_2))x + ((x_1 x_2)(y_1 y_2)) = 0" />

<BlockMath math="x^2 - (-2 + 28)x + (-2)(28) = 0" />

<BlockMath math="x^2 - 26x - 56 = 0" />
</MathContainer>

Jadi, persamaan kuadrat yang akar-akarnya <InlineMath math="x_1 + x_2" /> dan <InlineMath math="y_1 + y_2" /> adalah <InlineMath math="x^2 - 26x - 56 = 0" />.
