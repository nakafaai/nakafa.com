Substitusikan titik <InlineMath math="(a, b)" /> ke <InlineMath math="(x, y)" /> pada persamaan

<BlockMath math="b = a^3 - 3a^2 - 9a \quad ...(1)" />

Buat turunan pertamanya

<MathContainer>
<BlockMath math="f(x) = 3x^3 - 3x^2 - 9x" />

<BlockMath math="f'(x) = 9x^2 - 6x - 9" />
</MathContainer>

Ingat konsepnya bahwa turunan pertama fungsi sama dengan nilai gradien garis

<BlockMath math="f'(a) = m" />

<MathContainer>
<BlockMath math="3a^2 - 6a - 9 = 15" />

<BlockMath math="3a^2 - 6a - 24 = 0" />

<BlockMath math="a^2 - 2a - 8 = 0" />

<BlockMath math="(a + 2)(a - 4) = 0" />

<BlockMath math="a = -2 \cup a = 4" />
</MathContainer>

Menentukan nilai <InlineMath math="b" /> dari persamaan <InlineMath math="(1)" /> dan <InlineMath math="a + b" />

<BlockMath math="b = a^3 - 3a^2 - 9a" />

Saat <InlineMath math="a = -2" />

<MathContainer>
<BlockMath math="b = (-2)^3 - 3(-2)^2 - 9(-2)" />

<BlockMath math="b = -8 - 12 + 18" />

<BlockMath math="b = -2" />

<BlockMath math="a + b = -2 + (-2) = -4" />
</MathContainer>

Saat <InlineMath math="a = 4" />

<MathContainer>
<BlockMath math="b = 4^3 - 3(4)^2 - 9(4)" />

<BlockMath math="b = -20" />

<BlockMath math="a + b = 4 + (-20) = -16" />
</MathContainer>

Mengikuti opsi yang ada maka jawabannya adalah <InlineMath math="-4" />.
