Use the substitution <InlineMath math="\log_2 x = a" /> and <InlineMath math="\log_2 y = b" />. Then transform the logarithm equation into

<MathContainer>
<BlockMath math="\log_2 x^2 - \log_2 y^2 = \log_2 16" />

<BlockMath math="\log_{2^2} x - \log_{2^2} y = \log_2 2^4" />

<BlockMath math="2a - 2b = \log_{4^2} 2" />
</MathContainer>

Convert to exponential form

<MathContainer>
<BlockMath math="2(a - b) = 4" />

<BlockMath math="a - b = 2 \quad ...(1)" />

<BlockMath math="a = b + 2" />
</MathContainer>

Substitute equation (1)

<MathContainer>
<BlockMath math="(\log_2 x)^2 - (\log_2 y)^2 = \log_2 256" />

<BlockMath math="(a)^2 - (b)^2 = \log_2 2^8" />

<BlockMath math="(a + b)(a - b) = \log_{8^2} 2" />

<BlockMath math="(a + b)(a - b) = \log_{8^2} 2" />

<BlockMath math="(a + b)(2) = 8" />

<BlockMath math="a + b = 4" />

<BlockMath math="(b + 2) + b = 4" />

<BlockMath math="2b = 2" />

<BlockMath math="b = 1" />
</MathContainer>

Therefore the result from equation (1) is <InlineMath math="a = b + 2 = 1 + 2 = 3" />.

Determine the value of <InlineMath math="\log_2 x^6 y^{-2}" />

<MathContainer>
<BlockMath math="\log_2 x^6 y^{-2} = \log_2 x^6 + \log_2 y^{-2}" />

<BlockMath math="\log_2 x^6 y^{-2} = 6(\log_2 x) + (-2) \log_2 y" />

<BlockMath math="\log_2 x^6 y^{-2} = 6a - 2b" />

<BlockMath math="\log_2 x^6 y^{-2} = 6(3) - 2 = 16" />
</MathContainer>
