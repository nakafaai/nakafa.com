Diketahui sistem persamaan

<MathContainer>
<BlockMath math="3x + 2y - z = 3 \quad ...(1)" />

<BlockMath math="2x + y - 3z = 4 \quad ...(2)" />

<BlockMath math="x - y + 2z = -1 \quad ...(3)" />
</MathContainer>

#### Eliminasi persamaan pertama dan kedua

Kalikan persamaan pertama dengan <InlineMath math="3" /> dan persamaan kedua dengan <InlineMath math="1" />, lalu kurangkan.

<MathContainer>
<BlockMath math="3x + 2y - z = 3 \quad |\times 3| \quad 9x + 6y - 3z = 9" />

<BlockMath math="2x + y - 3z = 4 \quad |\times 1| \quad 2x + y - 3z = 4 \quad -" />

<BlockMath math="7x + 5y = 5 \quad ...(4)" />
</MathContainer>

#### Eliminasi persamaan pertama dan ketiga

Kalikan persamaan pertama dengan <InlineMath math="2" /> dan persamaan ketiga dengan <InlineMath math="1" />, lalu kurangkan.

<MathContainer>
<BlockMath math="3x + 2y - z = 3 \quad |\times 2| \quad 6x + 4y - 2z = 6" />

<BlockMath math="x - y + 2z = -1 \quad |\times 1| \quad x - y + 2z = -1 \quad -" />

<BlockMath math="7x + 3y = 5 \quad ...(5)" />
</MathContainer>

#### Eliminasi persamaan keempat dan kelima

Kurangkan persamaan keempat dengan persamaan kelima untuk mencari nilai <InlineMath math="y" />.

<MathContainer>
<BlockMath math="7x + 5y = 5" />

<BlockMath math="7x + 3y = 5 \quad -" />

<BlockMath math="2y = 0" />

<BlockMath math="y = 0" />
</MathContainer>

#### Substitusi nilai y

Substitusikan <InlineMath math="y = 0" /> ke persamaan keempat untuk mencari nilai <InlineMath math="x" />.

<MathContainer>
<BlockMath math="7x + 5y = 5" />

<BlockMath math="7x + 5(0) = 5" />

<BlockMath math="7x = 5" />

<BlockMath math="x = \frac{5}{7}" />
</MathContainer>

#### Substitusi nilai x dan y

Substitusikan <InlineMath math="x = \frac{5}{7}" /> dan <InlineMath math="y = 0" /> ke persamaan ketiga untuk mencari nilai <InlineMath math="z" />.

<MathContainer>
<BlockMath math="x - y + 2z = -1" />

<BlockMath math="\frac{5}{7} - 0 + 2z = -1" />

<BlockMath math="2z = -\frac{12}{7}" />

<BlockMath math="z = -\frac{6}{7}" />
</MathContainer>

#### Hitung nilai yang ditanyakan

Substitusikan nilai <InlineMath math="x = \frac{5}{7}" />, <InlineMath math="y = 0" />, dan <InlineMath math="z = -\frac{6}{7}" /> ke dalam <InlineMath math="2x + 2y - 3z" />.

<MathContainer>
<BlockMath math="2x + 2y - 3z = 2\left(\frac{5}{7}\right) + 2(0) - 3\left(-\frac{6}{7}\right)" />

<BlockMath math="2x + 2y - 3z = \frac{10}{7} - 0 + \frac{18}{7}" />

<BlockMath math="2x + 2y - 3z = \frac{28}{7}" />

<BlockMath math="2x + 2y - 3z = 4" />
</MathContainer>

Jadi, nilai dari <InlineMath math="2x + 2y - 3z" /> adalah <InlineMath math="4" />.
