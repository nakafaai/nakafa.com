We need to determine the distance from point <InlineMath math="(a, a^2 - 1)" /> to the line <InlineMath math="-4x + 3y + 6 = 0" />

<MathContainer>
<BlockMath math="distance = \frac{|-4x + 3y + 6|}{\sqrt{(-4)^2 + 3^2}}" />

<BlockMath math="f(a) = \frac{|-4a + 3(a^2 - 1) + 6|}{5}" />

<BlockMath math="f(a) = \frac{|3a^2 - 4a + 3|}{5}" />

<BlockMath math="f(a) = \frac{1}{5}(3a^2 - 4a + 3)" />
</MathContainer>

The condition for minimum value is <InlineMath math="f'(x) = 0" />

<MathContainer>
<BlockMath math="f'(a) = \frac{1}{5}(6a - 4)" />

<BlockMath math="f'(a) = 0" />

<BlockMath math="\frac{1}{5}(6a - 4) = 0" />

<BlockMath math="6a = 4" />

<BlockMath math="a = \frac{4}{6} = \frac{2}{3}" />
</MathContainer>

Distance between line and parabola when <InlineMath math="a = \frac{2}{3}" />

<MathContainer>
<BlockMath math="f(a) = \frac{1}{5}(3a^2 - 4a + 3)" />

<BlockMath math="f(a) = \frac{1}{5}\left(3\left(\frac{2}{3}\right)^2 - 4\left(\frac{2}{3}\right) + 3\right)" />

<BlockMath math="f(a) = \frac{1}{5}\left(\frac{4}{3} - \frac{8}{3} + \frac{9}{3}\right)" />

<BlockMath math="f(a) = \frac{1}{5}\left(\frac{5}{3}\right) = \frac{1}{3}" />
</MathContainer>
