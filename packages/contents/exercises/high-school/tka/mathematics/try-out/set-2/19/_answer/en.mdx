Recall some logarithm concepts

<MathContainer>
<BlockMath math="\log_a b = \frac{1}{\log_b a}" />

<BlockMath math="\log_a bc = \log_a b + \log_a c" />
</MathContainer>

And also <InlineMath math="\log_{a^m} b^n = \frac{n}{m} \cdot \log_a b" />.

Let's simplify the logarithm form

<MathContainer>
<BlockMath math="\log_2 2x + \log_3 3x = \log_4 4x^2" />

<BlockMath math="\log_2 2 + \log_2 x + \log_3 3 + \log_3 x = \log_4 4 + \log_4 x^2" />

<BlockMath math="1 + \log_2 x + 1 + \log_3 x = 1 + \log_{2^2} x^2" />

<BlockMath math="1 + \log_2 x + \log_3 x = \frac{2}{2} \cdot \log_2 x" />

<BlockMath math="\log_3 x = -1" />

<BlockMath math="\log_x 3 = \frac{1}{-1} = -1" />
</MathContainer>

Therefore the solution is <InlineMath math="x = a" /> so
