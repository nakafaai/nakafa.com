<BlockMath math="|x - 2a| + a \leq b" />

<BlockMath math="|x - 2a| \leq b - a" />

<BlockMath math="-(b - a) \leq x - 2a \leq b - a" />

<BlockMath math="-b + a \leq x - 2a \leq b - a" />
</MathContainer>

Kedua ruas ditambah <InlineMath math="2a" />

<MathContainer>
<BlockMath math="-b + a + 2a \leq x - 2a + 2a \leq b - a + 2a" />

<BlockMath math="-b + 3a \leq x \leq b + a" />
</MathContainer>

Maka solusinya adalah <InlineMath math="-b + 3a \leq x \leq b + a" />.
