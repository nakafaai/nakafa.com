The logarithm conditions are

<MathContainer>
<BlockMath math="|x + 1| > 0 \rightarrow x \neq -1" />

<BlockMath math="|2x - 1| > 0 \rightarrow x \neq \frac{1}{2}" />
</MathContainer>

Recall some basic concepts of absolute value inequality

<BlockMath math="\log_a f(x) \geq \log_a g(x) \rightarrow f(x) \geq g(x)" />

With condition <InlineMath math="a > 1" />.

Let's solve the inequality

<MathContainer>
<BlockMath math="\log|x + 1| \geq \log 3 + \log|2x - 1|" />

<BlockMath math="\log|x + 1| \geq \log 3|2x - 1|" />

<BlockMath math="\log|x + 1| \geq \log|6x - 3|" />

<BlockMath math="|x + 1| \geq |6x - 3|" />
</MathContainer>

The solution

<MathContainer>
<BlockMath math="[(x + 1) + (6x - 3)][(x + 1) - (6x - 3)] \geq 0" />

<BlockMath math="(7x - 2)(-5x + 4) \geq 0" />

<BlockMath math="x = \frac{2}{7} \vee x = \frac{4}{5}" />
</MathContainer>
