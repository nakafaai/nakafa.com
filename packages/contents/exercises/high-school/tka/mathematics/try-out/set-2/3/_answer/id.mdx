Bilangan bulat <InlineMath math="p" /> adalah akar dari <InlineMath math="f(x) = 0" />. Maka bisa kita simpulkan bahwa <InlineMath math="x = p" /> sehingga <InlineMath math="f(p) = 0" />

<MathContainer>
<BlockMath math="f(x) = px^2 - 3x - p - 3" />

<BlockMath math="f(p) = 0" />

<BlockMath math="p \cdot p^2 - 3p - p - 3 = 0" />

<BlockMath math="p^3 - 3p - p - 3 = 0" />

<BlockMath math="p^3 - 4p - 3 = 0" />

<BlockMath math="(p + 1)(p^2 - p - 3) = 0" />

<BlockMath math="p = -1 \cup p^2 - p - 3 = 0" />
</MathContainer>

Pemfaktoran diatas didapat dari cara Horner.

Karena bentuk <InlineMath math="p^2 - p - 3 = 0" /> tidak dapat difaktorkan lagi, maka kita hanya memperoleh <InlineMath math="p = -1" />. Sehingga fungsinya menjadi

<MathContainer>
<BlockMath math="f(x) = -1x^2 - 3x - (-1) - 3" />

<BlockMath math="f(x) = -x^2 - 3x - 2" />

<BlockMath math="f'(x) = -2x - 3" />
</MathContainer>

Dengan <InlineMath math="x = p = -1" /> maka gradiennya adalah

<MathContainer>
<BlockMath math="m = f'(x)" />

<BlockMath math="m = f'(-1)" />

<BlockMath math="m = -2(-1) - 3" />

<BlockMath math="m = 2 - 3" />

<BlockMath math="m = -1" />
</MathContainer>
