#### Faktorkan pembagi pertama

<MathContainer>

<MathContainer>

#### Faktorkan pembagi kedua

Diketahui suku banyak <InlineMath math="f(x)" /> dengan pembagi <InlineMath math="2x^2 + 3x + 1 = (2x + 1)(x + 1) = 0" />, maka <InlineMath math="x = -\frac{1}{2} \cup x = -1" />.

Sisa pembagian adalah <InlineMath math="s(x) = -2bx + a - 11" />.

Untuk <InlineMath math="x = -\frac{1}{2}" />

<MathContainer>
<BlockMath math="f\left(-\frac{1}{2}\right) = s\left(-\frac{1}{2}\right)" />

<BlockMath math="f\left(-\frac{1}{2}\right) = -2b\left(-\frac{1}{2}\right) + a - 11" />

<BlockMath math="f\left(-\frac{1}{2}\right) = a + b - 11 \quad ...(3)" />
</MathContainer>

#### Gunakan kondisi habis dibagi

<InlineMath math="f(x - 2)" /> habis dibagi oleh <InlineMath math="x - 3" />, dengan begitu pembaginya <InlineMath math="x - 3 = 0 \rightarrow x = 3" />.

Sisa pembagian adalah <InlineMath math="s(x) = 0" /> (karena habis dibagi).

<MathContainer>
<BlockMath math="x = 3 \rightarrow f(x - 2) = 0" />

<BlockMath math="f(3 - 2) = 0" />

<BlockMath math="f(1) = 0 \quad ...(4)" />
</MathContainer>

#### Selesaikan sistem persamaan

Dari persamaan kedua dan keempat

<MathContainer>
<BlockMath math="4a - b = 0" />

<BlockMath math="b = 4a \quad ...(5)" />
</MathContainer>

Dari persamaan pertama dan ketiga, serta <InlineMath math="b = 4a" />

<MathContainer>
<BlockMath math="f\left(-\frac{1}{2}\right) = f\left(-\frac{1}{2}\right)" />

<BlockMath math="a + b - 11 = -2a - b" />

<BlockMath math="3a + 2b = 11" />

<BlockMath math="3a + 2(4a) = 11" />

<BlockMath math="3a + 8a = 11" />

<BlockMath math="11a = 11" />

<BlockMath math="a = 1" />
</MathContainer>

Substitusi nilai <InlineMath math="a = 1" /> ke persamaan kelima

<BlockMath math="b = 4a = 4(1) = 4" />

Sehingga nilai dari <InlineMath math="a + 2b + 6 = 1 + 2(4) + 6 = 15" />.
