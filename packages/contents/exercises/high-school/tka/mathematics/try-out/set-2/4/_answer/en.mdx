Given <InlineMath math="f(a) = 19" /> then

<MathContainer>
<BlockMath math="f(x) = ax^2 + 2ax + a + 1" />

<BlockMath math="f(a) = 19" />

<BlockMath math="a \cdot a^2 + 2a \cdot a + a + 1 = 19" />

<BlockMath math="a^3 + 2a^2 + a - 18 = 0" />
</MathContainer>

Factor using Horner's method to get

<MathContainer>
<BlockMath math="a^3 + 2a^2 + a - 18 = 0" />

<BlockMath math="(a - 2)(a^2 + 4a + 9) = 0" />

<BlockMath math="a - 2 = 0 \cup (a^2 + 4a + 9) = 0" />

<BlockMath math="a = 2 \cup a^2 + 4a + 9 = 0" />
</MathContainer>

Thus we obtain <InlineMath math="a = 2" />, and the quadratic function becomes

<MathContainer>
<BlockMath math="f(x) = ax^2 + 2ax + a + 1" />

<BlockMath math="f(x) = 2x^2 + 2 \cdot 2x + 2 + 1" />

<BlockMath math="f(x) = 2x^2 + 4x + 3" />
</MathContainer>

Vertex point <InlineMath math="p" />

<MathContainer>
<BlockMath math="p = \frac{-b}{2a} = \frac{-4}{2(2)} = -1" />

<BlockMath math="q = f(p) = f(-1)" />

<BlockMath math="= 2(-1)^2 + 4(-1) + 3" />

<BlockMath math="q = 1" />
</MathContainer>

Therefore the value of <InlineMath math="p + 2q + 3a" /> is

<BlockMath math="-1 + 2(1) + 3(2) = -1 + 2 + 6 = 7" />
