Recall the derivative concepts for trigonometric functions

<MathContainer>
<BlockMath math="y = a \cdot \sin x \rightarrow y' = a \cdot \cos x" />

<BlockMath math="y = a \cdot \cos x \rightarrow y' = -a \cdot \sin x" />
</MathContainer>

#### Finding the stationary points

The condition for maximum value is when <InlineMath math="f'(x) = 0" />.

<MathContainer>
<BlockMath math="f'(x) = 0" />

<BlockMath math="3 \cos x - 3 \sin x = 0" />

<BlockMath math="3 \cos x = 3 \sin x" />

<BlockMath math="\frac{\sin x}{\cos x} = 1" />

<BlockMath math="\tan x = 1" />
</MathContainer>

Then the values of <InlineMath math="x" /> that satisfy <InlineMath math="\tan x = 1" /> are <InlineMath math="x = \frac{\pi}{4}" /> and <InlineMath math="x = \frac{5\pi}{4}" />.

#### Testing maximum and minimum values

Check first which function has maximum and minimum values.

For the value <InlineMath math="x = \frac{\pi}{4}" />

<MathContainer>
<BlockMath math="x = \frac{\pi}{4} \rightarrow f\left(\frac{\pi}{4}\right) = 3 \sin \frac{\pi}{4} + 3 \cos \frac{\pi}{4}" />

<BlockMath math="x = \frac{\pi}{4} \rightarrow f\left(\frac{\pi}{4}\right) = 3\left(\frac{1}{2}\sqrt{2}\right) + 3\left(\frac{1}{2}\sqrt{2}\right)" />

<BlockMath math="x = \frac{\pi}{4} \rightarrow f\left(\frac{\pi}{4}\right) = 3\sqrt{2} \text{ (max)}" />
</MathContainer>

For the value <InlineMath math="x = \frac{5\pi}{4}" />

<MathContainer>
<BlockMath math="x = \frac{5\pi}{4} \rightarrow f\left(\frac{5\pi}{4}\right) = 3 \sin \frac{5\pi}{4} + 3 \cos \frac{5\pi}{4}" />

<BlockMath math="x = \frac{5\pi}{4} \rightarrow f\left(\frac{5\pi}{4}\right) = 3\left(-\frac{1}{2}\sqrt{2}\right) + 3\left(-\frac{1}{2}\sqrt{2}\right)" />

<BlockMath math="x = \frac{5\pi}{4} \rightarrow f\left(\frac{5\pi}{4}\right) = -3\sqrt{2} \text{ (min)}" />
</MathContainer>

Therefore, the maximum value is reached when <InlineMath math="x = \frac{\pi}{4}" />.
