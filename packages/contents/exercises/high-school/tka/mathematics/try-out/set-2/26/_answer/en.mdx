Combine like terms so the function becomes

<BlockMath math="f(x) = (m - 2)x^2 + 2mx + (m - 3)" />

with <InlineMath math="a = m - 2" />, <InlineMath math="b = 2m" /> and <InlineMath math="c = m - 3" />.

The condition for the function to always be below the <InlineMath math="x" /> axis is <InlineMath math="D < 0" /> and <InlineMath math="a < 0" />.

#### Finding the discriminant value

<MathContainer>
<BlockMath math="D < 0" />

<BlockMath math="b^2 - 4ac < 0" />

<BlockMath math="(2m)^2 - 4(m - 2)(m - 3) < 0" />

<BlockMath math="4m^2 - 4(m^2 - 5m + 6) < 0" />

<BlockMath math="m^2 - m^2 + 5m - 6 < 0" />

<BlockMath math="5m - 6 < 0" />

<BlockMath math="5m < 6" />

<BlockMath math="m < \frac{6}{5}" />

<BlockMath math="m < 1\frac{1}{5}" />
</MathContainer>

#### Finding the coefficient value

<MathContainer>
<BlockMath math="a < 0" />

<BlockMath math="m - 2 < 0" />

<BlockMath math="m < 2" />
</MathContainer>

The intersection of both solutions is <InlineMath math="m < 1\frac{1}{5}" />.

Therefore, the value of <InlineMath math="m" /> below the <InlineMath math="x" /> axis that is possible is <InlineMath math="m < 1\frac{1}{5}" />.
