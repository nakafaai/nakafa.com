Definisikan nilai mutlak yang pertama untuk <InlineMath math="|x - 2|" />.

<InlineMath math="x - 2" /> bernilai positif untuk <InlineMath math="x - 2 \geq 0 \rightarrow x \geq 0" />.

<InlineMath math="x - 2" /> bernilai negatif untuk <InlineMath math="x - 2 < 0 \rightarrow x < 2" />.

Maka definisinya

<BlockMath math="|x - 2| = \begin{cases} x - 2; \text{untuk } x \geq 2 \\ -(x - 2); \text{untuk } x < 2 \end{cases}" />

Untuk <InlineMath math="|x^2 - 4|" />

<InlineMath math="x^2 - 4" /> positif untuk <InlineMath math="x^2 - 4 \geq 0 \rightarrow x \leq -2 \vee x \geq 0" />.

<InlineMath math="x^2 - 4" /> negatif untuk <InlineMath math="x^2 - 4 < 0 \rightarrow -2 < x < 2" />.

Maka definisinya

<BlockMath math="|x^2 - 4| = \begin{cases} x^2 - 4; \text{untuk } x \leq -2 \vee x \geq 2 \\ -(x^2 - 4); \text{untuk } -2 < x < 2 \end{cases}" />

Berdasarkan definisi diatas, bentuk mutlaknya dibatasi oleh <InlineMath math="x = -2" /> dan <InlineMath math="x = 2" />. Artinya ada <InlineMath math="3" /> kemungkinan daerah/nilai <InlineMath math="x" /> yaitu

Menyelesaikan soal berdasarkan daerahnya.

Daerah I saat <InlineMath math="x < -2" />

<MathContainer>
<BlockMath math="|x^2 - 4| = x + |x - 2|" />

<BlockMath math="x^2 - 4 = x + (-x + 2)" />

<BlockMath math="x^2 = 6" />

<BlockMath math="x = \pm\sqrt{6}" />
</MathContainer>

Karena daerah I merupakan negatif, maka akar yang memenuhi adalah <InlineMath math="x_1 = -\sqrt{6}" />.

Daerah II saat <InlineMath math="-2 \leq x < 2" />

<MathContainer>
<BlockMath math="|x^2 - 4| = x + |x - 2|" />

<BlockMath math="-x^2 + 4 = x + (-x + 2)" />

<BlockMath math="x^2 = 2" />

<BlockMath math="x = \pm\sqrt{2}" />
</MathContainer>

Karena daerah II mencakup positif dan negatif maka semua akar memenuhi daerah II.

Daerah III saat <InlineMath math="x \geq 2" />

<MathContainer>
<BlockMath math="|x^2 - 4| = x + |x - 2|" />

<BlockMath math="x^2 - 4 = x + x - 2" />

<BlockMath math="x^2 - 2x - 2 = 0" />
</MathContainer>

Tentukan nilai diskriminannya untuk mencari jenis akar-akar.

<MathContainer>
<BlockMath math="D = b^2 - 4ac" />

<BlockMath math="D = (-2)^2 - 4(1)(-2)" />

<BlockMath math="D = 4 + 8" />

<BlockMath math="D = 12" />
</MathContainer>

Karena <InlineMath math="D > 0" /> maka akarnya berlainan, satu akar bernilai positif yang memenuhi daerah III.

Sehingga jumlah semua himpunan penyelesaiannya adalah <InlineMath math="4" /> penyelesaian.
