The integer <InlineMath math="p" /> is a root of <InlineMath math="f(x) = 0" />. We can conclude that <InlineMath math="x = p" /> so <InlineMath math="f(p) = 0" />

<MathContainer>
<BlockMath math="f(x) = px^2 - 3x - p - 3" />

<BlockMath math="f(p) = 0" />

<BlockMath math="p \cdot p^2 - 3p - p - 3 = 0" />

<BlockMath math="p^3 - 3p - p - 3 = 0" />

<BlockMath math="p^3 - 4p - 3 = 0" />

<BlockMath math="(p + 1)(p^2 - p - 3) = 0" />

<BlockMath math="p = -1 \cup p^2 - p - 3 = 0" />
</MathContainer>

The factorization above is obtained using Horner's method.

Since the form <InlineMath math="p^2 - p - 3 = 0" /> cannot be factored further, we only obtain <InlineMath math="p = -1" />. Thus the function becomes

<MathContainer>
<BlockMath math="f(x) = -1x^2 - 3x - (-1) - 3" />

<BlockMath math="f(x) = -x^2 - 3x - 2" />

<BlockMath math="f'(x) = -2x - 3" />
</MathContainer>

With <InlineMath math="x = p = -1" />, the gradient is

<MathContainer>
<BlockMath math="m = f'(x)" />

<BlockMath math="m = f'(-1)" />

<BlockMath math="m = -2(-1) - 3" />

<BlockMath math="m = 2 - 3" />

<BlockMath math="m = -1" />
</MathContainer>
