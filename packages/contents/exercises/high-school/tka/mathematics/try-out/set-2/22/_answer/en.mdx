The absolute value has the definition

<BlockMath math="|x + 3| = \begin{cases} x + 3 ; \text{for } x \geq -3 \\ -(x + 3) ; \text{for } x < -3 \end{cases}" />

Solve the inequality

For <InlineMath math="x \geq -3" />

<MathContainer>
<BlockMath math="9 - x^2 \geq |x + 3|" />

<BlockMath math="9 - x^2 \geq x + 3" />

<BlockMath math="-x^2 - x + 6 \geq 0" />

<BlockMath math="(-x + 2)(x + 3) \geq 0" />

<BlockMath math="x = 2 \vee x = -3" />
</MathContainer>

Create the number line

<NumberLine
  title={<>Number Line for <InlineMath math="x \geq -3" /></>}

From the condition <InlineMath math="x \geq -3" /> and the number line region above, the solution set is <InlineMath math="\{-3 \leq x \leq 2\}" />.

For <InlineMath math="x < -3" />

<MathContainer>
<BlockMath math="9 - x^2 \geq |x + 3|" />

<BlockMath math="9 - x^2 \geq -(x + 3)" />

<BlockMath math="-x^2 + x + 12 \geq 0" />

<BlockMath math="-(x - 4)(x + 3) \geq 0" />

<BlockMath math="x = 4 \vee x = -3" />
</MathContainer>

From the condition <InlineMath math="x < -3" /> and the second number line region above, the solution set is an empty set.
