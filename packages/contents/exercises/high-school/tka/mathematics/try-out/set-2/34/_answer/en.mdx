#### Factor the first divisor

<MathContainer>
<BlockMath math="2x^2 - x - 1 = (2x + 1)(x - 1) = 0" />

<BlockMath math="x = -\frac{1}{2} \cup x = 1" />
</MathContainer>

The remainder is <InlineMath math="s(x) = 4ax - b" />. The roots of the divisor are <InlineMath math="-\frac{1}{2}" /> and <InlineMath math="1" />, so

For <InlineMath math="x = -\frac{1}{2}" />

<MathContainer>
<BlockMath math="f\left(-\frac{1}{2}\right) = s\left(-\frac{1}{2}\right)" />

<BlockMath math="f\left(-\frac{1}{2}\right) = 4a\left(-\frac{1}{2}\right) - b" />

<BlockMath math="f\left(-\frac{1}{2}\right) = -2a - b \quad ...(1)" />
</MathContainer>

For <InlineMath math="x = 1" />

<MathContainer>
<BlockMath math="f(1) = s(1)" />

<BlockMath math="f(1) = 4a - b \quad ...(2)" />
</MathContainer>

#### Factor the second divisor

Given the polynomial <InlineMath math="f(x)" /> with divisor <InlineMath math="2x^2 + 3x + 1 = (2x + 1)(x + 1) = 0" />, then <InlineMath math="x = -\frac{1}{2} \cup x = -1" />.

The remainder is <InlineMath math="s(x) = -2bx + a - 11" />.

For <InlineMath math="x = -\frac{1}{2}" />

<MathContainer>
<BlockMath math="f\left(-\frac{1}{2}\right) = s\left(-\frac{1}{2}\right)" />

<BlockMath math="f\left(-\frac{1}{2}\right) = -2b\left(-\frac{1}{2}\right) + a - 11" />

<BlockMath math="f\left(-\frac{1}{2}\right) = a + b - 11 \quad ...(3)" />
</MathContainer>

#### Use the divisibility condition

<InlineMath math="f(x - 2)" /> is divisible by <InlineMath math="x - 3" />, so the divisor <InlineMath math="x - 3 = 0 \rightarrow x = 3" />.

The remainder is <InlineMath math="s(x) = 0" /> (because it is divisible).

<MathContainer>
<BlockMath math="x = 3 \rightarrow f(x - 2) = 0" />

<BlockMath math="f(3 - 2) = 0" />

<BlockMath math="f(1) = 0 \quad ...(4)" />
</MathContainer>

#### Solve the system of equations

From the second and fourth equations

<MathContainer>
<BlockMath math="4a - b = 0" />

<BlockMath math="b = 4a \quad ...(5)" />
</MathContainer>

From the first and third equations, with <InlineMath math="b = 4a" />

<MathContainer>
<BlockMath math="f\left(-\frac{1}{2}\right) = f\left(-\frac{1}{2}\right)" />

<BlockMath math="a + b - 11 = -2a - b" />

<BlockMath math="3a + 2b = 11" />

<BlockMath math="3a + 2(4a) = 11" />

<BlockMath math="3a + 8a = 11" />

<BlockMath math="11a = 11" />

<BlockMath math="a = 1" />
</MathContainer>

Substitute the value <InlineMath math="a = 1" /> into the fifth equation

<BlockMath math="b = 4a = 4(1) = 4" />

Therefore, the value of <InlineMath math="a + 2b + 6 = 1 + 2(4) + 6 = 15" />.
