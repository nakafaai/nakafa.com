Given that the polynomial <InlineMath math="ax^3 + 2x^2 + 5x + b" /> divided by <InlineMath math="(x^2 - 1)" /> has a remainder of <InlineMath math="(6x + 5)" />.

What we can do is find the roots of the equation <InlineMath math="x^2 - 1" />.

<MathContainer>
<BlockMath math="x^2 - 1 = 0" />

<BlockMath math="(x + 1)(x - 1) = 0" />

<BlockMath math="x = -1 \text{ or } x = 1" />
</MathContainer>

#### Substitute for x = -1

<MathContainer>
<BlockMath math="P(x) = ax^3 + 2x^2 + 5x + b" />

<BlockMath math="P(-1) = a(-1)^3 + 2(-1)^2 + 5(-1) + b" />

<BlockMath math="P(-1) = -a + b - 3" />
</MathContainer>

#### Substitute for x = 1

<MathContainer>
<BlockMath math="P(x) = ax^3 + 2x^2 + 5x + b" />

<BlockMath math="P(1) = a(1)^3 + 2(1)^2 + 5(1) + b" />

<BlockMath math="P(1) = a + b + 7" />
</MathContainer>

#### Substitute roots into the remainder

Then substitute the roots of <InlineMath math="x^2 - 1" /> into the remainder <InlineMath math="(6x + 5)" />.

For <InlineMath math="x = -1" />

<MathContainer>
<BlockMath math="P(x) = 6x + 5" />

<BlockMath math="P(-1) = 6(-1) + 5" />

<BlockMath math="-a + b - 3 = -6 + 5" />

<BlockMath math="-a + b = 2 \quad ...(1)" />
</MathContainer>

For <InlineMath math="x = 1" />

<MathContainer>
<BlockMath math="P(x) = 6x + 5" />

<BlockMath math="P(1) = 6(1) + 5" />

<BlockMath math="a + b + 7 = 6 + 5" />

<BlockMath math="a + b = 4 \quad ...(2)" />
</MathContainer>

#### Eliminate the equations

Next, eliminate the first and second equations to get the values of <InlineMath math="a" /> and <InlineMath math="b" />.

<MathContainer>
<BlockMath math="-a + b = 2" />

<BlockMath math="a + b = 4 \quad -" />

<BlockMath math="-2a = -2" />

<BlockMath math="a = 1" />
</MathContainer>

Substitute the value of <InlineMath math="a" /> into the second equation to get the value of <InlineMath math="b" />.

<MathContainer>
<BlockMath math="a + b = 4" />

<BlockMath math="1 + b = 4" />

<BlockMath math="b = 3" />
</MathContainer>

Therefore, the value of <InlineMath math="a + 3b = 1 + 3(3) = 10" />.
