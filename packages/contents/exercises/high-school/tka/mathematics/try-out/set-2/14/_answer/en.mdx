Recall that <InlineMath math="\tan x = \frac{\sin x}{\cos x}" />

<BlockMath math="\lim_{x \to 0} \frac{\cos x \sin x - \tan x}{x^2 \sin x} = \lim_{x \to 0} \frac{\cos x \sin x - \frac{\sin x}{\cos x}}{x^2 \sin x}" />

Factor it

<BlockMath math="\lim_{x \to 0} \frac{\cos x \sin x - \tan x}{x^2 \sin x} = \lim_{x \to 0} \frac{\sin x \left(\cos x - \frac{1}{\cos x}\right)}{x^2 \sin x}" />

<BlockMath math="\lim_{x \to 0} \frac{\cos x \sin x - \tan x}{x^2 \sin x} = \lim_{x \to 0} \frac{\cos x - \frac{1}{\cos x}}{x^2}" />

Common denominator

<MathContainer>
<BlockMath math="\lim_{x \to 0} \frac{\cos x \sin x - \tan x}{x^2 \sin x} = \lim_{x \to 0} \frac{\cos^2 x - 1}{x^2 \cos x}" />

<BlockMath math="\lim_{x \to 0} \frac{\cos x \sin x - \tan x}{x^2 \sin x} = \lim_{x \to 0} \frac{-\sin^2 x}{x^2 \cos x}" />

<BlockMath math="\lim_{x \to 0} \frac{\cos x \sin x - \tan x}{x^2 \sin x} = -1" />
</MathContainer>
