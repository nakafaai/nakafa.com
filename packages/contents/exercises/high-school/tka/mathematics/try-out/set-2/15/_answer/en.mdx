<BlockMath math="\lim_{x \to -3} \frac{\frac{1}{ax} + \frac{1}{3}}{bx^3 + 27} = -\frac{1}{3^5}" />

Common denominator in the numerator

<BlockMath math="\lim_{x \to -3} \frac{\frac{3 + ax}{3ax(bx^3 + 27)}} = -\frac{1}{3^5}" />

For <InlineMath math="x = -3" /> then

<MathContainer>
<BlockMath math="3 + ax = 0" />

<BlockMath math="3 - 3a = 0" />

<BlockMath math="a = 1" />
</MathContainer>

Substitute <InlineMath math="x = -3" /> into the denominator

<MathContainer>
<BlockMath math="bx^3 + 27 = 0" />

<BlockMath math="b(-3)^3 + 27 = 0" />

<BlockMath math="-27b + 27 = 0" />

<BlockMath math="b = 1" />
</MathContainer>

Therefore the value of <InlineMath math="a + b = 1 + 1 = 2" />.
