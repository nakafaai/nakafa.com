Garis <InlineMath math="AH" /> dan <InlineMath math="BD" /> belum berpotongan, sehingga perlu digeser salah satunya agar mereka berpotongan, yaitu geser garis <InlineMath math="AH" /> ke garis <InlineMath math="BG" /> (karena <InlineMath math="AH" /> sejajar <InlineMath math="BG" />), sehingga sudutnya antara garis <InlineMath math="BG" /> dan <InlineMath math="BD" />.

<BlockMath math="\alpha = \angle(AH, BD) = \angle(BG, BD)" />

#### Menentukan panjang segmen

Kemudian menentukan panjang <InlineMath math="\triangle BDG" />

<MathContainer>
<BlockMath math="BD = \sqrt{AB^2 + AD^2} = \sqrt{6^2 + 8^2} = 10" />

<BlockMath math="DG = \sqrt{CD^2 + CG^2} = \sqrt{6^2 + 4^2} = \sqrt{52} = 2\sqrt{13}" />

<BlockMath math="BG = \sqrt{CB^2 + CG^2} = \sqrt{8^2 + 4^2} = \sqrt{80} = 4\sqrt{5}" />
</MathContainer>

#### Gunakan aturan cosinus

Gunakan aturan cosinus pada <InlineMath math="\triangle BDG" />

<MathContainer>
<BlockMath math="\cos \angle DBG = \frac{BD^2 + BG^2 - DG^2}{2 \cdot BD \cdot BG}" />

<BlockMath math="\cos \alpha = \frac{10^2 + (4\sqrt{5})^2 - (2\sqrt{13})^2}{2(10)(4\sqrt{5})}" />

<BlockMath math="\cos \alpha = \frac{100 + 80 - 52}{80\sqrt{5}}" />

<BlockMath math="\cos \alpha = \frac{128}{80\sqrt{5}}" />

<BlockMath math="\cos \alpha = \frac{8}{5\sqrt{5}}" />
</MathContainer>

Maka nilai dari <InlineMath math="\cos 2\alpha" /> dapat dihitung menggunakan rumus sudut ganda

<MathContainer>
<BlockMath math="\cos 2\alpha = 2\cos^2 \alpha - 1" />

<BlockMath math="\cos 2\alpha = 2\left(\frac{8}{5\sqrt{5}}\right)^2 - 1" />

<BlockMath math="\cos 2\alpha = 2\left(\frac{64}{125}\right) - 1" />

<BlockMath math="\cos 2\alpha = \frac{128}{125} - \frac{125}{125}" />

<BlockMath math="\cos 2\alpha = \frac{3}{125}" />
</MathContainer>

Jadi, nilai dari <InlineMath math="\cos 2\alpha" /> adalah <InlineMath math="\frac{3}{125}" />.
