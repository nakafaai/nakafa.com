Recall the cosine addition concept

<BlockMath math="\cos(A + B) = \cos A \cos B - \sin A \sin B" />

#### Finding sin A sin B

Substitute the given values into the cosine addition formula.

<MathContainer>
<BlockMath math="\cos(A + B) = \frac{2}{5}" />

<BlockMath math="\cos A \cos B - \sin A \sin B = \frac{2}{5}" />

<BlockMath math="\frac{3}{4} - \sin A \sin B = \frac{2}{5}" />

<BlockMath math="\sin A \sin B = \frac{7}{20}" />
</MathContainer>

#### Finding tan A tan B

Recall that <InlineMath math="\tan A = \frac{\sin A}{\cos A}" />, then

<MathContainer>
<BlockMath math="\tan A \tan B = \frac{\sin A \sin B}{\cos A \cos B}" />

<BlockMath math="\tan A \tan B = \frac{\frac{7}{20}}{\frac{3}{4}}" />

<BlockMath math="\tan A \tan B = \frac{7}{20} \times \frac{4}{3}" />

<BlockMath math="\tan A \tan B = \frac{7}{15}" />
</MathContainer>

Therefore, the value of <InlineMath math="\tan A \tan B" /> is <InlineMath math="\frac{7}{15}" />.
