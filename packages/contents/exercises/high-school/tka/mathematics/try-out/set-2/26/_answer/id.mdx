Gabungkan suku sejenis sehingga fungsi menjadi

<BlockMath math="f(x) = (m - 2)x^2 + 2mx + (m - 3)" />

dengan <InlineMath math="a = m - 2" />, <InlineMath math="b = 2m" /> dan <InlineMath math="c = m - 3" />.

Syarat fungsi tersebut senantiasa berada di bawah sumbu <InlineMath math="x" /> adalah <InlineMath math="D < 0" /> dan <InlineMath math="a < 0" />.

#### Mencari nilai diskriminan

<MathContainer>
<BlockMath math="D < 0" />

<BlockMath math="b^2 - 4ac < 0" />

<BlockMath math="(2m)^2 - 4(m - 2)(m - 3) < 0" />

<BlockMath math="4m^2 - 4(m^2 - 5m + 6) < 0" />

<BlockMath math="m^2 - m^2 + 5m - 6 < 0" />

<BlockMath math="5m - 6 < 0" />

<BlockMath math="5m < 6" />

<BlockMath math="m < \frac{6}{5}" />

<BlockMath math="m < 1\frac{1}{5}" />
</MathContainer>

#### Mencari nilai koefisien

<MathContainer>
<BlockMath math="a < 0" />

<BlockMath math="m - 2 < 0" />

<BlockMath math="m < 2" />
</MathContainer>

Solusi irisan keduanya adalah <InlineMath math="m < 1\frac{1}{5}" />.

Jadi, nilai <InlineMath math="m" /> yang dibawah sumbu <InlineMath math="x" /> yang mungkin adalah <InlineMath math="m < 1\frac{1}{5}" />.
