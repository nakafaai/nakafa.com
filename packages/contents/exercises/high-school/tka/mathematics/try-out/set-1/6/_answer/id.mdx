Untuk menganalisis fungsi <InlineMath math="H(t) = A \cos(Bt) + C" />, kita perlu menentukan tiga parameter berdasarkan data yang diberikan.

#### Amplitudo

Amplitudo <InlineMath math="A" /> adalah setengah dari selisih antara nilai maksimum dan minimum fungsi

<BlockMath math="A = \frac{\text{maksimum} - \text{minimum}}{2} = \frac{10 - 2}{2} = \frac{8}{2} = 4" />

Jadi, <InlineMath math="|A| = 4" />. Karena fungsi kosinus dimulai dari nilai maksimum, kita dapat mengambil <InlineMath math="A = 4" /> (positif).

#### Nilai Vertikal

Nilai vertikal <InlineMath math="C" /> adalah nilai tengah antara nilai maksimum dan minimum, atau rata-rata dari nilai maksimum dan minimum

<BlockMath math="C = \frac{\text{maksimum} + \text{minimum}}{2} = \frac{10 + 2}{2} = \frac{12}{2} = 6" />

Jadi, <InlineMath math="C = 6" />

#### Periode dan Nilai B

Periode fungsi <InlineMath math="\cos(Bt)" /> adalah <InlineMath math="\frac{2\pi}{B}" />

Diketahui periode pasang surut adalah <InlineMath math="12" /> jam

<MathContainer>
<BlockMath math="\text{Periode} = \frac{2\pi}{B} = 12" />

<BlockMath math="|B| = \frac{2\pi}{12} = \frac{\pi}{6}" />
</MathContainer>

Karena periode pasang surut biasanya positif, kita ambil <InlineMath math="B = \frac{\pi}{6}" /> (positif).

Dengan demikian, nilai-nilai parameter adalah <InlineMath math="A = 4" />, <InlineMath math="B = \frac{\pi}{6}" />, dan <InlineMath math="C = 6" />

Pilihan jawaban yang paling tepat adalah <InlineMath math="A = 4, B = \frac{\pi}{6}, C = 6" />
