Diketahui <InlineMath math="y = \cos^3 x" />

#### Turunan Menggunakan Aturan Rantai

Gunakan aturan rantai untuk menurunkan fungsi komposisi

<MathContainer>
<BlockMath math="y' = 3(\cos^2 x) \cdot (-\sin x)" />

<BlockMath math="= -3\cos^2 x \sin x" />

<BlockMath math="= -3\cos x \cos x \sin x" />
</MathContainer>

#### Menggunakan Identitas Trigonometri

Gunakan identitas <InlineMath math="2\sin x \cos x = \sin 2x" />

<MathContainer>
<BlockMath math="= \frac{-3}{2}\cos x (2\sin x \cos x)" />

<BlockMath math="= \frac{-3}{2}\cos x \cdot \sin 2x" />
</MathContainer>

Jadi, turunan pertama dari <InlineMath math="y = \cos^3 x" /> adalah <InlineMath math="y' = \frac{-3}{2}\cos x \cdot \sin 2x" />
