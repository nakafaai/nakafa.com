Diketahui persamaan <InlineMath math="\cos 2x - 2\cos x = -1" /> untuk <InlineMath math="0 \leq x \leq 2\pi" />

#### Mengubah Bentuk Persamaan

Gunakan identitas trigonometri <InlineMath math="\cos 2x = 2\cos^2 x - 1" />

<MathContainer>
<BlockMath math="\cos 2x - 2\cos x = -1" />

<BlockMath math="(2\cos^2 x - 1) - 2\cos x + 1 = 0" />

<BlockMath math="2\cos^2 x - 2\cos x = 0" />

<BlockMath math="\cos x(\cos x - 1) = 0" />
</MathContainer>

Sehingga <InlineMath math="\cos x = 0" /> atau <InlineMath math="\cos x = 1" />

#### Menyelesaikan untuk cos x = 0

Jika <InlineMath math="\cos x = \cos \alpha" />, maka

<BlockMath math="x_1 = \alpha + k \cdot 2\pi \quad \text{dan} \quad x_2 = -\alpha + k \cdot 2\pi" />

Untuk <InlineMath math="\cos x = \cos \frac{\pi}{2} = 0" />

<MathContainer>
<BlockMath math="k = 0 \Rightarrow x_1 = \frac{\pi}{2} + 0 \cdot 2\pi = \frac{1}{2}\pi" />

<BlockMath math="k = 1 \Rightarrow x_2 = -\frac{\pi}{2} + 2\pi = \frac{3}{2}\pi" />
</MathContainer>

#### Menyelesaikan untuk cos x = 1

Untuk <InlineMath math="\cos x = \cos 0 = 1" />

<MathContainer>
<BlockMath math="k = 0 \Rightarrow x_1 = 0 + 0 \cdot 2\pi = 0" />

<BlockMath math="k = 1 \Rightarrow x_2 = -0 + 2\pi = 2\pi" />
</MathContainer>

#### Himpunan Penyelesaian

Jadi, himpunan penyelesaian dari persamaan <InlineMath math="\cos 2x - 2\cos x = -1" /> untuk <InlineMath math="0 \leq x \leq 2\pi" /> adalah

<BlockMath math="\left\{0, \frac{1}{2}\pi, \frac{3}{2}\pi, 2\pi\right\}" />
