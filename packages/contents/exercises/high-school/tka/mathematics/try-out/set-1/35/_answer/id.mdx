Untuk menghitung luas daerah yang dibatasi oleh kedua kurva, cari terlebih dahulu titik potong antara kurva parabola dan garis.

#### Mencari Titik Potong

Titik potong diperoleh dengan menyamakan kedua persamaan

<MathContainer>
<BlockMath math="x^2 - 4x + 3 = 3 - x" />

<BlockMath math="x^2 - 3x = 0" />

<BlockMath math="x(x - 3) = 0" />
</MathContainer>

Jadi, <InlineMath math="x_1 = 0" /> atau <InlineMath math="x_2 = 3" />

#### Menentukan Kurva Atas dan Bawah

Di antara interval <InlineMath math="x = 0" /> dan <InlineMath math="x = 3" />, garis <InlineMath math="y = 3 - x" /> berada di atas parabola <InlineMath math="y = x^2 - 4x + 3" />

#### Menghitung Luas Daerah

Luas daerah dihitung menggunakan integral

<MathContainer>
<BlockMath math="L = \int_{x_1}^{x_2} (y_{\text{atas}} - y_{\text{bawah}}) dx" />

<BlockMath math="= \int_0^3 [(3 - x) - (x^2 - 4x + 3)] dx" />

<BlockMath math="= \int_0^3 (-x^2 + 3x) dx" />

<BlockMath math="= \left[-\frac{1}{3}x^3 + \frac{3}{2}x^2\right]_0^3" />

<BlockMath math="= \left(-\frac{1}{3}(3)^3 + \frac{3}{2}(3)^2\right) - \left(-\frac{1}{3}(0)^3 + \frac{3}{2}(0)^2\right)" />

<BlockMath math="= -9 + \frac{27}{2}" />

<BlockMath math="= \frac{9}{2}" />
</MathContainer>

Jadi, luas daerah antara keduanya adalah <InlineMath math="\frac{9}{2}" /> satuan luas.
