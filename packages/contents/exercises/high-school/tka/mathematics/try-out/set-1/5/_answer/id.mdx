Diketahui persamaan kuadrat <InlineMath math="x^2 + ax - 4 = 0" /> dengan akar-akar <InlineMath math="p" /> dan <InlineMath math="q" />

Berdasarkan rumus Vieta, kita tahu bahwa

<MathContainer>
<BlockMath math="p + q = -\frac{b}{a} = -a" />

<BlockMath math="p \cdot q = \frac{c}{a} = -4" />
</MathContainer>

Jika <InlineMath math="p^2 - 2pq + q^2 = 8a" />, maka kita dapat menyederhanakan ruas kiri

<MathContainer>
<BlockMath math="p^2 + q^2 - 2pq = 8a" />

<BlockMath math="(p + q)^2 - 2pq - 2pq = 8a" />

<BlockMath math="(p + q)^2 - 4pq = 8a" />
</MathContainer>

Substitusi nilai <InlineMath math="p + q = -a" /> dan <InlineMath math="pq = -4" />

<MathContainer>
<BlockMath math="(-a)^2 - 4(-4) = 8a" />

<BlockMath math="a^2 + 16 = 8a" />

<BlockMath math="a^2 - 8a + 16 = 0" />
</MathContainer>

Faktorkan persamaan kuadrat

<MathContainer>
<BlockMath math="(a - 4)^2 = 0" />

<BlockMath math="a = 4" />
</MathContainer>

Jadi, nilai <InlineMath math="a" /> yang memenuhi adalah <InlineMath math="4" />
