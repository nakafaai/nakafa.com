Misalkan harga <InlineMath math="1" /> buku <InlineMath math="= x" />, <InlineMath math="1" /> pulpen <InlineMath math="= y" />, <InlineMath math="1" /> pensil <InlineMath math="= z" />

Didapatkan persamaan berikut

<MathContainer>
<BlockMath math="\text{Adi: } 4x + 2y + 3z = 26.000 \quad \text{...(i)}" />

<BlockMath math="\text{Bima: } 3x + 3y + z = 21.500 \quad \text{...(ii)}" />

<BlockMath math="\text{Citra: } 3x + z = 12.500 \quad \text{...(iii)}" />
</MathContainer>

Dari persamaan <InlineMath math="\text{(iii)}" />, kita dapatkan

<BlockMath math="z = 12.500 - 3x" />

Substitusi <InlineMath math="z" /> ke persamaan <InlineMath math="\text{(ii)}" />

<MathContainer>
<BlockMath math="3x + 3y + (12.500 - 3x) = 21.500" />

<BlockMath math="3y + 12.500 = 21.500" />

<BlockMath math="3y = 9.000" />

<BlockMath math="y = 3.000" />
</MathContainer>

Substitusi <InlineMath math="z" /> dan <InlineMath math="y" /> ke persamaan <InlineMath math="\text{(i)}" />

<MathContainer>
<BlockMath math="4x + 2(3.000) + 3(12.500 - 3x) = 26.000" />

<BlockMath math="4x + 6.000 + 37.500 - 9x = 26.000" />

<BlockMath math="-5x + 43.500 = 26.000" />

<BlockMath math="-5x = -17.500" />

<BlockMath math="x = 3.500" />
</MathContainer>

Dari <InlineMath math="x = 3.500" />, kita dapat mencari <InlineMath math="z" />

<MathContainer>
<BlockMath math="z = 12.500 - 3(3.500)" />

<BlockMath math="z = 12.500 - 10.500 = 2.000" />
</MathContainer>

Dina membeli <InlineMath math="2" /> pulpen dan <InlineMath math="2" /> pensil, maka yang harus dibayar adalah

<MathContainer>
<BlockMath math="2y + 2z = 2(3.000) + 2(2.000)" />

<BlockMath math="= 6.000 + 4.000 = 10.000" />
</MathContainer>

Jadi, Dina harus membayar <InlineMath math="\text{Rp}10.000" />
