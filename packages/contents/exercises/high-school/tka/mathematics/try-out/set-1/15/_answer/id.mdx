Diketahui deret aritmetika dengan <InlineMath math="S_n = 2n^2 + 4n" />

#### Menentukan Rumus Suku ke-n

Jika <InlineMath math="U_n = S_n - S_{n-1}" />, maka

<MathContainer>
<BlockMath math="U_n = (2n^2 + 4n) - (2(n-1)^2 + 4(n-1))" />

<BlockMath math="= 2n^2 + 4n - (2(n^2 - 2n + 1) + 4n - 4)" />

<BlockMath math="= 2n^2 + 4n - (2n^2 - 4n + 2 + 4n - 4)" />

<BlockMath math="= 2n^2 + 4n - 2n^2 + 4n - 2 - 4n + 4" />

<BlockMath math="= 4n + 2" />
</MathContainer>

#### Menghitung Suku ke-10

Suku ke-10 deret tersebut

<MathContainer>
<BlockMath math="U_n = 4n + 2" />

<BlockMath math="U_{10} = 4(10) + 2" />

<BlockMath math="= 40 + 2 = 42" />
</MathContainer>

Jadi, suku ke-10 deret tersebut adalah <InlineMath math="42" />
