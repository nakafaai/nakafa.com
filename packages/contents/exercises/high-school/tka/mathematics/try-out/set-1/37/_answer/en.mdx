#### Determining the Modal Class

The modal class is the interval with the highest frequency, which is the interval <InlineMath math="59.5 - 64.5" /> with frequency <InlineMath math="14" />

#### Calculating Frequency Differences

Difference between modal class frequency and previous class frequency

<BlockMath math="d_1 = 14 - 9 = 5" />

Difference between modal class frequency and next class frequency

<BlockMath math="d_2 = 14 - 11 = 3" />

#### Determining Parameters

Lower boundary of modal class

<BlockMath math="T_b = 59.5" />

Class width

<BlockMath math="L = 64.5 - 59.5 = 5" />

#### Calculating the Mode

Using the mode formula for grouped data

<MathContainer>
<BlockMath math="M_o = T_b + \left(\frac{d_1}{d_1 + d_2}\right) \cdot L" />

<BlockMath math="= 59.5 + \left(\frac{5}{5 + 3}\right) \cdot 5" />

<BlockMath math="= 59.5 + \left(\frac{5}{8}\right) \cdot 5" />

<BlockMath math="= 59.5 + 3.125" />

<BlockMath math="= 62.625 \approx 62.6" />
</MathContainer>

Therefore, the mode of the data shown in the histogram is <InlineMath math="62.6" />
