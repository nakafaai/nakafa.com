Diketahui <InlineMath math="f(x) = \frac{2x - 5}{x - 4}" /> dengan <InlineMath math="x \neq 4" /> dan <InlineMath math="g(x) = 3x + 8" />

#### Menentukan Fungsi Komposisi

Hitung fungsi komposisi <InlineMath math="(f \circ g)(x) = f(g(x))" />

<MathContainer>
<BlockMath math="(f \circ g)(x) = f(g(x))" />

<BlockMath math="= f(3x + 8)" />

<BlockMath math="= \frac{2(3x + 8) - 5}{(3x + 8) - 4}" />

<BlockMath math="= \frac{6x + 16 - 5}{3x + 8 - 4}" />

<BlockMath math="= \frac{6x + 11}{3x + 4}" />
</MathContainer>

#### Menentukan Fungsi Invers

Misalkan <InlineMath math="y = \frac{6x + 11}{3x + 4}" />, kemudian selesaikan untuk <InlineMath math="x" /> dalam bentuk <InlineMath math="y" />

<MathContainer>
<BlockMath math="y = \frac{6x + 11}{3x + 4}" />

<BlockMath math="3xy + 4y = 6x + 11" />

<BlockMath math="3xy - 6x = 11 - 4y" />

<BlockMath math="x(3y - 6) = 11 - 4y" />

<BlockMath math="x = \frac{11 - 4y}{3y - 6}" />
</MathContainer>

Ganti <InlineMath math="y" /> dengan <InlineMath math="x" /> untuk mendapatkan fungsi invers

<BlockMath math="(f \circ g)^{-1}(x) = \frac{11 - 4x}{3x - 6}" />

Syarat <InlineMath math="3x - 6 \neq 0 \Rightarrow x \neq 2" />

Jadi, invers dari <InlineMath math="(f \circ g)(x)" /> adalah <InlineMath math="\frac{11 - 4x}{3x - 6}" /> dengan <InlineMath math="x \neq 2" />
