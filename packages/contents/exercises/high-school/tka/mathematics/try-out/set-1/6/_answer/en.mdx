To analyze the function <InlineMath math="H(t) = A \cos(Bt) + C" />, we need to determine three parameters based on the given data.

#### Amplitude

Amplitude <InlineMath math="A" /> is half the difference between the maximum and minimum values of the function

<BlockMath math="A = \frac{\text{maximum} - \text{minimum}}{2} = \frac{10 - 2}{2} = \frac{8}{2} = 4" />

Therefore, <InlineMath math="|A| = 4" />. Since the cosine function starts from the maximum value, we can take <InlineMath math="A = 4" /> (positive).

#### Vertical Shift

The vertical shift <InlineMath math="C" /> is the midpoint between the maximum and minimum values, or the average of the maximum and minimum values

<BlockMath math="C = \frac{\text{maximum} + \text{minimum}}{2} = \frac{10 + 2}{2} = \frac{12}{2} = 6" />

Therefore, <InlineMath math="C = 6" />

#### Period and Value of B

The period of the function <InlineMath math="\cos(Bt)" /> is <InlineMath math="\frac{2\pi}{B}" />

Given that the tidal period is <InlineMath math="12" /> hours

<MathContainer>
<BlockMath math="\text{Period} = \frac{2\pi}{B} = 12" />

<BlockMath math="|B| = \frac{2\pi}{12} = \frac{\pi}{6}" />
</MathContainer>

Since the tidal period is typically positive, we take <InlineMath math="B = \frac{\pi}{6}" /> (positive).

Thus, the parameter values are <InlineMath math="A = 4" />, <InlineMath math="B = \frac{\pi}{6}" />, and <InlineMath math="C = 6" />

The most appropriate answer choice is <InlineMath math="A = 4, B = \frac{\pi}{6}, C = 6" />
