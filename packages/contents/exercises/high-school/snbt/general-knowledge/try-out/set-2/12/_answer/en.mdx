The core of a sentence is the main information formed by its principal elements, such as the Subject (S), Predicate (P), Object (O), and Complement (if any). Additional elements like adjuncts (adverbs) or expansions (subordinate clauses) are not included in the sentence core.

Let's break down the elements in sentence <InlineMath math="(1)" />:
- *Summer (Musim panas)* = Subject
- *usually (biasanya)* = Adjunct (Adverb)
- *makes (membuat)* = Predicate
- *children (anak-anak)* = Object
- *happier (lebih bahagia)* = Complement
- *enjoying outdoor activities (dengan menikmati aktivitas di luar ruangan)* = Adjunct (Adverbial phrase)

If we extract only the main information by removing the adverbs and expansions, the core of sentence <InlineMath math="(1)" /> in the text is **Summer makes children happy** (*Musim panas membuat anak-anak bahagia*).
