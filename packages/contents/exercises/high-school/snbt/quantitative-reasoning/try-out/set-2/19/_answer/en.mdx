We need to find the value of <InlineMath math="Q" /> first, which is the value of <InlineMath math="f(g(a))" /> for <InlineMath math="a = 4" /> or <InlineMath math="a = -4" />.

#### Calculating the value of g(a)

Since <InlineMath math="a = 4" /> or <InlineMath math="a = -4" />, and the function <InlineMath math="g(x)" /> contains <InlineMath math="x^2" />, the result of <InlineMath math="g(a)" /> will be the same for both values of <InlineMath math="a" />.

<MathContainer>
  <BlockMath math="g(4) = 4^2 - 1 = 16 - 1 = 15" />
  <BlockMath math="g(-4) = (-4)^2 - 1 = 16 - 1 = 15" />
</MathContainer>

So, <InlineMath math="g(a) = 15" />.

#### Calculating the value of f(g(a))

Now we substitute the result <InlineMath math="g(a) = 15" /> into the function <InlineMath math="f(x)" />.

<MathContainer>
  <BlockMath math="f(15) = 7 + \sqrt{\frac{4}{15 + 10}}" />
  <BlockMath math="= 7 + \sqrt{\frac{4}{25}}" />
  <BlockMath math="= 7 + \frac{\sqrt{4}}{\sqrt{25}}" />
  <BlockMath math="= 7 + \frac{2}{5}" />
  <BlockMath math="= 7.4" />
</MathContainer>

So, the value of <InlineMath math="Q = 7.4" />.

#### Comparing P and Q

We know that <InlineMath math="P = 7" />.

Thus:
<BlockMath math="7 < 7.4 \Rightarrow P < Q" />

So, Quantity <InlineMath math="P" /> is less than <InlineMath math="Q" />.
