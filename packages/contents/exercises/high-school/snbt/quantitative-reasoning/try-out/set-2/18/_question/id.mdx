Fungsi <InlineMath math="f" /> didefinisikan sebagai <InlineMath math="f(x) = \frac{1}{\sqrt{x^2 - x + n}}" /> dengan <InlineMath math="f(-2) = \frac{1}{2}" />.

Berapakah banyaknya dari empat pernyataan berikut yang bernilai benar berdasarkan informasi di atas?

1. <InlineMath math="n" /> merupakan bilangan ganjil.
2. <InlineMath math="0" /> merupakan anggota daerah hasil fungsi <InlineMath math="f(x)" />.
3. Fungsi <InlineMath math="f" /> bernilai <InlineMath math="\frac{1}{4}" /> ketika <InlineMath math="x = -4" />.
4. Daerah asal fungsi <InlineMath math="f" /> adalah <InlineMath math="x < -1 \cup x > 2" />.
