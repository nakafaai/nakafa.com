Misalkan bilangan pertama adalah <InlineMath math="A" /> dan bilangan kedua adalah <InlineMath math="B" />.

Dari soal, kita mendapatkan informasi sebagai berikut:

1. Dua kali bilangan pertama dikurangi tiga kali bilangan kedua menghasilkan <InlineMath math="14" />:

   <BlockMath math="2A - 3B = 14" />

2. Bilangan pertama dikali <InlineMath math="k" /> ditambah dua kali bilangan kedua menghasilkan <InlineMath math="4" />:

   <BlockMath math="Ak + 2B = 4" />

3. Nilai bilangan pertama lebih banyak <InlineMath math="2" /> daripada <InlineMath math="k" /> (artinya <InlineMath math="A = k + 2" /> atau <InlineMath math="k = A - 2" />).

#### Langkah Penyelesaian

Substitusi nilai <InlineMath math="k = A - 2" /> ke dalam persamaan <InlineMath math="(2)" />:

<MathContainer>
  <BlockMath math="A(A - 2) + 2B = 4" />
  <BlockMath math="A^2 - 2A + 2B = 4" />
</MathContainer>

Ubah persamaan tersebut untuk mendapatkan nilai <InlineMath math="B" />:

<MathContainer>
  <BlockMath math="2B = 4 + 2A - A^2" />
  <BlockMath math="B = \frac{-A^2 + 2A + 4}{2}" />
</MathContainer>

Selanjutnya, substitusi nilai <InlineMath math="B" /> tersebut ke dalam persamaan <InlineMath math="(1)" />:

<MathContainer>
  <BlockMath math="2A - 3\left(\frac{-A^2 + 2A + 4}{2}\right) = 14" />
</MathContainer>

Kalikan kedua ruas dengan <InlineMath math="2" /> untuk menghilangkan penyebut:

<MathContainer>
  <BlockMath math="4A - 3(-A^2 + 2A + 4) = 28" />
  <BlockMath math="4A + 3A^2 - 6A - 12 = 28" />
  <BlockMath math="3A^2 - 2A - 12 = 28" />
  <BlockMath math="3A^2 - 2A - 40 = 0" />
</MathContainer>

Faktorkan persamaan kuadrat tersebut:

<MathContainer>
  <BlockMath math="(3A - 12)(3A + 10) = 0" />
  <BlockMath math="3(A - 4)(3A + 10) = 0" />
</MathContainer>

Maka diperoleh dua kemungkinan nilai <InlineMath math="A" />:

<BlockMath math="A = 4 \quad \text{atau} \quad A = -\frac{10}{3}" />

#### Menentukan Nilai Terbesar Bilangan Kedua

Kita cari nilai <InlineMath math="B" /> untuk masing-masing nilai <InlineMath math="A" /> menggunakan persamaan <InlineMath math="2A - 3B = 14 \Rightarrow 3B = 2A - 14 \Rightarrow B = \frac{2A - 14}{3}" />.

**Kasus 1:** Untuk <InlineMath math="A = 4" />

<BlockMath math="B = \frac{2(4) - 14}{3} = \frac{8 - 14}{3} = \frac{-6}{3} = -2" />

**Kasus 2:** Untuk <InlineMath math="A = -\frac{10}{3}" />

<BlockMath math="B = \frac{2(-\frac{10}{3}) - 14}{3} = \frac{-\frac{20}{3} - \frac{42}{3}}{3} = \frac{-\frac{62}{3}}{3} = -\frac{62}{9} \approx -6,88" />

Bandingkan kedua nilai <InlineMath math="B" /> yang diperoleh: <InlineMath math="-2" /> dan <InlineMath math="-\frac{62}{9}" />.

Nilai <InlineMath math="-2" /> lebih besar daripada <InlineMath math="-\frac{62}{9}" />.

Jadi, nilai terbesar dari bilangan kedua yang mungkin adalah <InlineMath math="-2" />.
