Given the function <InlineMath math="f(x) = (x+3)(x^2+6x+8)^4" /> and <InlineMath math="F(x) = \int f(x) \, dx" /> with the condition <InlineMath math="F(-3) = 2" />.

#### U-Substitution

To solve this integral, we use the substitution method.

Let <InlineMath math="u = x^2 + 6x + 8" />.

The derivative of <InlineMath math="u" /> with respect to <InlineMath math="x" />:

<BlockMath math="\frac{du}{dx} = 2x + 6" />

Factor:

<BlockMath math="\frac{du}{dx} = 2(x + 3)" />

Multiply both sides by <InlineMath math="dx" />:

<BlockMath math="du = 2(x + 3) \, dx" />

Divide both sides by <InlineMath math="2" />:

<BlockMath math="\frac{du}{2} = (x + 3) \, dx" />

#### Solving the Integral

Substitute into the integral:

<BlockMath math="F(x) = \int f(x) \, dx = \int (x+3)(x^2+6x+8)^4 \, dx" />

With substitution <InlineMath math="u = x^2 + 6x + 8" /> and <InlineMath math="(x+3) \, dx = \frac{du}{2}" />:

<MathContainer>
<BlockMath math="F(x) = \int u^4 \cdot \frac{du}{2}" />

<BlockMath math="= \frac{1}{2} \int u^4 \, du" />

<BlockMath math="= \frac{1}{2} \cdot \frac{u^5}{5} + C" />

<BlockMath math="= \frac{1}{10} u^5 + C" />
</MathContainer>

Substitute back <InlineMath math="u = x^2 + 6x + 8" />:

<BlockMath math="F(x) = \frac{1}{10}(x^2 + 6x + 8)^5 + C" />

#### Determining the Constant C

Use the condition <InlineMath math="F(-3) = 2" />:

<BlockMath math="F(-3) = \frac{1}{10}((-3)^2 + 6(-3) + 8)^5 + C = 2" />

Calculate the value inside the parentheses:

<MathContainer>
<BlockMath math="(-3)^2 + 6(-3) + 8 = 9 - 18 + 8 = -1" />
</MathContainer>

Substitute:

<MathContainer>
<BlockMath math="\frac{1}{10}(-1)^5 + C = 2" />

<BlockMath math="\frac{1}{10}(-1) + C = 2" />

<BlockMath math="-\frac{1}{10} + C = 2" />

<BlockMath math="C = 2 + \frac{1}{10}" />

<BlockMath math="C = \frac{20}{10} + \frac{1}{10}" />

<BlockMath math="C = \frac{21}{10}" />
</MathContainer>

#### Function F(x)

Substitute the value of <InlineMath math="C" /> into <InlineMath math="F(x)" />:

<BlockMath math="F(x) = \frac{1}{10}(x^2 + 6x + 8)^5 + \frac{21}{10}" />

Therefore, the function <InlineMath math="F(x)" /> is <InlineMath math="\frac{1}{10}(x^2 + 6x + 8)^5 + \frac{21}{10}" />.
