Diketahui garis <InlineMath math="g_1" /> melalui titik <InlineMath math="A(-1, 1)" /> dan <InlineMath math="B(2, 3)" />, serta persamaan garis <InlineMath math="g_2" />: <InlineMath math="mx + ny - 7 = 0" />.

#### Menghitung Gradien Garis gâ‚

Gradien garis yang melalui dua titik dapat dihitung dengan:

<BlockMath math="m = \frac{y_2 - y_1}{x_2 - x_1}" />

Untuk garis <InlineMath math="g_1" /> yang melalui <InlineMath math="A(-1, 1)" /> dan <InlineMath math="B(2, 3)" />:

<MathContainer>
<BlockMath math="m_1 = \frac{3 - 1}{2 - (-1)}" />

<BlockMath math="= \frac{2}{3}" />
</MathContainer>

Jadi, gradien garis <InlineMath math="g_1" /> adalah <InlineMath math="m_1 = \frac{2}{3}" />.

#### Menentukan Gradien Garis gâ‚‚

Persamaan garis <InlineMath math="g_2" />: <InlineMath math="mx + ny - 7 = 0" />.

Ubah ke bentuk <InlineMath math="y = mx + c" />:

<MathContainer>
<BlockMath math="mx + ny - 7 = 0" />

<BlockMath math="ny = -mx + 7" />

<BlockMath math="y = -\frac{m}{n}x + \frac{7}{n}" />
</MathContainer>

Jadi, gradien garis <InlineMath math="g_2" /> adalah <InlineMath math="m_2 = -\frac{m}{n}" />.

#### Evaluasi Pernyataan (1)

Pernyataan: Garis <InlineMath math="g_1" /> dan <InlineMath math="g_2" /> sejajar jika <InlineMath math="m = 4" /> dan <InlineMath math="n = 6" />.

Dua garis sejajar jika gradiennya sama.

Jika <InlineMath math="m = 4" /> dan <InlineMath math="n = 6" />, maka:

<BlockMath math="m_2 = -\frac{m}{n} = -\frac{4}{6} = -\frac{2}{3}" />

Karena <InlineMath math="m_1 = \frac{2}{3}" /> dan <InlineMath math="m_2 = -\frac{2}{3}" />, maka <InlineMath math="m_1 \neq m_2" />.

Jadi, pernyataan <InlineMath math="(1)" /> bernilai salah.

#### Evaluasi Pernyataan (2)

Pernyataan: Titik <InlineMath math="B" /> melalui garis <InlineMath math="g_2" /> apabila <InlineMath math="m = 2" /> dan <InlineMath math="n = -1" />.

Titik <InlineMath math="B(2, 3)" /> melalui garis <InlineMath math="g_2" /> jika memenuhi persamaan <InlineMath math="mx + ny - 7 = 0" />.

Substitusi <InlineMath math="m = 2" />, <InlineMath math="n = -1" />, dan <InlineMath math="B(2, 3)" />:

<MathContainer>
<BlockMath math="mx + ny - 7 = 2(2) + (-1)(3) - 7" />

<BlockMath math="= 4 - 3 - 7" />

<BlockMath math="= -6" />

<BlockMath math="\neq 0" />
</MathContainer>

Karena hasilnya tidak sama dengan <InlineMath math="0" />, titik <InlineMath math="B" /> tidak melalui garis <InlineMath math="g_2" />.

Jadi, pernyataan <InlineMath math="(2)" /> bernilai salah.

#### Evaluasi Pernyataan (3)

Pernyataan: Jika <InlineMath math="g_1" /> dan <InlineMath math="g_2" /> sejajar, maka <InlineMath math="\frac{m}{n} = -\frac{3}{2}" />.

Jika kedua garis sejajar, maka <InlineMath math="m_1 = m_2" />:

<MathContainer>
<BlockMath math="\frac{2}{3} = -\frac{m}{n}" />

<BlockMath math="\frac{m}{n} = -\frac{2}{3}" />
</MathContainer>

Jadi, jika <InlineMath math="g_1" /> dan <InlineMath math="g_2" /> sejajar, maka <InlineMath math="\frac{m}{n} = -\frac{2}{3}" />, bukan <InlineMath math="-\frac{3}{2}" />.

Jadi, pernyataan <InlineMath math="(3)" /> bernilai salah.

#### Evaluasi Pernyataan (4)

Pernyataan: Jika <InlineMath math="m = 3" /> dan <InlineMath math="n = 2" />, maka <InlineMath math="g_1" /> dan <InlineMath math="g_2" /> saling tegak lurus.

Dua garis saling tegak lurus jika hasil kali gradiennya sama dengan <InlineMath math="-1" />, yaitu <InlineMath math="m_1 \times m_2 = -1" />.

Jika <InlineMath math="m = 3" /> dan <InlineMath math="n = 2" />, maka:

<BlockMath math="m_2 = -\frac{m}{n} = -\frac{3}{2}" />

Periksa apakah <InlineMath math="m_1 \times m_2 = -1" />:

<MathContainer>
<BlockMath math="m_1 \times m_2 = \frac{2}{3} \times \left(-\frac{3}{2}\right)" />

<BlockMath math="= -\frac{6}{6}" />

<BlockMath math="= -1" />
</MathContainer>

Karena <InlineMath math="m_1 \times m_2 = -1" />, maka <InlineMath math="g_1" /> dan <InlineMath math="g_2" /> saling tegak lurus.

Jadi, pernyataan <InlineMath math="(4)" /> bernilai benar.
