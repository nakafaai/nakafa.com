Given the recursive formula for the sequence:

<BlockMath math="a_{n+1} = 4a_n + 1" />

We are asked to find the value of <InlineMath math="a_3" /> given <InlineMath math="a_4 = 85" />.

Use the formula for <InlineMath math="n = 3" />:

<MathContainer>
  <BlockMath math="a_{3+1} = 4a_3 + 1" />
  <BlockMath math="a_4 = 4a_3 + 1" />
</MathContainer>

Substitute the value <InlineMath math="a_4 = 85" />:

<MathContainer>
  <BlockMath math="85 = 4a_3 + 1" />
  <BlockMath math="85 - 1 = 4a_3" />
  <BlockMath math="84 = 4a_3" />
</MathContainer>

Divide both sides by <InlineMath math="4" />:

<MathContainer>
  <BlockMath math="a_3 = \frac{84}{4}" />
  <BlockMath math="a_3 = 21" />
</MathContainer>

Thus, the value of <InlineMath math="a_3" /> is <InlineMath math="21" />.
