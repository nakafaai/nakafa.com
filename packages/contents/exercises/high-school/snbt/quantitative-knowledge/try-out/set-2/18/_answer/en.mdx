The first step is to find the value of <InlineMath math="n" /> using the information <InlineMath math="f(-2) = \frac{1}{2}" />.

#### Finding the value of n

Substitute <InlineMath math="x = -2" /> into the function equation:

<MathContainer>
  <BlockMath math="f(-2) = \frac{1}{\sqrt{(-2)^2 - (-2) + n}}" />
  <BlockMath math="\frac{1}{2} = \frac{1}{\sqrt{4 + 2 + n}}" />
  <BlockMath math="\frac{1}{2} = \frac{1}{\sqrt{6 + n}}" />
</MathContainer>

Square both sides:

<MathContainer>
  <BlockMath math="\left(\frac{1}{2}\right)^2 = \left(\frac{1}{\sqrt{6 + n}}\right)^2" />
  <BlockMath math="\frac{1}{4} = \frac{1}{6 + n}" />
  <BlockMath math="4 = 6 + n" />
  <BlockMath math="n = -2" />
</MathContainer>

So, the function <InlineMath math="f(x)" /> is:

<BlockMath math="f(x) = \frac{1}{\sqrt{x^2 - x - 2}}" />

Now let's check the truth of each statement.

#### Statement 1

<InlineMath math="n" /> is an odd number.
Since <InlineMath math="n = -2" /> (even), statement 1 is **False**.

#### Statement 2

<InlineMath math="0" /> is a member of the range of function <InlineMath math="f(x)" />.
The function <InlineMath math="f(x) = \frac{1}{\sqrt{\dots}}" /> is a fraction with a numerator of 1. This fraction value will never reach zero because the numerator is non-zero.
So, <InlineMath math="0" /> is not in the range.
Statement 2 is **False**.

#### Statement 3

The function <InlineMath math="f" /> has a value of <InlineMath math="\frac{1}{4}" /> when <InlineMath math="x = -4" />.
Calculate <InlineMath math="f(-4)" />:

<MathContainer>
  <BlockMath math="f(-4) = \frac{1}{\sqrt{(-4)^2 - (-4) - 2}}" />
  <BlockMath math="= \frac{1}{\sqrt{16 + 4 - 2}}" />
  <BlockMath math="= \frac{1}{\sqrt{18}}" />
  <BlockMath math="= \frac{1}{3\sqrt{2}}" />
</MathContainer>

Since <InlineMath math="\frac{1}{3\sqrt{2}} \neq \frac{1}{4}" />, statement 3 is **False**.

#### Statement 4

The domain of function <InlineMath math="f" /> is <InlineMath math="x < -1 \cup x > 2" />.
For the function to be defined, the denominator must not be zero and the value inside the square root must be positive.
Since the root is in the denominator, the condition is:

<BlockMath math="x^2 - x - 2 > 0" />

Factor the quadratic inequality:

<BlockMath math="(x - 2)(x + 1) > 0" />

The zeroes are <InlineMath math="x = 2" /> and <InlineMath math="x = -1" />.
Testing the number line, we find the positive region is in the interval <InlineMath math="x < -1" /> or <InlineMath math="x > 2" />.

So, the domain of <InlineMath math="f" /> is <InlineMath math="\{x \mid x < -1 \text{ or } x > 2\}" />.
Statement 4 is **True**.

#### Conclusion

There is only **1** true statement, which is statement 4.
