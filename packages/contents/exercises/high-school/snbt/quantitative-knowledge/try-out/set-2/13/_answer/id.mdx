Diketahui rumus rekursif barisan bilangan:

<BlockMath math="a_{n+1} = 4a_n + 1" />

Kita diminta mencari nilai <InlineMath math="a_3" /> jika diketahui <InlineMath math="a_4 = 85" />.

Gunakan rumus tersebut untuk <InlineMath math="n = 3" />:

<MathContainer>
  <BlockMath math="a_{3+1} = 4a_3 + 1" />
  <BlockMath math="a_4 = 4a_3 + 1" />
</MathContainer>

Substitusikan nilai <InlineMath math="a_4 = 85" />:

<MathContainer>
  <BlockMath math="85 = 4a_3 + 1" />
  <BlockMath math="85 - 1 = 4a_3" />
  <BlockMath math="84 = 4a_3" />
</MathContainer>

Bagi kedua ruas dengan <InlineMath math="4" />:

<MathContainer>
  <BlockMath math="a_3 = \frac{84}{4}" />
  <BlockMath math="a_3 = 21" />
</MathContainer>

Jadi, nilai <InlineMath math="a_3" /> adalah <InlineMath math="21" />.
