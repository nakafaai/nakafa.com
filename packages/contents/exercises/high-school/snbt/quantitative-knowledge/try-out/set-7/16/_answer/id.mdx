Misalkan gaji Farhat adalah <InlineMath math="F" /> dan gaji Abas adalah <InlineMath math="A" />.

Diketahui perbandingan gaji awal mereka adalah <InlineMath math="2 : 3" />:

<MathContainer>
  <BlockMath math="F : A = 2 : 3 \Rightarrow F = \frac{2}{3}A" />
</MathContainer>

Setelah keduanya mendapatkan kenaikan gaji sebesar <InlineMath math="40.000" /> rupiah, perbandingannya menjadi <InlineMath math="5 : 7" />. Untuk mempermudah perhitungan, kita akan menggunakan angka dalam ribuan (hilangkan 3 nol terakhir sementara), sehingga kenaikannya adalah <InlineMath math="40" />.

<MathContainer>
  <BlockMath math="(F + 40) : (A + 40) = 5 : 7" />
</MathContainer>

Kita dapat menuliskan persamaan ini sebagai:

<MathContainer>
  <BlockMath math="\frac{F + 40}{A + 40} = \frac{5}{7}" />
</MathContainer>

Kalikan silang persamaan tersebut:

<MathContainer>
  <BlockMath math="7(F + 40) = 5(A + 40)" />
</MathContainer>

Substitusikan <InlineMath math="F = \frac{2}{3}A" /> ke dalam persamaan:

<MathContainer>
  <BlockMath math="7\left(\frac{2}{3}A + 40\right) = 5(A + 40)" />
  <BlockMath math="\frac{14}{3}A + 280 = 5A + 200" />
</MathContainer>

Kumpulkan suku yang mengandung <InlineMath math="A" /> di satu sisi dan konstanta di sisi lain:

<MathContainer>
  <BlockMath math="280 - 200 = 5A - \frac{14}{3}A" />
  <BlockMath math="80 = \frac{15}{3}A - \frac{14}{3}A" />
  <BlockMath math="80 = \frac{1}{3}A" />
</MathContainer>

Kalikan kedua sisi dengan 3 untuk mendapatkan nilai <InlineMath math="A" />:

<MathContainer>
  <BlockMath math="A = 80 \times 3" />
  <BlockMath math="A = 240" />
</MathContainer>

Karena kita menggunakan satuan ribuan, maka gaji Abas sebenarnya adalah <InlineMath math="240.000" /> rupiah.

Jadi, gaji Abas sekarang adalah <InlineMath math="240.000" /> rupiah.
