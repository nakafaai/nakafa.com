Diketahui fungsi <InlineMath math="y = -x^2 - 2x + 8" />. Kita akan mencari persamaan garis singgung di titik dengan absis <InlineMath math="x = -2" />.

#### Mencari Nilai Ordinat

Substitusikan <InlineMath math="x = -2" /> ke dalam persamaan kurva:

<MathContainer>
  <BlockMath math="y = -(-2)^2 - 2(-2) + 8" />
  <BlockMath math="y = -4 + 4 + 8" />
  <BlockMath math="y = 8" />
</MathContainer>

Jadi, titik singgungnya adalah <InlineMath math="(-2, 8)" />.

#### Mencari Gradien Garis Singgung

Gradien garis singgung <InlineMath math="m" /> adalah nilai turunan pertama <InlineMath math="y'" /> pada absis titik singgung.

<MathContainer>
  <BlockMath math="y' = -2x - 2" />
</MathContainer>

Substitusikan <InlineMath math="x = -2" />:

<MathContainer>
  <BlockMath math="m = -2(-2) - 2" />
  <BlockMath math="m = 4 - 2" />
  <BlockMath math="m = 2" />
</MathContainer>

#### Menentukan Persamaan Garis Singgung

Gunakan rumus persamaan garis yang melalui titik <InlineMath math="(x_1, y_1)" /> dengan gradien <InlineMath math="m" />:

<MathContainer>
  <BlockMath math="y - y_1 = m(x - x_1)" />
</MathContainer>

Substitusikan <InlineMath math="x_1 = -2" />, <InlineMath math="y_1 = 8" />, dan <InlineMath math="m = 2" />:

<MathContainer>
  <BlockMath math="y - 8 = 2(x - (-2))" />
  <BlockMath math="y - 8 = 2(x + 2)" />
  <BlockMath math="y - 8 = 2x + 4" />
  <BlockMath math="y = 2x + 4 + 8" />
  <BlockMath math="y = 2x + 12" />
</MathContainer>

Jadi, persamaan garis singgungnya adalah <InlineMath math="y = 2x + 12" />.
