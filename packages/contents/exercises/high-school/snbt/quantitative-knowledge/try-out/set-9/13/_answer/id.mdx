#### Sifat Rata-rata dan Jangkauan

-   **Rata-rata**: Nilainya akan berubah jika setiap data ditambah, dikurangi, dikali, atau dibagi.
-   **Jangkauan**: Nilainya **hanya** akan berubah jika setiap data dikali atau dibagi (tidak berubah jika ditambah atau dikurangi).

#### Menentukan Nilai <InlineMath math="b" />

Jangkauan awal adalah <InlineMath math="12" />, kemudian nilai setiap data dikurangi dengan <InlineMath math="a" /> dan dibagi dengan <InlineMath math="b" />. Karena nilai jangkauan hanya akan berubah saat dikali atau dibagi, maka nilai jangkauan baru menjadi:

<MathContainer>
  <BlockMath math="j_{baru} = \frac{j_{awal}}{b}" />
  <BlockMath math="6 = \frac{12}{b}" />
  <BlockMath math="b = \frac{12}{6}" />
  <BlockMath math="b = 2" />
</MathContainer>

#### Menentukan Nilai <InlineMath math="a" />

Diketahui rata-rata awal <InlineMath math="24" />, kemudian setiap data dikurangi dengan <InlineMath math="a" /> dan dibagi dengan <InlineMath math="b" />, sehingga rata-rata baru menjadi:

<MathContainer>
  <BlockMath math="\bar{x}_{baru} = \frac{\bar{x}_{awal} - a}{b}" />
  <BlockMath math="10 = \frac{24 - a}{b}" />
</MathContainer>

Substitusikan nilai <InlineMath math="b = 2" />:

<MathContainer>
  <BlockMath math="10 = \frac{24 - a}{2}" />
  <BlockMath math="20 = 24 - a" />
  <BlockMath math="a = 24 - 20" />
  <BlockMath math="a = 4" />
</MathContainer>

Jadi, nilai <InlineMath math="a = 4" /> dan <InlineMath math="b = 2" />.
