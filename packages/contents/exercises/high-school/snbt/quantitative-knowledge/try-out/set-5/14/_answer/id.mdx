Diketahui fungsi kuadrat <InlineMath math="y = -2x^2 - 5x + 7" />. Kita diminta mencari persamaan garis singgung di titik dengan absis <InlineMath math="x_1 = 2" />.

Langkah pertama, kita cari ordinat (<InlineMath math="y_1" />) titik singgung dengan mensubstitusikan <InlineMath math="x_1 = 2" /> ke persamaan kurva:
<MathContainer>
  <BlockMath math="y_1 = -2(2)^2 - 5(2) + 7" />
  <BlockMath math="y_1 = -2(4) - 10 + 7" />
  <BlockMath math="y_1 = -8 - 10 + 7" />
  <BlockMath math="y_1 = -11" />
</MathContainer>
Jadi, titik singgungnya adalah <InlineMath math="(2, -11)" />.

Langkah kedua, kita cari gradien garis singgung (<InlineMath math="m" />) dengan menggunakan turunan pertama (<InlineMath math="y'" />):
<MathContainer>
  <BlockMath math="y' = -4x - 5" />
</MathContainer>
Substitusikan <InlineMath math="x = 2" /> ke turunan pertama untuk mendapatkan gradien:
<MathContainer>
  <BlockMath math="m = -4(2) - 5" />
  <BlockMath math="m = -8 - 5" />
  <BlockMath math="m = -13" />
</MathContainer>

Langkah ketiga, kita tentukan persamaan garis singgung dengan rumus <InlineMath math="y - y_1 = m(x - x_1)" />:
<MathContainer>
  <BlockMath math="y - (-11) = -13(x - 2)" />
  <BlockMath math="y + 11 = -13x + 26" />
</MathContainer>
Pindahkan semua suku ke ruas kiri agar berbentuk <InlineMath math="Ax + By + C = 0" />:
<MathContainer>
  <BlockMath math="13x + y + 11 - 26 = 0" />
  <BlockMath math="13x + y - 15 = 0" />
</MathContainer>

Jadi, persamaan garis singgungnya adalah <InlineMath math="13x + y - 15 = 0" />.
