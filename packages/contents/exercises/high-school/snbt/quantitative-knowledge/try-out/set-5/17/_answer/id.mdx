Misalkan uang Andi mula-mula adalah <InlineMath math="A" />, uang Budi adalah <InlineMath math="B" />, dan uang Cici adalah <InlineMath math="C" />.

Diketahui uang Andi empat kali dari uang Budi, sehingga kita dapat menuliskan persamaan:

<MathContainer>
  <BlockMath math="A = 4B \implies B = \frac{1}{4}A" />
</MathContainer>

Jika Andi menyumbang Rp4.000,00, maka sisa uang Andi adalah <InlineMath math="A - 4.000" />.

Jika Budi belanja Rp1.000,00, maka sisa uang Budi adalah <InlineMath math="B - 1.000" />.

Diketahui sisa uang Andi sama dengan sisa uang Budi ditambah uang Cici. Maka persamaannya adalah:

<MathContainer>
  <BlockMath math="A - 4.000 = (B - 1.000) + C" />
</MathContainer>

Selanjutnya, kita substitusikan <InlineMath math="B = \frac{1}{4}A" /> ke dalam persamaan tersebut untuk mencari nilai <InlineMath math="A" />:

<MathContainer>
  <BlockMath math="A - 4.000 = \left(\frac{1}{4}A - 1.000\right) + C" />
  <BlockMath math="A - \frac{1}{4}A = 4.000 - 1.000 + C" />
  <BlockMath math="\frac{3}{4}A = 3.000 + C" />
</MathContainer>

Kalikan kedua ruas dengan <InlineMath math="\frac{4}{3}" />:

<MathContainer>
  <BlockMath math="A = \frac{4}{3}(3.000 + C)" />
  <BlockMath math="A = 4.000 + \frac{4}{3}C" />
  <BlockMath math="A = \frac{4}{3}C + 4.000" />
</MathContainer>

Jadi, uang Andi mula-mula adalah <InlineMath math="\frac{4}{3}C + \text{Rp}4.000,00" />.
