Kita analisis setiap pernyataan berdasarkan grafik:

1. **Pernyataan <InlineMath math="(1)" />**: Grafik memotong sumbu-<InlineMath math="y" /> di <InlineMath math="4" /> dan <InlineMath math="-2" />. Jadi titik potongnya adalah <InlineMath math="(0, 4)" /> dan <InlineMath math="(0, -2)" />. Pernyataan <InlineMath math="(1)" /> **BENAR**.

2. **Pernyataan <InlineMath math="(2)" />**:
   Karena grafik memotong sumbu-<InlineMath math="y" /> di <InlineMath math="y = 4" /> dan <InlineMath math="y = -2" />, maka persamaannya berbentuk:
   <MathContainer>
     <BlockMath math="x = a(y - 4)(y + 2)" />
   </MathContainer>
   Grafik melalui titik <InlineMath math="(8, 0)" /> (titik potong sumbu-<InlineMath math="x" />). Substitusikan ke persamaan:
   <MathContainer>
     <BlockMath math="8 = a(0 - 4)(0 + 2)" />
     <BlockMath math="8 = a(-4)(2)" />
     <BlockMath math="8 = -8a \implies a = -1" />
   </MathContainer>
   Jadi persamaan fungsinya adalah:
   <MathContainer>
     <BlockMath math="x = -(y - 4)(y + 2)" />
     <BlockMath math="x = -(y^2 - 2y - 8)" />
     <BlockMath math="x = -y^2 + 2y + 8" />
   </MathContainer>
   Mengubah ke bentuk kuadrat sempurna (vertex form):
   <MathContainer>
     <BlockMath math="x = -(y^2 - 2y + 1 - 1 - 8)" />
     <BlockMath math="x = -((y - 1)^2 - 9)" />
     <BlockMath math="x = -(y - 1)^2 + 9" />
   </MathContainer>
   Atau dalam notasi fungsi <InlineMath math="f(y)" />:
   <MathContainer>
     <BlockMath math="f(y) = -(y - 1)^2 + 9" />
   </MathContainer>
   Pernyataan <InlineMath math="(2)" /> **BENAR**.

3. **Pernyataan <InlineMath math="(3)" />**:
   Sumbu simetri parabola horizontal adalah <InlineMath math="y_p" /> (ordinat puncak).
   Dari bentuk <InlineMath math="x = a(y - y_p)^2 + x_p" />, kita dapatkan <InlineMath math="y_p = 1" />.
   Atau dari titik tengah akar-akarnya:
   <MathContainer>
     <BlockMath math="y = \frac{4 + (-2)}{2} = \frac{2}{2} = 1" />
   </MathContainer>
   Jadi sumbu simetri adalah <InlineMath math="y = 1" />. Pernyataan <InlineMath math="(3)" /> **BENAR**.

4. **Pernyataan <InlineMath math="(4)" />**:
   Titik potong dengan sumbu-<InlineMath math="x" /> terjadi saat <InlineMath math="y = 0" />.
   Dari grafik terlihat jelas memotong di <InlineMath math="x = 8" />.
   Secara perhitungan: <InlineMath math="x = -(0 - 1)^2 + 9 = -1 + 9 = 8" />.
   Jadi titik potongnya <InlineMath math="(8, 0)" />. Pernyataan <InlineMath math="(4)" /> **BENAR**.

Kesimpulan: Semua pernyataan <InlineMath math="(1)" />, <InlineMath math="(2)" />, <InlineMath math="(3)" />, dan <InlineMath math="(4)" /> adalah benar.
