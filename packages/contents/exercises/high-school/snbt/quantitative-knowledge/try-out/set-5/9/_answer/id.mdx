Kita periksa kebenaran masing-masing pernyataan berdasarkan fungsi yang diberikan pada grafik:
Fungsi eksponen: <InlineMath math="y = 2^x - 2" />
Fungsi linear: <InlineMath math="y = -2x + 4" />

#### Memeriksa Titik A

Titik <InlineMath math="A" /> adalah titik potong kurva eksponen dengan sumbu Y (<InlineMath math="x=0" />).
<MathContainer>
  <BlockMath math="y = 2^0 - 2 = 1 - 2 = -1" />
</MathContainer>
Jadi, <InlineMath math="A(0, -1)" />. Pernyataan <InlineMath math="(1)" /> **BENAR**.

#### Memeriksa Titik B

Titik <InlineMath math="B" /> adalah titik potong kurva eksponen dengan sumbu X (<InlineMath math="y=0" />).
<MathContainer>
  <BlockMath math="0 = 2^x - 2 \Rightarrow 2^x = 2 \Rightarrow x = 1" />
</MathContainer>
Jadi, <InlineMath math="B(1, 0)" />. Pernyataan <InlineMath math="(2)" /> **BENAR**.

#### Memeriksa Titik C

Titik <InlineMath math="C" /> adalah titik potong garis linear dengan sumbu X (<InlineMath math="y=0" />).
<MathContainer>
  <BlockMath math="0 = -2x + 4 \Rightarrow 2x = 4 \Rightarrow x = 2" />
</MathContainer>
Jadi, <InlineMath math="C(2, 0)" />. Pernyataan <InlineMath math="(3)" /> **BENAR**.

#### Memeriksa Titik D

Titik <InlineMath math="D" /> adalah titik potong antara kedua kurva atau titik pada garis. Pernyataan menyebutkan <InlineMath math="D(2, 2)" />.
Jika kita uji koordinat <InlineMath math="(2, 2)" /> ke dalam persamaan garis <InlineMath math="y = -2x + 4" />:
<MathContainer>
  <BlockMath math="y = -2(2) + 4 = -4 + 4 = 0" />
</MathContainer>
Hasilnya <InlineMath math="y = 0" />, bukan <InlineMath math="2" />. Jadi, titik <InlineMath math="(2, 2)" /> tidak terletak pada garis <InlineMath math="y = -2x + 4" />.
Pernyataan <InlineMath math="(4)" /> **SALAH**.

Kesimpulan: Pernyataan <InlineMath math="(1)" />, <InlineMath math="(2)" />, dan <InlineMath math="(3)" /> benar.
