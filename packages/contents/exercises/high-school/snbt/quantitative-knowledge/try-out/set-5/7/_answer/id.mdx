Kita diberikan fungsi <InlineMath math="f(x) = 2x + 3" /> dan persamaan <InlineMath math="3f^2(x) = f(x) + 7" />.
Kita akan mencari jumlah akar-akar persamaan tersebut, yaitu <InlineMath math="x_1 + x_2" />.

#### Substitusi Fungsi

Langkah pertama adalah mensubstitusikan <InlineMath math="f(x)" /> ke dalam persamaan yang diberikan.

<MathContainer>
  <BlockMath math="3(2x + 3)^2 = (2x + 3) + 7" />
</MathContainer>

#### Menyederhanakan Persamaan

Selanjutnya, kita jabarkan bentuk kuadrat dan sederhanakan persamaan menjadi bentuk umum persamaan kuadrat <InlineMath math="ax^2 + bx + c = 0" />.

<MathContainer>
  <BlockMath math="3(4x^2 + 12x + 9) = 2x + 10" />
  <BlockMath math="12x^2 + 36x + 27 = 2x + 10" />
</MathContainer>

Pindahkan semua suku ke ruas kiri:

<MathContainer>
  <BlockMath math="12x^2 + 36x - 2x + 27 - 10 = 0" />
  <BlockMath math="12x^2 + 34x + 17 = 0" />
</MathContainer>

#### Menggunakan Rumus Jumlah Akar

Dari persamaan kuadrat <InlineMath math="12x^2 + 34x + 17 = 0" />, kita peroleh nilai koefisien:
- <InlineMath math="a = 12" />
- <InlineMath math="b = 34" />
- <InlineMath math="c = 17" />

Jumlah akar-akar persamaan kuadrat dapat dihitung menggunakan rumus <InlineMath math="x_1 + x_2 = -\frac{b}{a}" />.

<MathContainer>
  <BlockMath math="x_1 + x_2 = -\frac{34}{12}" />
</MathContainer>

Sederhanakan pecahan tersebut dengan membagi pembilang dan penyebut dengan <InlineMath math="2" />:

<MathContainer>
  <BlockMath math="x_1 + x_2 = -\frac{17}{6}" />
</MathContainer>

#### Kesimpulan

Jadi, nilai <InlineMath math="x_1 + x_2" /> adalah <InlineMath math="-\frac{17}{6}" />.
