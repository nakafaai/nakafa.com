Perhatikan bahwa:

<MathContainer>
  <BlockMath math="22\frac{1}{4}\% = 22,25\%" />
</MathContainer>

Maka nilai <InlineMath math="A" /> adalah:

<MathContainer>
  <BlockMath math="A = \frac{22,25}{100} \times 25 = \frac{22,25}{4} = 5,5625" />
</MathContainer>

Dan nilai <InlineMath math="B" /> adalah:

<MathContainer>
  <BlockMath math="B = \frac{1}{8} \times 32 = 4" />
</MathContainer>

Dengan demikian, <InlineMath math="A > B" />.
