Persamaan garis singgung <InlineMath math="h" /> yaitu <InlineMath math="y = mx + c" />. Dengan <InlineMath math="y = 3" /> dan <InlineMath math="x = -1" />, maka:

<BlockMath math="\begin{aligned} 3 &= -m + c \\ c &= m + 3 \end{aligned}" />

Sehingga persamaan garis singgung dapat ditulis <InlineMath math="y = mx + (m + 3)" />. Diketahui bahwa garis <InlineMath math="h" /> memotong grafik <InlineMath math="y = -x^2" /> pada dua titik berbeda, maka:

<BlockMath math="\begin{aligned} mx + (m + 3) &= -x^2 \\ x^2 + mx + (m + 3) &= 0 \end{aligned}" />

Agar memotong di dua titik berbeda, diskriminan <InlineMath math="D > 0" />.

<BlockMath math="\begin{aligned} m^2 - 4(m + 3) &> 0 \\ m^2 - 4m - 12 &> 0 \\ (m + 2)(m - 6) &> 0 \end{aligned}" />

Pembuat nol adalah <InlineMath math="m = -2" /> dan <InlineMath math="m = 6" />. Dengan uji titik, diperoleh himpunan penyelesaian:

<BlockMath math="m < -2 \lor m > 6" />
