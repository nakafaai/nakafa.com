Given that the graph of quadratic function <InlineMath math="f(x) = ax^2 + bx + c" /> intersects the <InlineMath math="y" />-axis at point <InlineMath math="(0, 8)" /> and has a vertex at <InlineMath math="(2, -4)" />.

From point <InlineMath math="(0, 8)" />, substitute <InlineMath math="x = 0" /> and <InlineMath math="f(0) = 8" /> into the function equation

<MathContainer>
<BlockMath math="f(0) = a(0)^2 + b(0) + c" />

<BlockMath math="8 = c" />
</MathContainer>

So <InlineMath math="c = 8" />.

From the vertex <InlineMath math="(2, -4)" />, the <InlineMath math="x" />-coordinate of the vertex is <InlineMath math="x_p = -\frac{b}{2a} = 2" />, so

<MathContainer>
<BlockMath math="-\frac{b}{2a} = 2" />

<BlockMath math="-b = 4a" />

<BlockMath math="b = -4a" />
</MathContainer>

The <InlineMath math="y" />-coordinate of the vertex is <InlineMath math="f(2) = -4" />, so

<MathContainer>
<BlockMath math="f(2) = a(2)^2 + b(2) + c" />

<BlockMath math="-4 = 4a + 2b + 8" />

<BlockMath math="-12 = 4a + 2b" />
</MathContainer>

Substitute <InlineMath math="b = -4a" /> into the equation <InlineMath math="-12 = 4a + 2b" />

<MathContainer>
<BlockMath math="-12 = 4a + 2(-4a)" />

<BlockMath math="-12 = 4a - 8a" />

<BlockMath math="-12 = -4a" />

<BlockMath math="a = 3" />
</MathContainer>

From <InlineMath math="b = -4a" />, we get <InlineMath math="b = -4(3) = -12" />.

So the quadratic function equation is <InlineMath math="f(x) = 3x^2 - 12x + 8" />.

**Statement <InlineMath math="\text{(I)}" />: The complete equation of the quadratic function is <InlineMath math="f(x) = 3x^2 - 12x + 8" />**

Based on the calculation above, the quadratic function equation is <InlineMath math="f(x) = 3x^2 - 12x + 8" />. Therefore, statement <InlineMath math="\text{(I)}" /> is true.

**Statement <InlineMath math="\text{(II)}" />: The point <InlineMath math="(3, -1)" /> lies on function <InlineMath math="f" />**

Substitute <InlineMath math="x = 3" /> into the function <InlineMath math="f(x) = 3x^2 - 12x + 8" />

<MathContainer>
<BlockMath math="f(3) = 3(3)^2 - 12(3) + 8" />

<BlockMath math="= 3(9) - 36 + 8" />

<BlockMath math="= 27 - 36 + 8" />

<BlockMath math="= -1" />
</MathContainer>

So <InlineMath math="f(3) = -1" />, which means the point <InlineMath math="(3, -1)" /> lies on function <InlineMath math="f" />. Therefore, statement <InlineMath math="\text{(II)}" /> is true.

**Statement <InlineMath math="\text{(III)}" />: The value of <InlineMath math="f(-1)" /> is <InlineMath math="22" />**

Substitute <InlineMath math="x = -1" /> into the function <InlineMath math="f(x) = 3x^2 - 12x + 8" />

<MathContainer>
<BlockMath math="f(-1) = 3(-1)^2 - 12(-1) + 8" />

<BlockMath math="= 3(1) + 12 + 8" />

<BlockMath math="= 3 + 12 + 8" />

<BlockMath math="= 23" />
</MathContainer>

So <InlineMath math="f(-1) = 23" />, not <InlineMath math="22" />. Therefore, statement <InlineMath math="\text{(III)}" /> is false.

Therefore, the true statements are <InlineMath math="\text{(I)}" /> and <InlineMath math="\text{(II)}" />.
