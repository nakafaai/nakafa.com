Given function <InlineMath math="f(x) = \frac{1}{\sqrt{-x^2 + 4x + 21}}" />.

Quantity <InlineMath math="P = 7" />.

To determine quantity <InlineMath math="Q" />, we need to find the values of <InlineMath math="x" /> such that <InlineMath math="f(x)" /> is defined as a real number.

For <InlineMath math="f(x)" /> to be defined, the denominator must not be zero and the expression under the square root must be greater than zero

<MathContainer>
<BlockMath math="\sqrt{-x^2 + 4x + 21} \neq 0" />

<BlockMath math="-x^2 + 4x + 21 > 0" />
</MathContainer>

Multiply both sides by <InlineMath math="-1" /> (remember the inequality sign reverses)

<MathContainer>
<BlockMath math="x^2 - 4x - 21 < 0" />
</MathContainer>

Factor the quadratic expression

<MathContainer>
<BlockMath math="(x - 7)(x + 3) < 0" />
</MathContainer>

The roots of <InlineMath math="(x - 7)(x + 3) = 0" /> are <InlineMath math="x = 7" /> and <InlineMath math="x = -3" />.

Using a number line, the inequality <InlineMath math="(x - 7)(x + 3) < 0" /> is satisfied for <InlineMath math="-3 < x < 7" />.

So the domain of function <InlineMath math="f" /> is <InlineMath math="-3 < x < 7" />.

The integer values of <InlineMath math="x" /> that satisfy this are the integers in the interval <InlineMath math="(-3, 7)" />, which are <InlineMath math="x = -2, -1, 0, 1, 2, 3, 4, 5, 6" />.

The number of such values of <InlineMath math="x" /> is <InlineMath math="9" />, so <InlineMath math="Q = 9" />.

Since <InlineMath math="P = 7" /> and <InlineMath math="Q = 9" />, we have <InlineMath math="P < Q" />.

Therefore, quantity <InlineMath math="P" /> is less than quantity <InlineMath math="Q" />.
