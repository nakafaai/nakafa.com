Diketahui fungsi <InlineMath math="g(x) = 3x + 2" />. Kita akan mensubstitusikan fungsi ini ke dalam persamaan yang diberikan.

<MathContainer>
  <BlockMath math="2g^2(x) - g(x^2) - 5g(x) - 7 = 0" />
</MathContainer>

Pertama, kita tentukan bentuk <InlineMath math="g^2(x)" /> dan <InlineMath math="g(x^2)" />:

<MathContainer>
  <BlockMath math="g^2(x) = (3x + 2)^2 = 9x^2 + 12x + 4" />
  <BlockMath math="g(x^2) = 3(x^2) + 2 = 3x^2 + 2" />
</MathContainer>

Substitusikan bentuk-bentuk tersebut ke persamaan awal:

<MathContainer>
  <BlockMath math="2(9x^2 + 12x + 4) - (3x^2 + 2) - 5(3x + 2) - 7 = 0" />
  <BlockMath math="(18x^2 + 24x + 8) - (3x^2 + 2) - (15x + 10) - 7 = 0" />
</MathContainer>

Kelompokkan suku-suku yang sejenis:

<MathContainer>
  <BlockMath math="(18 - 3)x^2 + (24 - 15)x + (8 - 2 - 10 - 7) = 0" />
  <BlockMath math="15x^2 + 9x - 11 = 0" />
</MathContainer>

Persamaan kuadrat ini memiliki akar-akar <InlineMath math="x_1" /> dan <InlineMath math="x_2" />.
Jumlah akar-akar persamaan kuadrat <InlineMath math="ax^2 + bx + c = 0" /> adalah <InlineMath math="x_1 + x_2 = -\frac{b}{a}" />.

<MathContainer>
  <BlockMath math="x_1 + x_2 = -\frac{9}{15}" />
  <BlockMath math="x_1 + x_2 = -\frac{3}{5}" />
</MathContainer>

Jadi, nilai <InlineMath math="x_1 + x_2" /> adalah <InlineMath math="-\frac{3}{5}" />.
