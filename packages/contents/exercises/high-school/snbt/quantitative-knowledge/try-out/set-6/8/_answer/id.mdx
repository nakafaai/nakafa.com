Diketahui definisi operasi fungsi <InlineMath math="W(P@Q\#R) = 2P^2 - Q + 3\sqrt{R}" />.

Kita diminta untuk mencari nilai dari <InlineMath math="W(5@8\#4)" />.

Berdasarkan posisi variabel dalam definisi fungsi, kita dapat mengidentifikasi nilai-nilai berikut:
- <InlineMath math="P = 5" />
- <InlineMath math="Q = 8" />
- <InlineMath math="R = 4" />

Substitusikan nilai-nilai ini ke dalam persamaan fungsi:

<MathContainer>
  <BlockMath math="W(5@8\#4) = 2(5)^2 - 8 + 3\sqrt{4}" />
</MathContainer>

Lakukan perhitungan langkah demi langkah:
1. Hitung kuadrat: <InlineMath math="5^2 = 25" />
2. Hitung akar: <InlineMath math="\sqrt{4} = 2" />

Maka persamaan menjadi:

<MathContainer>
  <BlockMath math="W(5@8\#4) = 2(25) - 8 + 3(2)" />
  <BlockMath math="W(5@8\#4) = 50 - 8 + 6" />
  <BlockMath math="W(5@8\#4) = 42 + 6" />
  <BlockMath math="W(5@8\#4) = 48" />
</MathContainer>

Jadi, nilai dari <InlineMath math="W(5@8\#4)" /> adalah <InlineMath math="48" />.
