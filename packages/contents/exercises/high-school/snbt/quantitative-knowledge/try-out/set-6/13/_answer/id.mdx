Kita akan menganalisis kuantitas <InlineMath math="P" /> dan <InlineMath math="Q" /> secara terpisah.

#### Analisis Kuantitas P

Bentuk <InlineMath math="P" /> dapat disederhanakan sebagai berikut:

<MathContainer>
  <BlockMath math="P = \frac{1-x^4}{1-x^2}" />
  <BlockMath math="P = \frac{(1-x^2)(1+x^2)}{1-x^2}" />
  <BlockMath math="P = 1+x^2" />
</MathContainer>

Diketahui <InlineMath math="0 < x < 1" />. Maka:

<MathContainer>
  <BlockMath math="0 < x^2 < 1" />
  <BlockMath math="1 < 1+x^2 < 2" />
  <BlockMath math="1 < P < 2" />
</MathContainer>

#### Analisis Kuantitas Q

Diketahui <InlineMath math="Q = 2+x" />.
Karena <InlineMath math="0 < x < 1" />, maka:

<MathContainer>
  <BlockMath math="2 < 2+x < 3" />
  <BlockMath math="2 < Q < 3" />
</MathContainer>

#### Membandingkan P dan Q

Dari analisis di atas, kita peroleh:
*   <InlineMath math="1 < P < 2" />
*   <InlineMath math="2 < Q < 3" />

Terlihat jelas bahwa nilai maksimum <InlineMath math="P" /> kurang dari nilai minimum <InlineMath math="Q" />.

Jadi, <InlineMath math="P < Q" />.
