Kita akan mencari nilai bilangan yang diganti (nilai <InlineMath math="P" />) terlebih dahulu.

#### Analisis Kuantitas P

Misalkan ketiga bilangan tersebut adalah <InlineMath math="a, b," /> dan <InlineMath math="c" />.
Diketahui rata-rata awal adalah <InlineMath math="24" />:

<MathContainer>
  <BlockMath math="\frac{a + b + c}{3} = 24" />
  <BlockMath math="a + b + c = 72" />
</MathContainer>

Misalkan bilangan <InlineMath math="c" /> diganti dengan angka <InlineMath math="14" />.
Rata-rata baru berkurang <InlineMath math="3" /> dari rata-rata awal, sehingga rata-rata baru menjadi <InlineMath math="24 - 3 = 21" />.

<MathContainer>
  <BlockMath math="\frac{a + b + 14}{3} = 21" />
  <BlockMath math="a + b + 14 = 63" />
  <BlockMath math="a + b = 63 - 14" />
  <BlockMath math="a + b = 49" />
</MathContainer>

Kita substitusikan nilai <InlineMath math="a + b = 49" /> ke persamaan awal untuk mencari nilai <InlineMath math="c" />:

<MathContainer>
  <BlockMath math="(a + b) + c = 72" />
  <BlockMath math="49 + c = 72" />
  <BlockMath math="c = 72 - 49" />
  <BlockMath math="c = 23" />
</MathContainer>

Jadi, bilangan yang diganti adalah <InlineMath math="23" />.
Maka, <InlineMath math="P = 23" />.

#### Membandingkan P dan Q

Diketahui <InlineMath math="Q = 42" />.

<MathContainer>
  <BlockMath math="P = 23" />
  <BlockMath math="Q = 42" />
</MathContainer>

Jelas terlihat bahwa <InlineMath math="23 < 42" />.

Jadi, <InlineMath math="P < Q" />.
