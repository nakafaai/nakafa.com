Misalkan bilangan ganjil terkecil adalah <InlineMath math="a" />.
Karena bilangan ganjil berurutan memiliki selisih <InlineMath math="2" />, maka tiga bilangan tersebut adalah <InlineMath math="a, a + 2, a + 4" />.

Diketahui jumlah ketiga bilangan tersebut adalah <InlineMath math="1005" />.

<MathContainer>
  <BlockMath math="a + (a + 2) + (a + 4) = 1005" />
  <BlockMath math="3a + 6 = 1005" />
  <BlockMath math="3a = 1005 - 6" />
  <BlockMath math="3a = 999" />
  <BlockMath math="a = 333" />
</MathContainer>

Bilangan terbesar adalah <InlineMath math="a + 4" />.

<MathContainer>
  <BlockMath math="\text{Bilangan terbesar} = 333 + 4 = 337" />
</MathContainer>

Jadi, nilai <InlineMath math="P = 337" />.
Diketahui nilai <InlineMath math="Q = 401" />.

Maka, hubungan antara <InlineMath math="P" /> dan <InlineMath math="Q" /> adalah:

<BlockMath math="337 < 401 \Rightarrow P < Q" />
