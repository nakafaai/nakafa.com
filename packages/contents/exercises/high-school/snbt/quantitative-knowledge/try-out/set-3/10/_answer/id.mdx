Diketahui data: <InlineMath math="55, a, 73, b" />.
Rata-rata data tersebut adalah <InlineMath math="68" />.

<MathContainer>
  <BlockMath math="\frac{55 + a + 73 + b}{4} = 68" />
  <BlockMath math="128 + a + b = 272" />
  <BlockMath math="a + b = 272 - 128" />
  <BlockMath math="a + b = 144 \quad \ldots \text{(1)}" />
</MathContainer>

Diketahui selisih antara nilai <InlineMath math="b" /> dan <InlineMath math="a" /> adalah <InlineMath math="22" />.

<MathContainer>
  <BlockMath math="b - a = 22 \quad \ldots \text{(2)}" />
</MathContainer>

Kita eliminasi persamaan (1) dan (2).

<MathContainer>
  <BlockMath math="\begin{aligned} a + b &= 144 \\ -a + b &= 22 \end{aligned}" />
</MathContainer>

Jumlahkan kedua persamaan:

<MathContainer>
  <BlockMath math="2b = 166 \Rightarrow b = 83" />
</MathContainer>

Substitusikan <InlineMath math="b = 83" /> ke persamaan (1):

<MathContainer>
  <BlockMath math="a + 83 = 144" />
  <BlockMath math="a = 144 - 83" />
  <BlockMath math="a = 61" />
</MathContainer>

Jadi, empat bilangan tersebut adalah <InlineMath math="55, 61, 73, 83" />. Data ini sudah terurut.
Median adalah nilai tengah dari data yang telah diurutkan. Untuk 4 data, median adalah rata-rata dari data ke-2 dan ke-3.

<MathContainer>
  <BlockMath math="\text{Median} = \frac{a + 73}{2} = \frac{61 + 73}{2}" />
  <BlockMath math="= \frac{134}{2}" />
  <BlockMath math="= 67" />
</MathContainer>

Jadi, median dari keempat data tersebut adalah <InlineMath math="67" />.
