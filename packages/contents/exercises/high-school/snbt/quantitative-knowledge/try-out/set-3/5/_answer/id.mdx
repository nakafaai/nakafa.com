Diketahui:
*   Keliling (<InlineMath math="K" />) = <InlineMath math="x \text{ cm}" />
*   Panjang (<InlineMath math="p" />) = <InlineMath math="(8y + 5) \text{ cm}" />
*   Lebar (<InlineMath math="l" />) = <InlineMath math="(6y + 1) \text{ cm}" />

Rumus keliling persegi panjang adalah <InlineMath math="K = 2(p + l)" />. Kita substitusikan nilai-nilai yang diketahui ke dalam rumus tersebut.

<MathContainer>
  <BlockMath math="x = 2((8y + 5) + (6y + 1))" />
  <BlockMath math="x = 2(14y + 6)" />
  <BlockMath math="x = 28y + 12" />
</MathContainer>

Selanjutnya, kita ubah persamaan tersebut untuk mencari nilai <InlineMath math="y" />.

<MathContainer>
  <BlockMath math="28y = x - 12" />
  <BlockMath math="y = \frac{x - 12}{28}" />
  <BlockMath math="y = \frac{x}{28} - \frac{12}{28}" />
  <BlockMath math="y = \frac{x}{28} - \frac{3}{7}" />
</MathContainer>

Jadi, nilai <InlineMath math="y" /> adalah <InlineMath math="\left(\frac{x}{28} - \frac{3}{7}\right) \text{ cm}" />.
