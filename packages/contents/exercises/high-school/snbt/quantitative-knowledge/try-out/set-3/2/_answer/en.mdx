Given the operation <InlineMath math="x \ast y = (x + y + 1)x + xy" />.

First, we calculate the value of <InlineMath math="-1 \ast 2" />.

<MathContainer>
  <BlockMath math="(-1 \ast 2) = (-1 + 2 + 1)(-1) + (-1)(2)" />
  <BlockMath math="= (2)(-1) - 2" />
  <BlockMath math="= -2 - 2" />
  <BlockMath math="= -4" />
</MathContainer>

Next, substitute this result into the equation <InlineMath math="a \ast (-1 \ast 2) = -10" />, which becomes <InlineMath math="a \ast (-4) = -10" />.

<MathContainer>
  <BlockMath math="a \ast (-4) = (a + (-4) + 1)a + a(-4) = -10" />
  <BlockMath math="(a - 3)a - 4a = -10" />
  <BlockMath math="a^2 - 3a - 4a = -10" />
  <BlockMath math="a^2 - 7a + 10 = 0" />
</MathContainer>

We factor the quadratic equation to find the value of <InlineMath math="a" />.

<MathContainer>
  <BlockMath math="(a - 2)(a - 5) = 0" />
  <BlockMath math="a = 2 \quad \text{or} \quad a = 5" />
</MathContainer>

Thus, the values of <InlineMath math="a" /> that satisfy the equation are <InlineMath math="2" /> or <InlineMath math="5" />.
