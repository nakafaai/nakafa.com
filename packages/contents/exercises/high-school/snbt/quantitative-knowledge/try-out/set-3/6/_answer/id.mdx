Mari kita analisis setiap pernyataan untuk garis <InlineMath math="5x - 10y = 3" />.

**Pernyataan 1: Gradien**

Persamaan garis dalam bentuk <InlineMath math="ax + by = c" /> memiliki gradien <InlineMath math="m = -\frac{a}{b}" />.
Untuk <InlineMath math="5x - 10y = 3" />:
*   <InlineMath math="a = 5" />
*   <InlineMath math="b = -10" />

<MathContainer>
  <BlockMath math="m = -\frac{5}{-10} = \frac{1}{2}" />
</MathContainer>

Pernyataan 1 mengatakan gradiennya <InlineMath math="-2" />. Jadi, **Pernyataan 1 salah**.

**Pernyataan 2: Titik potong sumbu <InlineMath math="x" />**

Titik potong sumbu <InlineMath math="x" /> terjadi saat <InlineMath math="y = 0" />.

<MathContainer>
  <BlockMath math="5x - 10(0) = 3" />
  <BlockMath math="5x = 3" />
  <BlockMath math="x = \frac{3}{5}" />
</MathContainer>

Titik koordinatnya adalah <InlineMath math="\left(\frac{3}{5}, 0\right)" />. Jadi, **Pernyataan 2 benar**.

**Pernyataan 3 dan 4: Titik potong sumbu <InlineMath math="y" />**

Titik potong sumbu <InlineMath math="y" /> terjadi saat <InlineMath math="x = 0" />.

<MathContainer>
  <BlockMath math="5(0) - 10y = 3" />
  <BlockMath math="-10y = 3" />
  <BlockMath math="y = -\frac{3}{10}" />
</MathContainer>

Titik koordinatnya adalah <InlineMath math="\left(0, -\frac{3}{10}\right)" />.
Jadi, **Pernyataan 3 salah** dan **Pernyataan 4 benar**.

Kesimpulan: Pernyataan yang benar adalah (2) dan (4).
