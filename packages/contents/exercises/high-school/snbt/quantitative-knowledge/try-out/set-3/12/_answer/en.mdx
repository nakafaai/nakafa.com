The ratio between spoons of tea and spoons of sugar to make a cup of tea is:

<BlockMath math="\text{Tea} : \text{Sugar} = \frac{1}{2} : 1\frac{1}{2} = \frac{1}{2} : \frac{3}{2}" />

We simplify the ratio by multiplying both sides by <InlineMath math="2" />:

<BlockMath math="\text{Tea} : \text{Sugar} = 1 : 3" />

If <InlineMath math="2" /> spoons of tea are used, we can find the number of spoons of sugar (<InlineMath math="a" />) using the ratio:

<MathContainer>
  <BlockMath math="\frac{a}{2} = \frac{3}{1}" />
  <BlockMath math="a = 2 \times 3" />
  <BlockMath math="a = 6" />
</MathContainer>

Thus, the value of <InlineMath math="P = a = 6" />.
Given the value of <InlineMath math="Q = 8" />.

Therefore, the relationship between <InlineMath math="P" /> and <InlineMath math="Q" /> is:

<BlockMath math="6 < 8 \Rightarrow P < Q" />
