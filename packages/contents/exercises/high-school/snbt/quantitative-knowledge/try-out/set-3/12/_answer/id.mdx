Perbandingan antara sendok teh dan sendok gula untuk membuat secangkir teh adalah:

<BlockMath math="\text{Teh} : \text{Gula} = \frac{1}{2} : 1\frac{1}{2} = \frac{1}{2} : \frac{3}{2}" />

Kita sederhanakan perbandingan dengan mengalikan kedua sisi dengan <InlineMath math="2" />:

<BlockMath math="\text{Teh} : \text{Gula} = 1 : 3" />

Jika digunakan <InlineMath math="2" /> sendok teh, maka kita bisa mencari jumlah sendok gula (<InlineMath math="a" />) menggunakan perbandingan:

<MathContainer>
  <BlockMath math="\frac{a}{2} = \frac{3}{1}" />
  <BlockMath math="a = 2 \times 3" />
  <BlockMath math="a = 6" />
</MathContainer>

Jadi, nilai <InlineMath math="P = a = 6" />.
Diketahui nilai <InlineMath math="Q = 8" />.

Maka, hubungan antara <InlineMath math="P" /> dan <InlineMath math="Q" /> adalah:

<BlockMath math="6 < 8 \Rightarrow P < Q" />
