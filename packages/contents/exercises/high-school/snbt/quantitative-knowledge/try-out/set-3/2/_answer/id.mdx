Diketahui operasi <InlineMath math="x \ast y = (x + y + 1)x + xy" />.

Kita akan menghitung nilai <InlineMath math="-1 \ast 2" /> terlebih dahulu.

<MathContainer>
  <BlockMath math="(-1 \ast 2) = (-1 + 2 + 1)(-1) + (-1)(2)" />
  <BlockMath math="= (2)(-1) - 2" />
  <BlockMath math="= -2 - 2" />
  <BlockMath math="= -4" />
</MathContainer>

Selanjutnya, kita substitusikan hasil tersebut ke persamaan <InlineMath math="a \ast (-1 \ast 2) = -10" />, sehingga menjadi <InlineMath math="a \ast (-4) = -10" />.

<MathContainer>
  <BlockMath math="a \ast (-4) = (a + (-4) + 1)a + a(-4) = -10" />
  <BlockMath math="(a - 3)a - 4a = -10" />
  <BlockMath math="a^2 - 3a - 4a = -10" />
  <BlockMath math="a^2 - 7a + 10 = 0" />
</MathContainer>

Kita faktorkan persamaan kuadrat tersebut untuk mendapatkan nilai <InlineMath math="a" />.

<MathContainer>
  <BlockMath math="(a - 2)(a - 5) = 0" />
  <BlockMath math="a = 2 \quad \text{atau} \quad a = 5" />
</MathContainer>

Jadi, nilai <InlineMath math="a" /> yang memenuhi adalah <InlineMath math="2" /> atau <InlineMath math="5" />.
