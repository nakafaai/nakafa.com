Given the function <InlineMath math="f(x) = 2x - 1" />. We want to find the value of <InlineMath math="x_1 + x_2" /> that satisfies the equation <InlineMath math="6f(x^2) - f^2(x) = 6x^2 - x" />.

First, we find the forms of <InlineMath math="f(x^2)" /> and <InlineMath math="f^2(x)" /> or <InlineMath math="(f(x))^2" />.

<MathContainer>
  <BlockMath math="f(x^2) = 2(x^2) - 1 = 2x^2 - 1" />
  <BlockMath math="f^2(x) = (2x - 1)^2 = 4x^2 - 4x + 1" />
</MathContainer>

Next, we substitute these results into the given equation.

<MathContainer>
  <BlockMath math="6f(x^2) - f^2(x) = 6x^2 - x" />
  <BlockMath math="6(2x^2 - 1) - (4x^2 - 4x + 1) = 6x^2 - x" />
  <BlockMath math="12x^2 - 6 - 4x^2 + 4x - 1 = 6x^2 - x" />
  <BlockMath math="8x^2 + 4x - 7 = 6x^2 - x" />
</MathContainer>

We move all terms to the left side to form a quadratic equation <InlineMath math="ax^2 + bx + c = 0" />.

<MathContainer>
  <BlockMath math="8x^2 - 6x^2 + 4x + x - 7 = 0" />
  <BlockMath math="2x^2 + 5x - 7 = 0" />
</MathContainer>

From the quadratic equation <InlineMath math="2x^2 + 5x - 7 = 0" />, we get <InlineMath math="a = 2" />, <InlineMath math="b = 5" />, and <InlineMath math="c = -7" />.

The sum of the roots of the quadratic equation (<InlineMath math="x_1 + x_2" />) can be calculated using the formula <InlineMath math="-\frac{b}{a}" />.

<MathContainer>
  <BlockMath math="x_1 + x_2 = -\frac{5}{2}" />
</MathContainer>

So, the value of <InlineMath math="x_1 + x_2" /> is <InlineMath math="-\frac{5}{2}" />.
