To determine the equation of the tangent line, we need to find the gradient (<InlineMath math="m" />) and the tangent point first.

#### Determining the Gradient

The gradient is the first derivative of the curve function (<InlineMath math="y'" />).

<MathContainer>
  <BlockMath math="y = x^2 - x + 2" />
  <BlockMath math="y' = 2x - 1" />
</MathContainer>

Substitute the abscissa <InlineMath math="x = 1" /> into <InlineMath math="y'" /> to get the gradient.

<MathContainer>
  <BlockMath math="m = y'(1) = 2(1) - 1 = 1" />
</MathContainer>

#### Determining the Tangent Point

We find the ordinate value (<InlineMath math="y" />) when the abscissa <InlineMath math="x = 1" />.

<MathContainer>
  <BlockMath math="y = (1)^2 - 1 + 2" />
  <BlockMath math="y = 1 - 1 + 2 = 2" />
</MathContainer>

So, the tangent point is <InlineMath math="(1, 2)" />.

#### Determining the Equation of the Tangent Line

Use the formula for the equation of a tangent line <InlineMath math="y - y_1 = m(x - x_1)" /> with <InlineMath math="m = 1" /> and point <InlineMath math="(1, 2)" />.

<MathContainer>
  <BlockMath math="y - 2 = 1(x - 1)" />
  <BlockMath math="y - 2 = x - 1" />
  <BlockMath math="y = x - 1 + 2" />
  <BlockMath math="y = x + 1" />
</MathContainer>

So, the equation of the tangent line is <InlineMath math="y = x + 1" />.
