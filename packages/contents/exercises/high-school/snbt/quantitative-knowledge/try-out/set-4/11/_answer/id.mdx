Diketahui persamaan <InlineMath math="(f(x))^2 - 6f(x) = -8" /> dengan <InlineMath math="f(x) = 2x - 4" />.

Pertama, kita ubah persamaan tersebut menjadi bentuk umum persamaan kuadrat:

<MathContainer>
  <BlockMath math="(f(x))^2 - 6f(x) + 8 = 0" />
</MathContainer>

Kemudian kita substitusikan <InlineMath math="f(x) = 2x - 4" /> ke dalam persamaan:

<MathContainer>
  <BlockMath math="(2x - 4)^2 - 6(2x - 4) + 8 = 0" />
</MathContainer>

Uraikan bentuk kuadrat dan selesaikan persamaan:

<MathContainer>
  <BlockMath math="(4x^2 - 16x + 16) - (12x - 24) + 8 = 0" />
  <BlockMath math="4x^2 - 16x - 12x + 16 + 24 + 8 = 0" />
  <BlockMath math="4x^2 - 28x + 48 = 0" />
</MathContainer>

Dari persamaan kuadrat <InlineMath math="ax^2 + bx + c = 0" />, hasil kali akar-akar (<InlineMath math="x_1 \cdot x_2" />) dirumuskan sebagai <InlineMath math="\frac{c}{a}" />.

Diketahui:
<InlineMath math="a = 4" />
<InlineMath math="b = -28" />
<InlineMath math="c = 48" />

Maka:

<MathContainer>
  <BlockMath math="x_1 \cdot x_2 = \frac{48}{4} = 12" />
</MathContainer>

Jadi, nilai <InlineMath math="x_1 \cdot x_2" /> adalah <InlineMath math="12" />.
