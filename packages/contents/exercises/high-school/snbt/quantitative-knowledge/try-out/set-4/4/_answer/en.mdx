Given the functions <InlineMath math="f(x) = x^2 - x" /> and <InlineMath math="g(x) = 2x - 1" />.
We are asked to find the composite function <InlineMath math="(fog)(x)" />.

<MathContainer>
  <BlockMath math="(fog)(x) = f(g(x))" />
</MathContainer>

We substitute <InlineMath math="g(x)" /> into the function <InlineMath math="f(x)" />.

<MathContainer>
  <BlockMath math="(fog)(x) = f(2x - 1)" />
  <BlockMath math="(fog)(x) = (2x - 1)^2 - (2x - 1)" />
</MathContainer>

Next, we expand the quadratic form and simplify the equation.

<MathContainer>
  <BlockMath math="(fog)(x) = (4x^2 - 4x + 1) - (2x - 1)" />
  <BlockMath math="(fog)(x) = 4x^2 - 4x + 1 - 2x + 1" />
  <BlockMath math="(fog)(x) = 4x^2 - 6x + 2" />
</MathContainer>

So, the value of <InlineMath math="(fog)(x)" /> is <InlineMath math="4x^2 - 6x + 2" />.
