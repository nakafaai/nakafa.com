Untuk menentukan persamaan garis singgung, kita perlu mencari gradien (<InlineMath math="m" />) dan titik singgungnya terlebih dahulu.

#### Menentukan Gradien

Gradien adalah turunan pertama dari fungsi kurva (<InlineMath math="y'" />).

<MathContainer>
  <BlockMath math="y = x^2 - x + 2" />
  <BlockMath math="y' = 2x - 1" />
</MathContainer>

Substitusikan absis <InlineMath math="x = 1" /> ke dalam <InlineMath math="y'" /> untuk mendapatkan gradien.

<MathContainer>
  <BlockMath math="m = y'(1) = 2(1) - 1 = 1" />
</MathContainer>

#### Menentukan Titik Singgung

Kita cari nilai ordinat (<InlineMath math="y" />) saat absis <InlineMath math="x = 1" />.

<MathContainer>
  <BlockMath math="y = (1)^2 - 1 + 2" />
  <BlockMath math="y = 1 - 1 + 2 = 2" />
</MathContainer>

Jadi, titik singgungnya adalah <InlineMath math="(1, 2)" />.

#### Menentukan Persamaan Garis Singgung

Gunakan rumus persamaan garis singgung <InlineMath math="y - y_1 = m(x - x_1)" /> dengan <InlineMath math="m = 1" /> dan titik <InlineMath math="(1, 2)" />.

<MathContainer>
  <BlockMath math="y - 2 = 1(x - 1)" />
  <BlockMath math="y - 2 = x - 1" />
  <BlockMath math="y = x - 1 + 2" />
  <BlockMath math="y = x + 1" />
</MathContainer>

Jadi, persamaan garis singgungnya adalah <InlineMath math="y = x + 1" />.
