Diketahui aturan operasi:

<MathContainer>
  <BlockMath math="\beta(w \oplus x \ominus y \otimes z) = z(y + w) - \frac{z}{x}" />
</MathContainer>

Kita diminta mencari nilai dari <InlineMath math="\beta(1 \oplus 2 \ominus 3 \otimes -4)" />.

Dengan membandingkan pola soal dengan rumus yang diberikan, kita peroleh:

*   <InlineMath math="w = 1" />
*   <InlineMath math="x = 2" />
*   <InlineMath math="y = 3" />
*   <InlineMath math="z = -4" />

Selanjutnya, kita substitusikan nilai-nilai tersebut ke dalam rumus.

<MathContainer>
  <BlockMath math="\beta(1 \oplus 2 \ominus 3 \otimes -4) = -4(3 + 1) - \frac{-4}{2}" />
  <BlockMath math="= -4(4) - (-2)" />
  <BlockMath math="= -16 + 2" />
  <BlockMath math="= -14" />
</MathContainer>

Jadi, nilai dari operasi tersebut adalah <InlineMath math="-14" />.
