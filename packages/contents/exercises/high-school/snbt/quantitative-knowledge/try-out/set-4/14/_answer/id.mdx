Diketahui data terurut: <InlineMath math="1, 1, 2, x, 10, y, 17, 18" /> (total <InlineMath math="8" /> data).

Rata-rata dan median data adalah <InlineMath math="9" />.

#### Mencari Nilai x dari Median

Karena ada <InlineMath math="8" /> data (genap), median adalah rata-rata dari data ke-4 dan ke-5.

<MathContainer>
  <BlockMath math="\text{Median} = \frac{x + 10}{2} = 9" />
  <BlockMath math="x + 10 = 18" />
  <BlockMath math="x = 8" />
</MathContainer>

#### Mencari Nilai y dari Rata-rata

Total semua data dibagi banyaknya data sama dengan rata-rata.

<MathContainer>
  <BlockMath math="\bar{x} = \frac{1 + 1 + 2 + 8 + 10 + y + 17 + 18}{8} = 9" />
  <BlockMath math="\frac{57 + y}{8} = 9" />
  <BlockMath math="57 + y = 72" />
  <BlockMath math="y = 15" />
</MathContainer>

#### Menghitung Rata-rata 4 Bilangan Terakhir

Data lengkapnya: <InlineMath math="1, 1, 2, 8, 10, 15, 17, 18" />.

4 bilangan terakhir adalah: <InlineMath math="10, 15, 17, 18" />.

<MathContainer>
  <BlockMath math="P = \frac{10 + 15 + 17 + 18}{4} = \frac{60}{4} = 15" />
</MathContainer>

#### Membandingkan P dan Q

Nilai <InlineMath math="P = 15" /> dan <InlineMath math="Q = 15" />.
