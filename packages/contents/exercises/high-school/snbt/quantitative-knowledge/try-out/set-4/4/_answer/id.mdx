Diketahui fungsi <InlineMath math="f(x) = x^2 - x" /> dan <InlineMath math="g(x) = 2x - 1" />.
Kita diminta untuk mencari nilai fungsi komposisi <InlineMath math="(fog)(x)" />.

<MathContainer>
  <BlockMath math="(fog)(x) = f(g(x))" />
</MathContainer>

Kita substitusikan <InlineMath math="g(x)" /> ke dalam fungsi <InlineMath math="f(x)" />.

<MathContainer>
  <BlockMath math="(fog)(x) = f(2x - 1)" />
  <BlockMath math="(fog)(x) = (2x - 1)^2 - (2x - 1)" />
</MathContainer>

Selanjutnya, kita uraikan bentuk kuadrat dan selesaikan persamaannya.

<MathContainer>
  <BlockMath math="(fog)(x) = (4x^2 - 4x + 1) - (2x - 1)" />
  <BlockMath math="(fog)(x) = 4x^2 - 4x + 1 - 2x + 1" />
  <BlockMath math="(fog)(x) = 4x^2 - 6x + 2" />
</MathContainer>

Jadi, nilai <InlineMath math="(fog)(x)" /> adalah <InlineMath math="4x^2 - 6x + 2" />.
