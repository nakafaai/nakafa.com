Kita akan mencari panjang alas balok menggunakan rumus keliling alas persegi panjang.

#### Menggunakan Rumus Keliling Alas

Alas kardus berbentuk persegi panjang, sehingga rumus kelilingnya (<InlineMath math="K" />) adalah:

<BlockMath math="K = 2(p + l)" />

Diketahui:
- Keliling alas (<InlineMath math="K" />) = <InlineMath math="6x" />
- Lebar alas (<InlineMath math="l" />) = <InlineMath math="4x - 5" />

Kita substitusikan nilai-nilai tersebut ke dalam rumus:

<MathContainer>
  <BlockMath math="6x = 2(p + (4x - 5))" />
</MathContainer>

#### Menghitung Panjang Alas

Kita sederhanakan persamaan untuk mendapatkan nilai <InlineMath math="p" />.

Bagi kedua ruas dengan 2:

<MathContainer>
  <BlockMath math="\frac{6x}{2} = p + 4x - 5" />
  <BlockMath math="3x = p + 4x - 5" />
</MathContainer>

Pindahkan suku-suku agar <InlineMath math="p" /> berdiri sendiri:

<MathContainer>
  <BlockMath math="p = 3x - 4x + 5" />
  <BlockMath math="p = -x + 5" />
  <BlockMath math="p = 5 - x" />
</MathContainer>

Jadi, panjang alas kardus tersebut adalah <InlineMath math="(5 - x) \text{ cm}" />.
