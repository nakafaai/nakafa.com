Kita akan menghitung rata-rata massa apel Belle terlebih dahulu, kemudian menggunakan hubungan rata-rata Ferdian dan Belle untuk mencari nilai <InlineMath math="x" />.

#### Menghitung Rata-rata Massa Apel Belle

Massa apel yang dikumpulkan Belle adalah 180, 175, 162, 154, 161, dan 170.

<MathContainer>
  <BlockMath math="\bar{x}_{\text{Belle}} = \frac{180 + 175 + 162 + 154 + 161 + 170}{6}" />
  <BlockMath math="\bar{x}_{\text{Belle}} = \frac{1.002}{6} = 167" />
</MathContainer>

Rata-rata massa apel Belle adalah 167 gram.

#### Menentukan Rata-rata Massa Apel Ferdian

Diketahui rata-rata massa apel Ferdian 3 gram lebih besar dari Belle.

<MathContainer>
  <BlockMath math="\bar{x}_{\text{Ferdian}} = \bar{x}_{\text{Belle}} + 3" />
  <BlockMath math="\bar{x}_{\text{Ferdian}} = 167 + 3 = 170" />
</MathContainer>

#### Mencari Nilai <InlineMath math="x" />

Massa apel yang dikumpulkan Ferdian adalah 157, 163, 182, 171, 167, dan <InlineMath math="x" />. Rata-ratanya adalah 170.

<MathContainer>
  <BlockMath math="\bar{x}_{\text{Ferdian}} = \frac{157 + 163 + 182 + 171 + 167 + x}{6}" />
  <BlockMath math="170 = \frac{840 + x}{6}" />
</MathContainer>

Kalikan kedua ruas dengan 6:

<MathContainer>
  <BlockMath math="170 \times 6 = 840 + x" />
  <BlockMath math="1.020 = 840 + x" />
  <BlockMath math="x = 1.020 - 840" />
  <BlockMath math="x = 180" />
</MathContainer>

Jadi, nilai dari <InlineMath math="x" /> adalah 180.
