#### Membuat Model Matematika

Misalkan:
- <InlineMath math="A" /> = besar modal untuk produk <InlineMath math="A" />
- <InlineMath math="B" /> = besar modal untuk produk <InlineMath math="B" />

Diketahui total modal adalah <InlineMath math="\text{Rp}10.000.000,00" />:

<BlockMath math="A + B = 10.000.000 \quad ...(1)" />

Diketahui keuntungan total adalah <InlineMath math="\text{Rp}904.000,00" /> dengan keuntungan produk <InlineMath math="A" /> sebesar <InlineMath math="8\%" /> dan produk <InlineMath math="B" /> sebesar <InlineMath math="10\%" />:

<MathContainer>
  <BlockMath math="8\% A + 10\% B = 904.000" />
  <BlockMath math="0,08 A + 0,1 B = 904.000" />
</MathContainer>

Kalikan kedua ruas dengan <InlineMath math="10" /> untuk memudahkan perhitungan:

<BlockMath math="0,8 A + B = 9.040.000 \quad ...(2)" />

#### Menyelesaikan Sistem Persamaan

Eliminasi variabel <InlineMath math="B" /> dari persamaan <InlineMath math="(1)" /> dan <InlineMath math="(2)" />:

<MathContainer>
  <BlockMath math="A + B = 10.000.000" />
  <BlockMath math="0,8 A + B = 9.040.000" />
  <BlockMath math="\rule{5cm}{0.4pt} \, -" />
  <BlockMath math="0,2 A = 960.000" />
</MathContainer>

Hitung nilai <InlineMath math="A" />:

<MathContainer>
  <BlockMath math="A = \frac{960.000}{0,2}" />
  <BlockMath math="A = 4.800.000" />
</MathContainer>

Jadi, modal untuk produk <InlineMath math="A" /> adalah <InlineMath math="\text{Rp}4.800.000,00" />.
