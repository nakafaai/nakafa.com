#### Pemodelan Matematika

Diketahui:

- Pendapatan (<InlineMath math="y" />) = UMR + (banyak penumpang <InlineMath math="\times" /> indeks kepuasan)
- Banyak penumpang = <InlineMath math="x" />
- Indeks kepuasan (<InlineMath math="z" />) = <InlineMath math="x - 100" />
- UMR = <InlineMath math="\text{Rp}4.680.000,00" />

Maka model matematikanya adalah:

<MathContainer>
  <BlockMath math="y = \text{UMR} + (x \cdot z)" />
  <BlockMath math="y = 4.680.000 + x(x - 100)" />
  <BlockMath math="y = x^2 - 100x + 4.680.000" />
</MathContainer>

#### Perhitungan Pendapatan

Jika diketahui jumlah penumpang <InlineMath math="x = 200" />, maka substitusikan ke dalam persamaan:

<MathContainer>
  <BlockMath math="y = (200)^2 - 100(200) + 4.680.000" />
  <BlockMath math="y = 40.000 - 20.000 + 4.680.000" />
  <BlockMath math="y = 20.000 + 4.680.000" />
  <BlockMath math="y = 4.700.000" />
</MathContainer>

Jadi, pendapatan supir tersebut adalah <InlineMath math="\text{Rp}4.700.000,00" />.
