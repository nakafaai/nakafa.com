Kita mulai dengan mendefinisikan variabel untuk kecepatan dan waktu. Misalkan <InlineMath math="v_1" /> adalah kecepatan mobil pertama, maka kecepatan mobil kedua adalah <InlineMath math="v_2 = v_1 + 15" />. Diketahui selisih waktu tempuh mereka adalah 1 jam.

Karena waktu adalah jarak dibagi kecepatan (<InlineMath math="t = \frac{s}{v}" />), kita dapat menyusun persamaan berdasarkan selisih waktu tersebut:

<MathContainer>
  <BlockMath math="t_1 - t_2 = 1" />
  <BlockMath math="\frac{450}{v_1} - \frac{450}{v_1 + 15} = 1" />
</MathContainer>

Selanjutnya, kita selesaikan persamaan ini untuk mencari nilai <InlineMath math="v_1" />. Kalikan kedua ruas dengan <InlineMath math="v_1(v_1 + 15)" /> untuk menghilangkan penyebut:

<MathContainer>
  <BlockMath math="450(v_1 + 15) - 450v_1 = v_1(v_1 + 15)" />
  <BlockMath math="450v_1 + 6750 - 450v_1 = v_1^2 + 15v_1" />
  <BlockMath math="6750 = v_1^2 + 15v_1" />
  <BlockMath math="v_1^2 + 15v_1 - 6750 = 0" />
</MathContainer>

Kita faktorkan persamaan kuadrat tersebut:

<MathContainer>
  <BlockMath math="(v_1 + 90)(v_1 - 75) = 0" />
</MathContainer>

Karena kecepatan tidak mungkin negatif, maka kita ambil <InlineMath math="v_1 = 75" /> km/jam. Dengan demikian, kecepatan mobil kedua adalah:

<MathContainer>
  <BlockMath math="v_2 = 75 + 15 = 90 \text{ km/jam}" />
</MathContainer>

Terakhir, kita hitung rata-rata kecepatan kedua mobil tersebut:

<MathContainer>
  <BlockMath math="\bar{v} = \frac{v_1 + v_2}{2} = \frac{75 + 90}{2} = \frac{165}{2} = 82,5 \text{ km/jam}" />
</MathContainer>
