**Analisis Dimensi**

Diketahui balok memiliki hubungan dimensi:
- Panjang <InlineMath math="p = 2l" />
- Tinggi <InlineMath math="t = l" />
- Lebar <InlineMath math="l" />
- Diagonal ruang <InlineMath math="DF = 9" /> cm

**Perhitungan Lebar Balok**

Kita gunakan rumus diagonal ruang balok <InlineMath math="d^2 = p^2 + l^2 + t^2" />.

Substitusikan nilai yang diketahui:

<MathContainer>
  <BlockMath math="DF^2 = p^2 + l^2 + t^2" />
  <BlockMath math="9^2 = (2l)^2 + l^2 + l^2" />
  <BlockMath math="81 = 4l^2 + l^2 + l^2" />
  <BlockMath math="81 = 6l^2" />
</MathContainer>

Selesaikan persamaan untuk <InlineMath math="l" />:

<MathContainer>
  <BlockMath math="l^2 = \frac{81}{6} = \frac{27}{2}" />
  <BlockMath math="l = \sqrt{\frac{27}{2}}" />
  <BlockMath math="l = \frac{\sqrt{27}}{\sqrt{2}} = \frac{3\sqrt{3}}{\sqrt{2}}" />
</MathContainer>

Rasionalkan penyebut:

<MathContainer>
  <BlockMath math="l = \frac{3\sqrt{3}}{\sqrt{2}} \times \frac{\sqrt{2}}{\sqrt{2}}" />
  <BlockMath math="l = \frac{3\sqrt{6}}{2}" />
</MathContainer>

Jadi, lebar balok adalah <InlineMath math="\frac{3\sqrt{6}}{2}" /> cm.
