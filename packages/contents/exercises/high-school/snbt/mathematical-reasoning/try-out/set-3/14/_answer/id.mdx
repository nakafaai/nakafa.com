**Model Matematika**

Misalkan uang Amri, Beni, dan Dimas masing-masing adalah <InlineMath math="A, B," /> dan <InlineMath math="D" />. Berdasarkan informasi soal, kita dapat menyusun sistem persamaan:

<MathContainer>
  <BlockMath math="\begin{cases} A = B + 2D + 20.000 & \text{(1)} \\ A + B + D = 100.000 & \text{(2)} \\ B - D = 5.000 & \text{(3)} \end{cases}" />
</MathContainer>

**Penyelesaian Sistem Persamaan**

Kita akan mencari nilai <InlineMath math="A" />. Substitusikan persamaan (1) ke persamaan (2):

<MathContainer>
  <BlockMath math="(B + 2D + 20.000) + B + D = 100.000" />
  <BlockMath math="2B + 3D = 80.000 \quad \text{(4)}" />
</MathContainer>

Selanjutnya, kita eliminasi <InlineMath math="B" /> menggunakan persamaan (3) dan (4). Kalikan persamaan (3) dengan <InlineMath math="2" /> agar koefisien <InlineMath math="B" /> sama:

<MathContainer>
  <BlockMath math="\begin{aligned} 2B - 2D &= 10.000 \\ 2B + 3D &= 80.000 \end{aligned}" />
</MathContainer>

Kurangkan kedua persamaan tersebut:

<MathContainer>
  <BlockMath math="-5D = -70.000 \implies D = 14.000" />
</MathContainer>

Substitusikan nilai <InlineMath math="D = 14.000" /> ke persamaan (3) untuk mencari <InlineMath math="B" />:

<MathContainer>
  <BlockMath math="B - 14.000 = 5.000 \implies B = 19.000" />
</MathContainer>

Terakhir, substitusikan nilai <InlineMath math="B" /> dan <InlineMath math="D" /> ke persamaan (1) untuk mendapatkan <InlineMath math="A" />:

<MathContainer>
  <BlockMath math="A = 19.000 + 2(14.000) + 20.000" />
  <BlockMath math="A = 19.000 + 28.000 + 20.000" />
  <BlockMath math="A = 67.000" />
</MathContainer>

Jadi, uang Amri adalah <InlineMath math="\text{Rp}67.000,00" />.
