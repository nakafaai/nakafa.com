Masalah ini dapat diselesaikan dengan menggunakan konsep perbandingan senilai antara tinggi objek dan panjang bayangannya. Misalkan <InlineMath math="h_M" /> adalah tinggi Mirza.

Diketahui:
- Tinggi Nafis = <InlineMath math="1,5" /> meter
- Bayangan Nafis = <InlineMath math="4,5" /> meter
- Bayangan Mirza = <InlineMath math="6" /> meter

Kita susun persamaan perbandingannya:

<MathContainer>
  <BlockMath math="\frac{\text{Tinggi Mirza}}{\text{Tinggi Nafis}} = \frac{\text{Bayangan Mirza}}{\text{Bayangan Nafis}}" />
  <BlockMath math="\frac{h_M}{1,5} = \frac{6}{4,5}" />
  <BlockMath math="h_M = \frac{6 \times 1,5}{4,5}" />
  <BlockMath math="h_M = \frac{9}{4,5}" />
  <BlockMath math="h_M = 2" />
</MathContainer>

Jadi, tinggi Mirza adalah **<InlineMath math="2,00" /> meter**.
