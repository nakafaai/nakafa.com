To calculate the percentage increase in the number of drug users in the <InlineMath math="20-24" /> age range from 2005 to 2006, we need to identify the number of users in both years based on the table.

- Number of users in 2005 (<InlineMath math="\text{Initial}" />): <InlineMath math="5503" />
- Number of users in 2006 (<InlineMath math="\text{Final}" />): <InlineMath math="8383" />

First, we calculate the absolute increase in the number of users:

<MathContainer>
  <BlockMath math="\text{Increase} = \text{Final} - \text{Initial} = 8383 - 5503 = 2880" />
</MathContainer>

Next, we calculate the percentage increase relative to the initial number (year 2005):

<MathContainer>
  <BlockMath math="\% \text{ Increase} = \frac{\text{Increase}}{\text{Initial}} \times 100\%" />
  <BlockMath math="= \frac{2880}{5503} \times 100\%" />
  <BlockMath math="\approx 0.5233 \times 100\%" />
  <BlockMath math="= 52.33\%" />
</MathContainer>

Rounding to one decimal place, the percentage increase is <InlineMath math="52.3\%" />.
