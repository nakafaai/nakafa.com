Kita perlu menghitung persentase penurunan untuk setiap pilihan jawaban.
Rumus persentase penurunan adalah:

<MathContainer>
  <BlockMath math="\text{Persentase Penurunan} = \frac{\text{Nilai Awal} - \text{Nilai Akhir}}{\text{Nilai Awal}} \times 100\%" />
</MathContainer>

1. **Baju pada tahun 2011—2012:**
   - Nilai 2011 = 50
   - Nilai 2012 = 35
   - Penurunan = <InlineMath math="\frac{50 - 35}{50} \times 100\% = \frac{15}{50} \times 100\% = 30\%" />

2. **Jas pada tahun 2014—2015:**
   - Nilai 2014 = 60
   - Nilai 2015 = 50
   - Penurunan = <InlineMath math="\frac{60 - 50}{60} \times 100\% = \frac{10}{60} \times 100\% = 16,67\%" />

3. **Baju pada tahun 2012—2013:**
   - Nilai 2012 = 35
   - Nilai 2013 = 20
   - Penurunan = <InlineMath math="\frac{35 - 20}{35} \times 100\% = \frac{15}{35} \times 100\% \approx 42,86\%" />

4. **Jas pada tahun 2012—2013:**
   - Nilai 2012 = 55
   - Nilai 2013 = 45
   - Penurunan = <InlineMath math="\frac{55 - 45}{55} \times 100\% = \frac{10}{55} \times 100\% \approx 18,18\%" />

5. **Celana pada tahun 2013—2014:**
   - Nilai 2013 = 50
   - Nilai 2014 = 40
   - Penurunan = <InlineMath math="\frac{50 - 40}{50} \times 100\% = \frac{10}{50} \times 100\% = 20\%" />

Dari perhitungan di atas, persentase penurunan terkecil adalah **Jas pada tahun 2014—2015** yaitu sebesar **16,67%**.
