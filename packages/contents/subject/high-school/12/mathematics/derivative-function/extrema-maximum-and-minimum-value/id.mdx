## Memahami Titik Ekstrim

Dalam kehidupan sehari-hari, kita sering mencari nilai "terbaik", misalnya keuntungan terbesar, biaya terkecil, atau jarak terdekat. Dalam matematika, nilai-nilai optimal ini dikenal sebagai **nilai ekstrim**, yang terdiri dari **nilai maksimum** dan **nilai minimum**. Titik di mana nilai-nilai ini terjadi disebut **titik ekstrim**.

Titik ekstrim ini sangat erat kaitannya dengan titik stasioner, yaitu titik di mana gradien kurva sama dengan nol (<InlineMath math="f'(x)=0" />). Ada dua cara utama untuk menguji dan menentukan jenis titik stasioner, yaitu Uji Turunan Pertama dan Uji Turunan Kedua.

## Uji Turunan Pertama

Metode ini berfokus pada perubahan perilaku fungsi, yaitu naik atau turun, di sekitar titik stasioner. Dengan melihat perubahan tanda dari <InlineMath math="f'(x)" />, kita bisa mengidentifikasi jenis titik stasionernya.

-   **Nilai Balik Maksimum:** Terjadi jika fungsi berubah dari **naik menjadi turun**. Ini berarti tanda <InlineMath math="f'(x)" /> berubah dari **positif (+) menjadi negatif (-)**. Pikirkan ini seperti puncak sebuah bukit.

-   **Nilai Balik Minimum:** Terjadi jika fungsi berubah dari **turun menjadi naik**. Tanda <InlineMath math="f'(x)" /> berubah dari **negatif (-) menjadi positif (+)**. Ini seperti dasar sebuah lembah.

-   **Titik Belok Horizontal:** Terjadi jika fungsi tidak berubah arah (tetap naik atau tetap turun). Tanda <InlineMath math="f'(x)" /> tidak berubah (positif ke positif atau negatif ke negatif).

Mari kita terapkan ini pada sebuah contoh. Tentukan nilai-nilai stasioner dari fungsi <InlineMath math="f(x) = x(x-3)^2" />.

**Penyelesaian:**

**Langkah 1: Cari turunan pertama**

Kita jabarkan dulu fungsinya: <InlineMath math="f(x) = x(x^2 - 6x + 9) = x^3 - 6x^2 + 9x" />.

Turunannya adalah:

<BlockMath math="f'(x) = 3x^2 - 12x + 9" />

**Langkah 2: Temukan titik stasioner**

Atur <InlineMath math="f'(x) = 0" />.

<div className="flex flex-col gap-4">
    <BlockMath math="3x^2 - 12x + 9 = 0" />
    <BlockMath math="x^2 - 4x + 3 = 0" />
    <BlockMath math="(x-1)(x-3) = 0" />
</div>

Titik stasioner terjadi pada <InlineMath math="x = 1" /> dan <InlineMath math="x = 3" />.

**Langkah 3: Uji tanda di sekitar titik stasioner**

-   **Sekitar <InlineMath math="x = 1" />:** Untuk <InlineMath math="x < 1" /> (misal <InlineMath math="x=0" />), <InlineMath math="f'(0) = 9" /> (positif). Untuk <InlineMath math="x > 1" /> (misal <InlineMath math="x=2" />), <InlineMath math="f'(2) = -3" /> (negatif). Karena tanda berubah dari (+) ke (-), maka <InlineMath math="x=1" /> adalah titik balik maksimum.

-   **Sekitar <InlineMath math="x = 3" />:** Untuk <InlineMath math="x < 3" /> (misal <InlineMath math="x=2" />), <InlineMath math="f'(2) = -3" /> (negatif). Untuk <InlineMath math="x > 3" /> (misal <InlineMath math="x=4" />), <InlineMath math="f'(4) = 9" /> (positif). Karena tanda berubah dari (-) ke (+), maka <InlineMath math="x=3" /> adalah titik balik minimum.

**Langkah 4: Tentukan nilai stasioner**

Untuk mendapatkan nilai balik (nilai y), substitusikan absis titik stasioner (<InlineMath math="x=1" /> dan <InlineMath math="x=3" />) kembali ke fungsi *awal* <InlineMath math="f(x)" />.

-   Nilai maksimum: <InlineMath math="f(1) = 1(1-3)^2 = 4" />.

-   Nilai minimum: <InlineMath math="f(3) = 3(3-3)^2 = 0" />.

Jadi, nilai balik maksimum fungsi adalah 4 yang terjadi di titik (1,4), dan nilai balik minimumnya adalah 0 yang terjadi di titik (3,0).

## Uji Turunan Kedua

Metode ini seringkali lebih cepat karena tidak memerlukan pengujian interval. Uji ini menggunakan turunan kedua, <InlineMath math="f''(x)" />, untuk menentukan kecekungan kurva pada titik stasioner.

Misalkan <InlineMath math="f'(c) = 0" />.

-   Jika <InlineMath math="f''(c) < 0" />, maka <InlineMath math="f(c)" /> adalah **nilai balik maksimum**. Artinya, di titik itu kurva cekung ke bawah, seperti mangkuk terbalik.

-   Jika <InlineMath math="f''(c) > 0" />, maka <InlineMath math="f(c)" /> adalah **nilai balik minimum**. Artinya, di titik itu kurva cekung ke atas, seperti mangkuk terbuka.

-   Jika <InlineMath math="f''(c) = 0" />, uji ini **gagal** dan kita harus kembali menggunakan Uji Turunan Pertama. Titik ini kemungkinan adalah titik belok.

Tentukan nilai-nilai ekstrim dari <InlineMath math="f(x) = \frac{1}{3}x^3 - x^2 - 3x + 2" /> dengan uji turunan kedua.

**Penyelesaian:**

**Langkah 1: Cari turunan pertama dan kedua**

<div className="flex flex-col gap-4">
<BlockMath math="f'(x) = x^2 - 2x - 3" />
<BlockMath math="f''(x) = 2x - 2" />
</div>

**Langkah 2: Temukan titik stasioner**

<div className="flex flex-col gap-4">
    <BlockMath math="x^2 - 2x - 3 = 0" />
    <BlockMath math="(x-3)(x+1) = 0" />
</div>

Titik stasioner terjadi pada <InlineMath math="x = 3" /> dan <InlineMath math="x = -1" />.

**Langkah 3: Uji titik stasioner ke turunan kedua**

-   Untuk <InlineMath math="x = 3" />: <InlineMath math="f''(3) = 2(3) - 2 = 4" />. Karena <InlineMath math="4 > 0" />, ini adalah **nilai balik minimum**.

-   Untuk <InlineMath math="x = -1" />: <InlineMath math="f''(-1) = 2(-1) - 2 = -4" />. Karena <InlineMath math="-4 < 0" />, ini adalah **nilai balik maksimum**.

**Langkah 4: Hitung nilai ekstrim**

Substitusikan <InlineMath math="x=3" /> dan <InlineMath math="x=-1" /> ke fungsi awal untuk mendapatkan nilai ekstrimnya.

-   Nilai minimum: <InlineMath math="f(3) = \frac{1}{3}(3)^3 - (3)^2 - 3(3) + 2 = 9 - 9 - 9 + 2 = -7" />.

-   Nilai maksimum: <InlineMath math="f(-1) = \frac{1}{3}(-1)^3 - (-1)^2 - 3(-1) + 2 = -\frac{1}{3} - 1 + 3 + 2 = \frac{11}{3}" />.

Jika kita perhatikan pada visualisasi di bawah ini, kurva cekung ke bawah di titik <InlineMath math="x=-1" /> dan cekung ke atas di titik <InlineMath math="x=3" />.

### Kunci Jawaban

1.  **Penyelesaian:**

    **Langkah 1: Cari turunan pertama**

    <BlockMath math="f'(x) = 2x + 2" />
    
    **Langkah 2: Cari titik stasioner**

    Atur <InlineMath math="f'(x) = 0" />.

    <BlockMath math="2x + 2 = 0 \implies x = -1" />

    **Langkah 3: Cari nilai y**

    Substitusikan <InlineMath math="x = -1" /> ke fungsi awal.

    <BlockMath math="y = (-1)^2 + 2(-1) - 3 = 1 - 2 - 3 = -4" />

    Jadi, titik stasionernya adalah <InlineMath math="(-1, -4)" />. Dengan Uji Turunan Kedua (<InlineMath math="f''(x)=2 > 0" />), titik ini adalah titik minimum.
