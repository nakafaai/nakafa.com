## Memahami Sudut Lebih dari 90°

Pernahkah kalian memperhatikan jarum jam? Saat jarum menit bergerak dari angka 12 ke angka 6, ia membentuk sudut 180°. Bahkan dalam satu putaran penuh, jarum tersebut membentuk sudut 360°.

Dalam matematika, kita perlu memahami nilai trigonometri untuk sudut-sudut seperti ini. Tidak hanya terbatas pada sudut lancip dalam segitiga siku-siku.

## Lingkaran Satuan

Untuk memahami fungsi trigonometri sebarang sudut, kita menggunakan lingkaran satuan. Lingkaran dengan jari-jari tepat 1 satuan yang berpusat di titik <InlineMath math="O(0,0)" />.

<UnitCircle
  title="Eksplorasi Lingkaran Satuan"
  description="Geser slider untuk melihat bagaimana koordinat berubah saat sudut berputar."
  angle={30}
/>

Mari kita pahami dengan detail:

- Sudut <InlineMath math="\theta" /> selalu diukur dari sumbu x positif
- Arah positif adalah berlawanan arah jarum jam
- Setiap titik pada lingkaran memiliki koordinat <InlineMath math="(x, y)" />

**Definisi penting:**

<div className="flex flex-col gap-4">
  <BlockMath math="\sin \theta = y \text{ (koordinat vertikal)}" />
  <BlockMath math="\cos \theta = x \text{ (koordinat horizontal)}" />
  <BlockMath math="\tan \theta = \frac{y}{x} = \frac{\sin \theta}{\cos \theta}, \quad x \neq 0" />
</div>

## Mengapa Tanda Berubah di Setiap Kuadran?

Perhatikan bahwa saat titik bergerak mengelilingi lingkaran, koordinat x dan y bisa positif atau negatif. Ini yang menyebabkan tanda fungsi trigonometri berubah.

<Triangle

  size={2}
  labels={{

**Tanda di setiap kuadran:**

| Kuadran | Rentang Sudut                              | x   | y   | sin | cos | tan |
| ------- | ------------------------------------------ | --- | --- | --- | --- | --- |
| I       | <InlineMath math="0° < \theta < 90°" />    | +   | +   | +   | +   | +   |
| II      | <InlineMath math="90° < \theta < 180°" />  | -   | +   | +   | -   | -   |
| III     | <InlineMath math="180° < \theta < 270°" /> | -   | -   | -   | -   | +   |
| IV      | <InlineMath math="270° < \theta < 360°" /> | +   | -   | -   | +   | -   |

Supaya kalian tidak bingung, kita bisa mengingatnya dengan **"All Students Take Calculus"**. Di kuadran I **A**ll positif, di kuadran II hanya **S**in positif, di kuadran III hanya **T**an positif, di kuadran IV hanya **C**os positif.

## Sudut Referensi

Sudut referensi adalah sudut lancip (0° sampai 90°) yang dibentuk antara sisi terminal sudut dengan sumbu x terdekat. Konsep ini memungkinkan kita menggunakan nilai trigonometri sudut lancip yang sudah kita hafal.

<UnitCircle
  title="Memahami Sudut Referensi"
  description="Perhatikan sudut lancip yang terbentuk dengan sumbu x saat sudut berubah."
  angle={135}
/>

**Cara menentukan sudut referensi (<InlineMath math="\alpha" />):**

- Kuadran I: <InlineMath math="\alpha = \theta" />
- Kuadran II: <InlineMath math="\alpha = 180° - \theta" />
- Kuadran III: <InlineMath math="\alpha = \theta - 180°" />
- Kuadran IV: <InlineMath math="\alpha = 360° - \theta" />

## Menentukan Nilai Trigonometri

Berikut langkah sistematis untuk menentukan nilai fungsi trigonometri:

1. **Sederhanakan sudut** (jika lebih dari 360° atau negatif)
2. **Tentukan kuadran** tempat sudut berada
3. **Hitung sudut referensi**
4. **Gunakan nilai sudut referensi** dengan tanda sesuai kuadran

### Sudut di Kuadran II

**Soal:** Tentukan <InlineMath math="\sin 120°" />, <InlineMath math="\cos 120°" />, dan <InlineMath math="\tan 120°" />

**Penyelesaian:**

- Sudut 120° terletak di kuadran II (karena <InlineMath math="90° < 120° < 180°" />)
- Sudut referensi: <InlineMath math="\alpha = 180° - 120° = 60°" />
- Di kuadran II: <InlineMath math="\sin(+), \cos(-), \tan(-)" />

Menggunakan nilai sudut istimewa 60°:

<div className="flex flex-col gap-4">
  <BlockMath math="\sin 120° = +\sin 60° = \frac{\sqrt{3}}{2}" />
  <BlockMath math="\cos 120° = -\cos 60° = -\frac{1}{2}" />
  <BlockMath math="\tan 120° = -\tan 60° = -\sqrt{3}" />
</div>

### Sudut di Kuadran III

**Soal:** Tentukan nilai trigonometri untuk sudut 240°

**Penyelesaian:**

- Sudut 240° terletak di kuadran III (karena <InlineMath math="180° < 240° < 270°" />)
- Sudut referensi: <InlineMath math="\alpha = 240° - 180° = 60°" />
- Di kuadran III: <InlineMath math="\sin(-), \cos(-), \tan(+)" />

Menggunakan nilai sudut istimewa 60°:

<div className="flex flex-col gap-4">
  <BlockMath math="\sin 240° = -\sin 60° = -\frac{\sqrt{3}}{2}" />
  <BlockMath math="\cos 240° = -\cos 60° = -\frac{1}{2}" />
  <BlockMath math="\tan 240° = +\tan 60° = \sqrt{3}" />
</div>

### Sudut di Kuadran IV

**Soal:** Tentukan nilai trigonometri untuk sudut 300°

**Penyelesaian:**

- Sudut 300° terletak di kuadran IV (karena <InlineMath math="270° < 300° < 360°" />)

- Sudut referensi: <InlineMath math="\alpha = 360° - 300° = 60°" />
- Di kuadran IV: <InlineMath math="\sin(-), \cos(+), \tan(-)" />

Menggunakan nilai sudut istimewa 60°:

<div className="flex flex-col gap-4">
  <BlockMath math="\sin 300° = -\sin 60° = -\frac{\sqrt{3}}{2}" />
  <BlockMath math="\cos 300° = +\cos 60° = \frac{1}{2}" />
  <BlockMath math="\tan 300° = -\tan 60° = -\sqrt{3}" />
</div>

## Menangani Sudut Khusus

### Sudut Negatif

Ketika sudut negatif, kita bergerak searah jarum jam. Gunakan sifat:

- <InlineMath math="\sin(-\theta) = -\sin \theta" /> (fungsi ganjil)
- <InlineMath math="\cos(-\theta) = \cos \theta" /> (fungsi genap)
- <InlineMath math="\tan(-\theta) = -\tan \theta" /> (fungsi ganjil)

**Contoh:** <InlineMath math="\sin(-30°) = -\sin 30° = -\frac{1}{2}" />

### Sudut Lebih dari 360°

Gunakan sifat periodisitas. Kurangi atau tambahkan kelipatan 360° sampai sudut berada di rentang 0° hingga 360°.

**Contoh:**

- <InlineMath math="750° = 750° - 2(360°) = 750° - 720° = 30°" />
- Maka <InlineMath math="\sin 750° = \sin 30° = \frac{1}{2}" />

## Latihan

1. Tentukan nilai <InlineMath math="\sin 315°" />, <InlineMath math="\cos 315°" />, dan <InlineMath math="\tan 315°" />.

2. Hitunglah <InlineMath math="\sin(-60°) + \cos 210° - \tan(-135°)" />.

3. Jika <InlineMath math="\sin \theta = \frac{3}{5}" /> dan <InlineMath math="\theta" /> berada di kuadran II, tentukan <InlineMath math="\cos \theta" /> dan <InlineMath math="\tan \theta" />.

4. Sederhanakan <InlineMath math="\sin 840° \cdot \cos(-330°)" />.

5. Sebuah kincir angin berputar 1050° dari posisi awal. Jika posisi awal bilah berada di sumbu x positif, tentukan koordinat ujung bilah pada lingkaran satuan setelah putaran tersebut.

### Kunci Jawaban

1. Untuk sudut 315°, kita perlu menentukan kuadrannya terlebih dahulu.

   Karena <InlineMath math="270° < 315° < 360°" />, sudut berada di kuadran IV.

   Sudut referensinya adalah <InlineMath math="360° - 315° = 45°" />.

   <div className="flex flex-col gap-4">
     <BlockMath math="\sin 315° = -\sin 45° = -\frac{\sqrt{2}}{2}" />
     <BlockMath math="\cos 315° = +\cos 45° = \frac{\sqrt{2}}{2}" />
     <BlockMath math="\tan 315° = -\tan 45° = -1" />
   </div>

2. Mari kita hitung setiap suku secara terpisah. Untuk <InlineMath math="\sin(-60°)" />, gunakan sifat fungsi ganjil.

   Untuk <InlineMath math="\cos 210°" />, sudut di kuadran III dengan referensi 30°.

   Untuk <InlineMath math="\tan(-135°)" />, ubah dulu ke sudut positif.

   <div className="flex flex-col gap-4">
     <BlockMath math="\sin(-60°) = -\sin 60° = -\frac{\sqrt{3}}{2}" />
     <BlockMath math="\cos 210° = -\cos 30° = -\frac{\sqrt{3}}{2}" />
     <BlockMath math="\tan(-135°) = -\tan 135° = -(-\tan 45°) = 1" />
     <BlockMath math="\text{Hasil} = -\frac{\sqrt{3}}{2} - \frac{\sqrt{3}}{2} + 1 = -\sqrt{3} + 1" />
   </div>

3. Diketahui <InlineMath math="\sin \theta = \frac{3}{5}" /> di kuadran II.

   Gunakan identitas Pythagoras untuk mencari <InlineMath math="\cos \theta" />.

   Ingat bahwa di kuadran II, <InlineMath math="\cos" /> bernilai negatif.

   <div className="flex flex-col gap-4">
     <BlockMath math="\sin^2 \theta + \cos^2 \theta = 1" />
     <BlockMath math="\left(\frac{3}{5}\right)^2 + \cos^2 \theta = 1" />
     <BlockMath math="\cos^2 \theta = 1 - \frac{9}{25} = \frac{16}{25}" />
     <BlockMath math="\cos \theta = -\frac{4}{5} \text{ (negatif di kuadran II)}" />
     <BlockMath math="\tan \theta = \frac{\sin \theta}{\cos \theta} = \frac{3/5}{-4/5} = -\frac{3}{4}" />
   </div>

4. Pertama sederhanakan sudut-sudutnya.

   <BlockMath math="840° = 840° - 2(360°) = 120°" />

   Untuk <InlineMath math="-330°" />, tambahkan 360° menjadi 30°.

   <div className="flex flex-col gap-4">
     <BlockMath math="\sin 840° = \sin 120° = \sin(180° - 60°) = \sin 60° = \frac{\sqrt{3}}{2}" />
     <BlockMath math="\cos(-330°) = \cos 330° = \cos(360° - 30°) = \cos 30° = \frac{\sqrt{3}}{2}" />
     <BlockMath math="\sin 840° \cdot \cos(-330°) = \frac{\sqrt{3}}{2} \cdot \frac{\sqrt{3}}{2} = \frac{3}{4}" />
   </div>

5. Sudut 1050° perlu disederhanakan terlebih dahulu.

   <BlockMath math="1050° = 1050° - 2(360°) = 1050° - 720° = 330°" />

   Sudut 330° berada di kuadran IV dengan sudut referensi 30°.

   <div className="flex flex-col gap-4">
     <BlockMath math="x = \cos 330° = \cos 30° = \frac{\sqrt{3}}{2}" />
     <BlockMath math="y = \sin 330° = -\sin 30° = -\frac{1}{2}" />
     <BlockMath math="\text{Koordinat ujung bilah: } \left(\frac{\sqrt{3}}{2}, -\frac{1}{2}\right)" />
   </div>
