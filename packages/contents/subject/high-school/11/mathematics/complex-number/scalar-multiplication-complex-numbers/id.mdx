## Apa itu Perkalian Skalar?

Perkalian skalar adalah mengalikan bilangan kompleks dengan bilangan real (skalar).

<LineEquation
  title="Visualisasi Perkalian Skalar"
  description={
    <>
      Perhatikan bagaimana vektor <InlineMath math="z = 1+2i" /> berubah ketika
      dikalikan skalar <InlineMath math="c=2" /> dan{" "}
      <InlineMath math="c=-0.5" />.
    </>
  }
  cameraPosition={[0, 0, 12]}
  showZAxis={false}
  data={[
    // Vektor z asli
    {
      points: [
        { x: 0, y: 0, z: 0 },
        { x: 1, y: 2, z: 0 },
      ],
      color: getColor("SKY"),
      labels: [{ text: "z", at: 1, offset: [0.5, 0.5, 0] }],
      cone: { position: "end" },
    },
    // Vektor 2z
    {
      points: [
        { x: 0, y: 0, z: 0 },
        { x: 2, y: 4, z: 0 }, // 2 * (1+2i) = 2+4i
      ],
      color: getColor("EMERALD"),
      labels: [{ text: "2z", at: 1, offset: [0.5, 0.5, 0] }],
      cone: { position: "end" },
    },
    // Vektor -0.5z
    {
      points: [
        { x: 0, y: 0, z: 0 },
        { x: -0.5, y: -1, z: 0 }, // -0.5 * (1+2i) = -0.5 - i
      ],
      color: getColor("ROSE"),
      labels: [{ text: "-0.5z", at: 1, offset: [-0.5, -0.5, 0] }],
      cone: { position: "end" },
    },
  ]}
/>

Dari visualisasi di atas, kita bisa lihat:

- Mengalikan <InlineMath math="z" /> dengan skalar <InlineMath math="c > 1" /> (seperti 2) akan **memperpanjang** vektor <InlineMath math="z" /> searah.
- Mengalikan <InlineMath math="z" /> dengan skalar <InlineMath math="0 < c < 1" /> akan **memperpendek** vektor <InlineMath math="z" /> searah.
- Mengalikan <InlineMath math="z" /> dengan skalar <InlineMath math="c < 0" /> (seperti -0.5) akan **membalikkan arah** vektor <InlineMath math="z" /> (180 derajat) dan mengubah panjangnya sesuai nilai <InlineMath math="|c|" />.

## Definisi Matematis

Jika <InlineMath math="z = x + iy" /> adalah bilangan kompleks dan <InlineMath math="c" /> adalah skalar (bilangan real), maka perkalian skalarnya adalah:

<BlockMath math="cz = c(x + iy) = (cx) + i(cy)" />

Artinya, kita cukup kalikan skalar <InlineMath math="c" /> ke bagian real (<InlineMath math="x" />) dan bagian imajiner (<InlineMath math="y" />) secara terpisah.

### Contoh Perhitungan

Jika <InlineMath math="z = -3 + 4i" /> dan <InlineMath math="c = 3" />, maka:

<BlockMath math="cz = 3(-3 + 4i) = 3(-3) + i(3 \times 4) = -9 + 12i" />

Jika <InlineMath math="z = 5 - i" /> dan <InlineMath math="c = -2" />, maka:

<BlockMath math="cz = -2(5 - i) = -2(5) + i(-2 \times -1) = -10 + 2i" />
