## Understanding Reflection across the X-Axis

Reflection across the X-axis is a type of geometric transformation that moves every point of an object to a new position symmetrical to the X-axis. Imagine the X-axis as a flat mirror.

If a point <InlineMath math="P" /> has coordinates <InlineMath math="(x,y)" />, then its reflection, which we'll call <InlineMath math="P'" />, will have the same <InlineMath math="x" />-coordinate, but its <InlineMath math="y" />-coordinate will be the negative of the original <InlineMath math="y" /> value.

Mathematically, if the initial point is <InlineMath math="P(x,y)" />, then after reflection across the X-axis, its image is <InlineMath math="P'(x,-y)" />.

Based on the interactive visualization above, we can observe the relationship between the original points (pre-image) and their reflections (image) as follows:

- Point <InlineMath math="A(-5,2)" /> becomes <InlineMath math="A'(-5,-2)" />
- Point <InlineMath math="B(-3,1)" /> becomes <InlineMath math="B'(-3,-1)" />

- Point <InlineMath math="D(4,-2)" /> becomes <InlineMath math="D'(4,2)" />

The visible pattern is that the <InlineMath math="x" /> value remains constant, and the <InlineMath math="y" /> value changes sign (becomes its opposite).

## Property of Reflection across the X-Axis

Based on the observations above, we can formulate the property of reflection across the X-axis:

<BlockMath math="P(x,y) \xrightarrow{\text{X-axis}} P'(x,-y)" />

This means the image of point <InlineMath math="P(x,y)" /> reflected across the X-axis is <InlineMath math="P'(x,-y)" />. The X-axis in this case acts as the line <InlineMath math="y=0" />.

## Application Examples

### Reflecting a Triangle

Determine the image of triangle <InlineMath math="ABC" /> with vertices <InlineMath math="A(-1,4)" />, <InlineMath math="B(2,1)" />, and <InlineMath math="C(-2,-1)" /> reflected across the X-axis.

To determine the image of triangle <InlineMath math="ABC" />, we apply the reflection property to each of its vertices:

<div className="flex flex-col gap-4">
  <BlockMath math="A(-1,4) \xrightarrow{\text{X-axis}} A'(-1,-4)" />
  <BlockMath math="B(2,1) \xrightarrow{\text{X-axis}} B'(2,-1)" />
  <BlockMath math="C(-2,-1) \xrightarrow{\text{X-axis}} C'(-2,1)" />
</div>

Consequently, the image of triangle <InlineMath math="ABC" /> is triangle <InlineMath math="A'B'C'" /> with vertices <InlineMath math="A'(-1,-4)" />, <InlineMath math="B'(2,-1)" />, and <InlineMath math="C'(-2,1)" />.

### Reflecting a Line

If a line has the equation <InlineMath math="2x - 3y = 0" /> and is reflected across the X-axis, determine the equation of its reflected line.

**Alternative Solution:**

Let an arbitrary point <InlineMath math="P(a,b)" /> lie on the line <InlineMath math="2x - 3y = 0" />. Then, the following holds:

<BlockMath math="2a - 3b = 0" />

The point <InlineMath math="P(a,b)" /> reflected across the X-axis produces the image <InlineMath math="P'(a,-b)" />.

To obtain the equation of the reflected line, we substitute the coordinates of the image into new variables. Let <InlineMath math="x' = a" /> and <InlineMath math="y' = -b" />.

From this, we get <InlineMath math="a = x'" /> and <InlineMath math="b = -y'" />.

Substitute <InlineMath math="a = x'" /> and <InlineMath math="b = -y'" /> into the original equation <InlineMath math="2a - 3b = 0" />:

<div className="flex flex-col gap-4">
  <BlockMath math="2(x') - 3(-y') = 0" />
  <BlockMath math="2x' + 3y' = 0" />
</div>

Since <InlineMath math="x'" /> and <InlineMath math="y'" /> are arbitrary variables representing the coordinates on the reflected line, we can rewrite them as <InlineMath math="x" /> and <InlineMath math="y" />.

Thus, the equation of the reflected line is:

<div className="flex flex-col gap-4">
  <BlockMath math="2x + 3y = 0" />
  <LineEquation
    title={
      <>
        Reflection of Line <InlineMath math="2x - 3y = 0" /> across the X-Axis
      </>
    }
    description={
      <>
        The original line <InlineMath math="2x - 3y = 0" /> (lime green) and its
        reflection <InlineMath math="2x + 3y = 0" /> (magenta) after mirroring.
      </>
    }
    cameraPosition={[0, 0, 15]}
    showZAxis={false}
    data={[
      {
        // Original Line: 2x - 3y = 0  => y = (2/3)x
        points: [
          { x: -6, y: (2 / 3) * -6, z: 0 },
          { x: 6, y: (2 / 3) * 6, z: 0 },
        ],
        color: getColor("LIME"),
        labels: [{ text: "2x - 3y = 0", at: 1, offset: [0.5, 0.5, 0] }],
      },
      {
        // Reflected Line: 2x + 3y = 0 => y = -(2/3)x
        points: [
          { x: -6, y: -(2 / 3) * -6, z: 0 },
          { x: 6, y: -(2 / 3) * 6, z: 0 },
        ],
        color: getColor("MAGENTA"),
        labels: [{ text: "2x + 3y = 0", at: 1, offset: [0.5, -0.5, 0] }],
      },
    ]}
  />
</div>

This shows how the equation of a line changes after being reflected across the X-axis.
