## Memahami Pencerminan terhadap Sumbu X

Pencerminan terhadap sumbu X adalah salah satu jenis transformasi geometri yang memindahkan setiap titik pada suatu objek ke posisi baru yang simetris terhadap sumbu X. Bayangkan sumbu X sebagai cermin datar.

Jika sebuah titik <InlineMath math="P" /> memiliki koordinat <InlineMath math="(x,y)" />, maka bayangannya, kita sebut <InlineMath math="P'" />, akan memiliki koordinat <InlineMath math="x" /> yang sama, namun koordinat <InlineMath math="y" />-nya akan menjadi negatif dari nilai <InlineMath math="y" /> semula.

Secara matematis, jika titik awal adalah <InlineMath math="P(x,y)" />, maka setelah dicerminkan terhadap sumbu X, bayangannya adalah <InlineMath math="P'(x,-y)" />.

### Visualisasi Titik dan Bayangannya

Mari kita amati beberapa titik dan bayangannya setelah dicerminkan terhadap sumbu X.

Perhatikan bagaimana koordinat <InlineMath math="y" /> berubah tanda, sementara koordinat <InlineMath math="x" /> tetap.

Berdasarkan visualisasi interaktif di atas, kita dapat mengamati hubungan antara titik-titik asli (prapeta) dan bayangannya (peta) sebagai berikut:

- Titik <InlineMath math="A(-5,2)" /> menjadi <InlineMath math="A'(-5,-2)" />
- Titik <InlineMath math="B(-3,1)" /> menjadi <InlineMath math="B'(-3,-1)" />
- Titik <InlineMath math="C(1,2)" /> menjadi <InlineMath math="C'(1,-2)" />
- Titik <InlineMath math="D(4,-2)" /> menjadi <InlineMath math="D'(4,2)" />

Pola yang tampak adalah nilai <InlineMath math="x" /> tetap, dan nilai <InlineMath math="y" /> berubah tanda (menjadi lawannya).

## Sifat Pencerminan terhadap Sumbu X

Berdasarkan pengamatan di atas, kita dapat merumuskan sifat pencerminan terhadap sumbu X:

<BlockMath math="P(x,y) \xrightarrow{\text{Sumbu X}} P'(x,-y)" />

Artinya, peta dari titik <InlineMath math="P(x,y)" /> yang dicerminkan terhadap sumbu X adalah <InlineMath math="P'(x,-y)" />. Sumbu X dalam kasus ini bertindak sebagai garis <InlineMath math="y=0" />.

## Contoh Aplikasi

### Mencerminkan Segitiga

Tentukan peta dari segitiga <InlineMath math="ABC" /> dengan titik sudut <InlineMath math="A(-1,4)" />, <InlineMath math="B(2,1)" />, dan <InlineMath math="C(-2,-1)" /> yang dicerminkan terhadap sumbu X.

Untuk menentukan peta dari segitiga <InlineMath math="ABC" />, kita gunakan sifat pencerminan pada setiap titik sudutnya:

<div className="flex flex-col gap-4">
  <BlockMath math="A(-1,4) \xrightarrow{\text{Sumbu X}} A'(-1,-4)" />
  <BlockMath math="B(2,1) \xrightarrow{\text{Sumbu X}} B'(2,-1)" />
  <BlockMath math="C(-2,-1) \xrightarrow{\text{Sumbu X}} C'(-2,1)" />
</div>

Akibatnya, peta dari segitiga <InlineMath math="ABC" /> adalah segitiga <InlineMath math="A'B'C'" /> dengan titik sudut <InlineMath math="A'(-1,-4)" />, <InlineMath math="B'(2,-1)" />, dan <InlineMath math="C'(-2,1)" />.

### Mencerminkan Garis

Jika sebuah garis memiliki persamaan <InlineMath math="2x - 3y = 0" /> dicerminkan terhadap sumbu X, tentukan persamaan garis bayangannya.

**Alternatif Penyelesaian:**

Misalkan sebuah titik sembarang <InlineMath math="P(a,b)" /> terletak pada garis <InlineMath math="2x - 3y = 0" />. Maka, berlaku:

<BlockMath math="2a - 3b = 0" />

Titik <InlineMath math="P(a,b)" /> dicerminkan terhadap sumbu X menghasilkan bayangan <InlineMath math="P'(a,-b)" />.

Untuk mendapatkan persamaan garis bayangan, kita substitusikan koordinat bayangan ke dalam variabel baru. Misalkan <InlineMath math="x' = a" /> dan <InlineMath math="y' = -b" />.

Dari sini, kita peroleh <InlineMath math="a = x'" /> dan <InlineMath math="b = -y'" />.

Substitusikan <InlineMath math="a = x'" /> dan <InlineMath math="b = -y'" /> ke persamaan awal <InlineMath math="2a - 3b = 0" />:

<div className="flex flex-col gap-4">
  <BlockMath math="2(x') - 3(-y') = 0" />
  <BlockMath math="2x' + 3y' = 0" />
</div>

Karena <InlineMath math="x'" /> dan <InlineMath math="y'" /> adalah variabel sembarang yang mewakili koordinat pada garis bayangan, kita dapat menuliskannya kembali sebagai <InlineMath math="x" /> dan <InlineMath math="y" />.

Jadi, persamaan garis bayangannya adalah:

<div className="flex flex-col gap-4">
  <BlockMath math="2x + 3y = 0" />
  <LineEquation
    title={
      <>
        Pencerminan Garis <InlineMath math="2x - 3y = 0" /> terhadap Sumbu X
      </>
    }
    description={
      <>
        Garis asal <InlineMath math="2x - 3y = 0" /> (hijau limau) dan
        bayangannya <InlineMath math="2x + 3y = 0" /> (magenta) setelah
        pencerminan.
      </>
    }
    cameraPosition={[0, 0, 15]}
    showZAxis={false}
    data={[
      {
        // Garis Asal: 2x - 3y = 0  => y = (2/3)x
        points: [
          { x: -6, y: (2 / 3) * -6, z: 0 },
          { x: 6, y: (2 / 3) * 6, z: 0 },
        ],
        color: getColor("LIME"),
        labels: [{ text: "2x - 3y = 0", at: 1, offset: [0.5, 0.5, 0] }],
      },
      {
        // Garis Bayangan: 2x + 3y = 0 => y = -(2/3)x
        points: [
          { x: -6, y: -(2 / 3) * -6, z: 0 },
          { x: 6, y: -(2 / 3) * 6, z: 0 },
        ],
        color: getColor("MAGENTA"),
        labels: [{ text: "2x + 3y = 0", at: 1, offset: [0.5, -0.5, 0] }],
      },
    ]}
  />
</div>

Ini menunjukkan bagaimana persamaan sebuah garis berubah setelah dicerminkan terhadap sumbu X.
