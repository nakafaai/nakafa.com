## Memahami Translasi

Translasi, juga dikenal sebagai pergeseran, adalah jenis transformasi geometri yang memindahkan setiap titik suatu objek sejauh jarak tertentu dalam arah tertentu. Transformasi ini tidak mengubah orientasi, ukuran, atau bentuk objek; hanya posisinya yang berubah.

### Definisi Translasi

Diberikan sembarang titik <InlineMath math="P(x,y)" />. Translasi yang berkaitan dengan vektor <InlineMath math="\begin{pmatrix} a \\ b \end{pmatrix}" /> untuk titik <InlineMath math="P(x,y)" />, ditulis sebagai <InlineMath math="T_{(a,b)}(x,y)" /> atau <InlineMath math="\tau_{(a,b)}(x,y)" />, didefinisikan sebagai:

<BlockMath math="P'(x',y') = (x+a, y+b)" />

Ini berarti:

<div className="flex flex-col gap-4">
  <BlockMath math="x' = x + a" />
  <BlockMath math="y' = y + b" />
</div>

Di sini, <InlineMath math="a" /> adalah pergeseran horizontal (positif ke kanan, negatif ke kiri) dan <InlineMath math="b" /> adalah pergeseran vertikal (positif ke atas, negatif ke bawah).

## Mentranslasikan sebuah Titik

Sebuah titik <InlineMath math="(3,2)" /> ditranslasikan dengan vektor <InlineMath math="\begin{pmatrix} -2 \\ 3 \end{pmatrix}" />. Tentukan titik peta dari translasi tersebut.

Di sini, <InlineMath math="x=3" />, <InlineMath math="y=2" />, <InlineMath math="a=-2" />, dan <InlineMath math="b=3" />.

Menggunakan rumus:

<div className="flex flex-col gap-4">
  <BlockMath math="x' = x + a = 3 + (-2) = 1" />
  <BlockMath math="y' = y + b = 2 + 3 = 5" />
</div>

Jadi, bayangan titik <InlineMath math="(3,2)" /> adalah <InlineMath math="(1,5)" />.

<LineEquation
  title={
    <>
      Translasi Titik <InlineMath math="P(3,2)" /> oleh Vektor{" "}
      <InlineMath math="\begin{pmatrix} -2 \\ 3 \end{pmatrix}" />
    </>
  }
  description={
    <>
      Visualisasi translasi titik <InlineMath math="P(3,2)" /> menjadi{" "}
      <InlineMath math="P'(1,5)" /> menggunakan vektor translasi.
    </>
  }
  data={[
    {
      points: [{ x: 3, y: 2, z: 0 }],
      color: getColor("SKY"),
      showPoints: true,
      labels: [{ text: "P(3,2) - Asli", at: 0, offset: [0.3, -0.3, 0] }],
    }, // Titik Asli
    {
      points: [{ x: 1, y: 5, z: 0 }],
      color: getColor("EMERALD"),
      showPoints: true,
      labels: [{ text: "P'(1,5) - Peta", at: 0, offset: [0.3, 0.3, 0] }],
    }, // Titik Peta
    {
      points: [
        { x: 3, y: 2, z: 0 },
        { x: 1, y: 5, z: 0 },
      ],
      color: getColor("ROSE"),
      labels: [{ text: "vektor (-2,3)", at: 0, offset: [-1, 1.5, 0] }],
    }, // Vektor Translasi dari P ke P'
  ]}
  showZAxis={false}
/>

## Mentranslasikan sebuah Garis

Tentukan peta dari garis <InlineMath math="l \equiv 2x + 3y - 1 = 0" /> ditranslasikan dengan vektor <InlineMath math="\begin{pmatrix} -1 \\ 1 \end{pmatrix}" />.

Misalkan <InlineMath math="P(x,y)" /> adalah sembarang titik pada garis <InlineMath math="l" />. Jika ditranslasikan oleh vektor <InlineMath math="\begin{pmatrix} -1 \\ 1 \end{pmatrix}" />, bayangannya adalah <InlineMath math="P'(x',y')" /> di mana:

<div className="flex flex-col gap-4">
  <BlockMath math="x' = x + (-1) = x - 1 \implies x = x' + 1" />
  <BlockMath math="y' = y + 1 \implies y = y' - 1" />
</div>

Substitusikan nilai <InlineMath math="x" /> dan <InlineMath math="y" /> ini ke dalam persamaan garis <InlineMath math="l" />:

<div className="flex flex-col gap-4">
  <BlockMath math="2(x' + 1) + 3(y' - 1) - 1 = 0" />
  <BlockMath math="2x' + 2 + 3y' - 3 - 1 = 0" />
  <BlockMath math="2x' + 3y' - 2 = 0" />
</div>

Mengganti <InlineMath math="x'" /> dan <InlineMath math="y'" /> kembali ke <InlineMath math="x" /> dan <InlineMath math="y" />, persamaan garis bayangan <InlineMath math="l'" /> adalah:

<div className="flex flex-col gap-4">
  <BlockMath math="2x + 3y - 2 = 0" />
  <LineEquation
    title={
      <>
        Translasi Garis <InlineMath math="2x+3y-1=0" /> oleh Vektor{" "}
        <InlineMath math="\begin{pmatrix} -1 \\ 1 \end{pmatrix}" />
      </>
    }
    description={
      <>
        Garis asli <InlineMath math="2x+3y-1=0" /> ditranslasikan menghasilkan
        garis bayangan <InlineMath math="2x+3y-2=0" />.
      </>
    }
    data={[
      {
        // Garis Asli: 2x + 3y - 1 = 0  => y = (-2/3)x + 1/3
        points: Array.from({ length: 11 }, (_, i) => {
          const xVal = i - 5;
          return { x: xVal, y: (-2 / 3) * xVal + 1 / 3, z: 0 };
        }),
        color: getColor("PURPLE"),
        labels: [{ text: "2x+3y-1=0", at: 2, offset: [-1, -0.5, 0] }],
      },
      {
        // Garis Bayangan: 2x + 3y - 2 = 0 => y = (-2/3)x + 2/3
        points: Array.from({ length: 11 }, (_, i) => {
          const xVal = i - 5;
          return { x: xVal, y: (-2 / 3) * xVal + 2 / 3, z: 0 };
        }),
        color: getColor("PINK"),
        labels: [{ text: "2x+3y-2=0", at: 8, offset: [1, 0.5, 0] }],
      },
    ]}
    showZAxis={false}
    cameraPosition={[0, 0, 10]}
  />
</div>

## Latihan

1.  Sebuah titik <InlineMath math="(-4,4)" /> ditranslasikan dengan vektor <InlineMath math="\begin{pmatrix} -2 \\ -3 \end{pmatrix}" />. Tentukan titik peta dari translasi tersebut.
2.  Tentukan peta dari garis <InlineMath math="l' \equiv 5x - 2y + 3 = 0" /> ditranslasikan dengan vektor <InlineMath math="\begin{pmatrix} 2 \\ -1 \end{pmatrix}" />.
3.  Sebuah segitiga dengan titik sudut <InlineMath math="A(1,1)" />, <InlineMath math="B(4,1)" />, dan <InlineMath math="C(1,5)" /> ditranslasikan oleh vektor <InlineMath math="\begin{pmatrix} 2 \\ 3 \end{pmatrix}" />. Tentukan koordinat bayangan segitiga <InlineMath math="A'B'C'" />!

### Kunci Jawaban

1.  Titik <InlineMath math="(-4,4)" />, vektor <InlineMath math="\begin{pmatrix} -2 \\ -3 \end{pmatrix}" />. <InlineMath math="x=-4, y=4, a=-2, b=-3" />.

    <div className="flex flex-col gap-4">
      <BlockMath math="x' = -4 + (-2) = -6" />
      <BlockMath math="y' = 4 + (-3) = 1" />
    </div>

    Jadi, bayangan titiknya adalah <InlineMath math="(-6,1)" />.

2.  Garis <InlineMath math="5x - 2y + 3 = 0" />, vektor <InlineMath math="\begin{pmatrix} 2 \\ -1 \end{pmatrix}" />. <InlineMath math="a=2, b=-1" />.

    <div className="flex flex-col gap-4">
      <BlockMath math="x' = x + 2 \implies x = x' - 2" />
      <BlockMath math="y' = y - 1 \implies y = y' + 1" />
    </div>

    Substitusi ke persamaan garis:

    <div className="flex flex-col gap-4">
      <BlockMath math="5(x' - 2) - 2(y' + 1) + 3 = 0" />
      <BlockMath math="5x' - 10 - 2y' - 2 + 3 = 0" />
      <BlockMath math="5x' - 2y' - 9 = 0" />
    </div>

    Persamaan garis bayangan: <InlineMath math="5x - 2y - 9 = 0" />.

3.  Titik <InlineMath math="A(1,1)" />, <InlineMath math="B(4,1)" />, <InlineMath math="C(1,5)" />. Vektor <InlineMath math="\begin{pmatrix} 2 \\ 3 \end{pmatrix}" />.

    <div className="flex flex-col gap-4">
      <BlockMath math="A'(1+2, 1+3) = A'(3,4)" />
      <BlockMath math="B'(4+2, 1+3) = B'(6,4)" />
      <BlockMath math="C'(1+2, 5+3) = C'(3,8)" />
    </div>

    Koordinat bayangan segitiga adalah <InlineMath math="A'(3,4)" />, <InlineMath math="B'(6,4)" />, dan <InlineMath math="C'(3,8)" />.
