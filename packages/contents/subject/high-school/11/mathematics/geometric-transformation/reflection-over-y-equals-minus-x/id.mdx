## Memahami Pencerminan terhadap Garis y = -x

Pencerminan terhadap garis <InlineMath math="y = -x" /> adalah transformasi geometri di mana setiap titik objek dipetakan ke posisi baru, dengan garis <InlineMath math="y = -x" /> berfungsi sebagai cermin.

Garis yang menghubungkan titik asli ke bayangannya akan tegak lurus terhadap garis <InlineMath math="y = -x" />, dan jarak dari titik asli ke garis cermin sama dengan jarak dari bayangannya ke garis cermin.

### Aturan Pencerminan terhadap Garis y = -x

Jika sebuah titik <InlineMath math="P(x, y)" /> dicerminkan terhadap garis <InlineMath math="y = -x" />, maka koordinat bayangannya, <InlineMath math="P'(x', y')" />, ditentukan oleh aturan:

<div className="flex flex-col gap-4">
  <BlockMath math="x' = -y" />
  <BlockMath math="y' = -x" />
</div>

Jadi, peta dari titik <InlineMath math="P(x, y)" /> adalah <InlineMath math="P'(-y, -x)" />. Perhatikan bahwa nilai koordinat x dan y saling bertukar tempat DAN berubah tanda (menjadi negatifnya).

## Mencerminkan Titik

Misalkan kita memiliki titik <InlineMath math="D(2, 1)" />. Jika titik D dicerminkan terhadap garis <InlineMath math="y = -x" />, maka bayangannya, <InlineMath math="D'" />, adalah:

<div className="flex flex-col gap-4">
  <BlockMath math="x' = -(1) = -1" />
  <BlockMath math="y' = -(2) = -2" />
</div>

Jadi, bayangan titik D adalah <InlineMath math="D'(-1, -2)" />.

## Mencerminkan Segitiga

Tentukan peta dari segitiga ABC dengan titik sudut <InlineMath math="A(-2,4)" />, <InlineMath math="B(3,1)" />, dan <InlineMath math="C(-3,-1)" /> yang dicerminkan terhadap garis <InlineMath math="y=-x" />.

Untuk mencerminkan segitiga, kita cerminkan setiap titik sudutnya:

1.  Titik <InlineMath math="A(-2, 4)" />: Bayangannya <InlineMath math="A'(-4, 2)" />.
2.  Titik <InlineMath math="B(3, 1)" />: Bayangannya <InlineMath math="B'(-1, -3)" />.
3.  Titik <InlineMath math="C(-3, -1)" />: Bayangannya <InlineMath math="C'(1, 3)" />.

Segitiga bayangan <InlineMath math="A'B'C'" /> terbentuk dengan menghubungkan titik-titik <InlineMath math="A'(-4, 2)" />, <InlineMath math="B'(-1, -3)" />, dan <InlineMath math="C'(1, 3)" />.

## Mencerminkan Persamaan Garis

Jika sebuah garis memiliki persamaan <InlineMath math="y = -4x - 2" /> dicerminkan terhadap garis <InlineMath math="y=-x" />, tentukan persamaan garis bayangannya.

Untuk mencari persamaan bayangan, kita gunakan aturan <InlineMath math="x' = -y" /> dan <InlineMath math="y' = -x" />. Ini berarti kita mengganti setiap <InlineMath math="x" /> dalam persamaan asli dengan <InlineMath math="-y" /> dan setiap <InlineMath math="y" /> dengan <InlineMath math="-x" />.

Persamaan asli:

<BlockMath math="y = -4x - 2" />

Substitusikan <InlineMath math="x \rightarrow -y" /> dan <InlineMath math="y \rightarrow -x" />:

<BlockMath math="(-x) = -4(-y) - 2" />

Sederhanakan persamaan untuk garis bayangan:

<div className="flex flex-col gap-4">
  <BlockMath math="-x = 4y - 2" />
  <BlockMath math="-x + 2 = 4y" />
  <BlockMath math="y = -\frac{1}{4}x + \frac{2}{4}" />
  <BlockMath math="y = -\frac{1}{4}x + \frac{1}{2}" />
</div>

Jadi, persamaan bayangan dari garis <InlineMath math="y = -4x - 2" /> setelah dicerminkan terhadap <InlineMath math="y=-x" /> adalah <InlineMath math="y = -\frac{1}{4}x + \frac{1}{2}" />.

      Garis <InlineMath math="y=-4x-2" /> dan Bayangannya terhadap{" "}
      <InlineMath math="y=-x" />

  }

        return { x: xVal, y: -4 * xVal - 2, z: 0 };

        return { x: xVal, y: (-1 / 4) * xVal + 1 / 2, z: 0 };

  ]}
  showZAxis={false}
  cameraPosition={[0, 0, 18]}
/>

## Latihan

1.  Tentukan koordinat bayangan dari titik <InlineMath math="S(5, -9)" /> jika dicerminkan terhadap garis <InlineMath math="y = -x" />!
2.  Tentukan peta dari segitiga ABC dengan titik sudut <InlineMath math="A(1, -2)" />, <InlineMath math="B(2, 3)" />, dan <InlineMath math="C(-2, 0)" /> yang dicerminkan terhadap garis <InlineMath math="y = -x" />.
3.  Jika sebuah garis memiliki persamaan <InlineMath math="4x + 3y - 2 = 0" /> dicerminkan terhadap garis <InlineMath math="y = -x" />, tentukan persamaan garis bayangannya.

### Kunci Jawaban

1.  Bayangan titik <InlineMath math="S(5, -9)" /> adalah <InlineMath math="S'(9, -5)" />.

    **Penjelasan:**

    <div className="flex flex-col gap-4">
      <BlockMath math="x' = -(-9) = 9" />
      <BlockMath math="y' = -(5) = -5" />
    </div>

2.  Koordinat bayangan segitiga <InlineMath math="A'B'C'" /> adalah:

    - <InlineMath math="A'(2, -1)" /> (dari <InlineMath math="A(1, -2)" />)
    - <InlineMath math="B'(-3, -2)" /> (dari <InlineMath math="B(2, 3)" />)
    - <InlineMath math="C'(0, 2)" /> (dari <InlineMath math="C(-2, 0)" />)

3.  Persamaan bayangan garis <InlineMath math="4x + 3y - 2 = 0" /> adalah <InlineMath math="4(-y) + 3(-x) - 2 = 0" />.

    **Penjelasan:** Substitusi <InlineMath math="x \rightarrow -y" /> dan <InlineMath math="y \rightarrow -x" /> ke dalam persamaan asli:

    <div className="flex flex-col gap-4">
      <BlockMath math="4(-y) + 3(-x) - 2 = 0" />
      <BlockMath math="-4y - 3x - 2 = 0" />
      <BlockMath math="3x + 4y + 2 = 0" />
    </div>
