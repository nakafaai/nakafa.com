## Penerapan Sifat-sifat Sudut dalam Lingkaran

### Menentukan Besar Sudut

Pada lingkaran dengan pusat O, diketahui sudut pusat AOB = 100°. Tentukan besar sudut keliling ACB!

**Penyelesaian:**

Menggunakan sifat sudut pusat dan sudut keliling:

<div className="flex flex-col gap-4">
  <BlockMath math="\angle ACB = \frac{1}{2} \times \angle AOB" />
  <BlockMath math="\angle ACB = \frac{1}{2} \times 100°" />
  <BlockMath math="\angle ACB = 50°" />
</div>

### Mencari Sudut Berhadapan

Dalam segiempat tali busur ABCD, diketahui <InlineMath math="\angle A = 75°" />. Tentukan besar <InlineMath math="\angle C" />!

**Penyelesaian:**

Menggunakan sifat segiempat tali busur:

<div className="flex flex-col gap-4">
  <BlockMath math="\angle A + \angle C = 180°" />
  <BlockMath math="75° + \angle C = 180°" />
  <BlockMath math="\angle C = 180° - 75°" />
  <BlockMath math="\angle C = 105°" />
</div>

### Menggunakan Teorema Thales

Titik C terletak pada lingkaran dengan AB sebagai diameter. Tentukan besar <InlineMath math="\angle ACB" />!

**Penyelesaian:**

Karena AB adalah diameter dan C terletak pada lingkaran, maka berdasarkan Teorema Thales:

<BlockMath math="\angle ACB = 90°" />

## Latihan Soal

1. Pada lingkaran dengan pusat O, sudut pusat AOB = 140°. Jika C dan D adalah dua titik berbeda pada lingkaran, tentukan:

   - Besar sudut ACB
   - Besar sudut ADB

2. Dalam segiempat tali busur PQRS, diketahui:

   <div className="flex flex-col gap-4">
     <BlockMath math="\angle P = 85°" />
     <BlockMath math="\angle Q = 110°" />
   </div>

   Tentukan besar <InlineMath math="\angle R" /> dan <InlineMath math="\angle S" />!

3. Titik A, B, dan C terletak pada lingkaran. Jika AB adalah diameter dan BC = AC, tentukan besar <InlineMath math="\angle BAC" />!

4. Pada lingkaran, sudut keliling APB = 35°. Tentukan besar sudut pusat AOB!

5. Dalam segiempat tali busur KLMN, sudut luar di titik K adalah 65°. Tentukan besar sudut dalam di titik M!

       {

         color: getColor("AMBER"),

       },
       {
         points: (() => {
           const C = { x: 0, y: 4 };
           const A = { x: -4, y: 0 };
           const B = { x: 4, y: 0 };
           
           // Calculate angle ACB at point C (should be 90°)
           const angleCA = Math.atan2(A.y - C.y, A.x - C.x);
           const angleCB = Math.atan2(B.y - C.y, B.x - C.x);
           
           return Array.from({ length: 16 }, (_, i) => {
             const t = i / 15;
             const angle = angleCA + t * (angleCB - angleCA);
             return {
               x: C.x + 0.8 * Math.cos(angle),
               y: C.y + 0.8 * Math.sin(angle),
               z: 0,
             };
           });
         })(),
         color: getColor("PINK"),
         showPoints: false,
         labels: [{ text: "90°", at: 7, offset: [-0.3, -0.5, 0] }],
       },
     ]}
     cameraPosition={[0, 0, 12]}
     showZAxis={false}
   />

   **Penyelesaian:**

   Karena AB adalah diameter, maka berdasarkan Teorema Thales:

   <BlockMath math="\angle ACB = 90°" />

   Karena BC = AC, maka segitiga ABC adalah segitiga siku-siku sama kaki.

   Dalam segitiga siku-siku sama kaki, kedua sudut alasnya sama besar:

   <div className="flex flex-col gap-4">
     <BlockMath math="\angle BAC + \angle ABC + \angle ACB = 180°" />
     <BlockMath math="\angle BAC + \angle ABC + 90° = 180°" />
     <BlockMath math="\angle BAC + \angle ABC = 90°" />
   </div>

   Karena <InlineMath math="\angle BAC = \angle ABC" /> (sudut alas segitiga sama kaki):

   <div className="flex flex-col gap-4">
     <BlockMath math="2 \times \angle BAC = 90°" />
     <BlockMath math="\angle BAC = 45°" />
   </div>

4. **Menentukan sudut pusat dari sudut keliling**

   **Penyelesaian:**

5. **Menentukan sudut dalam dari sudut luar segiempat tali busur**

   **Penyelesaian:**

   Dalam segiempat tali busur, sudut luar di suatu titik sama dengan sudut dalam di titik yang berseberangan.

   Jika sudut luar di K = 65°, maka:

   <BlockMath math="\angle \text{dalam di M} = \angle \text{luar di K} = 65°" />

   Ini karena titik K dan M berseberangan dalam segiempat tali busur KLMN.
