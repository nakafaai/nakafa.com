## Penerapan Sifat-sifat Sudut dalam Lingkaran

### Menentukan Besar Sudut

Pada lingkaran dengan pusat <InlineMath math="O" />, diketahui sudut pusat <InlineMath math="\angle AOB = 100^\circ" />. Tentukan besar sudut keliling <InlineMath math="\angle ACB" />!

**Penyelesaian:**

Menggunakan sifat sudut pusat dan sudut keliling:

<MathContainer>
  <BlockMath math="\angle ACB = \frac{1}{2} \times \angle AOB" />
  <BlockMath math="\angle ACB = \frac{1}{2} \times 100^\circ" />
  <BlockMath math="\angle ACB = 50^\circ" />
</MathContainer>

### Mencari Sudut Berhadapan

Dalam segiempat tali busur <InlineMath math="ABCD" />, diketahui <InlineMath math="\angle A = 75^\circ" />. Tentukan besar <InlineMath math="\angle C" />!

**Penyelesaian:**

Menggunakan sifat segiempat tali busur:

<MathContainer>
  <BlockMath math="\angle A + \angle C = 180^\circ" />
  <BlockMath math="75^\circ + \angle C = 180^\circ" />
  <BlockMath math="\angle C = 180^\circ - 75^\circ" />
  <BlockMath math="\angle C = 105^\circ" />
</MathContainer>

### Menggunakan Teorema Thales

Titik <InlineMath math="C" /> terletak pada lingkaran dengan <InlineMath math="AB" /> sebagai diameter. Tentukan besar <InlineMath math="\angle ACB" />!

**Penyelesaian:**

Karena <InlineMath math="AB" /> adalah diameter dan <InlineMath math="C" /> terletak pada lingkaran, maka berdasarkan Teorema Thales:

<BlockMath math="\angle ACB = 90^\circ" />

## Latihan Soal

1. Pada lingkaran dengan pusat <InlineMath math="O" />, sudut pusat <InlineMath math="\angle AOB = 140^\circ" />. Jika <InlineMath math="C" /> dan <InlineMath math="D" /> adalah dua titik berbeda pada lingkaran, tentukan:

   - Besar sudut <InlineMath math="\angle ACB" />
   - Besar sudut <InlineMath math="\angle ADB" />

2. Dalam segiempat tali busur <InlineMath math="PQRS" />, diketahui:

   <MathContainer>
     <BlockMath math="\angle P = 85^\circ" />
     <BlockMath math="\angle Q = 110^\circ" />
   </MathContainer>

   Tentukan besar <InlineMath math="\angle R" /> dan <InlineMath math="\angle S" />!

3. Titik <InlineMath math="A" />, <InlineMath math="B" />, dan <InlineMath math="C" /> terletak pada lingkaran. Jika <InlineMath math="AB" /> adalah diameter dan <InlineMath math="BC = AC" />, tentukan besar <InlineMath math="\angle BAC" />!

4. Pada lingkaran, sudut keliling <InlineMath math="\angle APB = 35^\circ" />. Tentukan besar sudut pusat <InlineMath math="\angle AOB" />!

5. Dalam segiempat tali busur <InlineMath math="KLMN" />, sudut luar di titik <InlineMath math="K" /> adalah <InlineMath math="65^\circ" />. Tentukan besar sudut dalam di titik <InlineMath math="M" />!

   <BlockMath math="\angle ADB = \angle ACB = 70^\circ" />

2. **Menentukan sudut-sudut dalam segiempat tali busur**

   **Penyelesaian:**

Dalam segiempat tali busur, <InlineMath math="\text{jumlah sudut berhadapan} = 180^\circ" />

   Untuk sudut <InlineMath math="R" /> (berhadapan dengan <InlineMath math="P" />):

   <MathContainer>
     <BlockMath math="\angle P + \angle R = 180^\circ" />
     <BlockMath math="85^\circ + \angle R = 180^\circ" />
     <BlockMath math="\angle R = 180^\circ - 85^\circ" />
     <BlockMath math="\angle R = 95^\circ" />
   </MathContainer>

   Untuk sudut <InlineMath math="S" /> (berhadapan dengan <InlineMath math="Q" />):

   <MathContainer>
     <BlockMath math="\angle Q + \angle S = 180^\circ" />
     <BlockMath math="110^\circ + \angle S = 180^\circ" />
     <BlockMath math="\angle S = 180^\circ - 110^\circ" />
     <BlockMath math="\angle S = 70^\circ" />
   </MathContainer>

5. **Menentukan sudut dalam dari sudut luar segiempat tali busur**

   **Penyelesaian:**
