## Sifat Dasar Logaritma

Seperti halnya eksponen, logaritma juga memiliki beberapa sifat penting yang perlu dipahami. Sifat-sifat ini akan sangat membantu dalam menyelesaikan berbagai persoalan logaritma.

Misalkan <InlineMath math="a > 0" /> dan <InlineMath math="a \neq 1" />, <InlineMath math="b > 0" />, <InlineMath math="c > 0" />, <InlineMath math="m > 0" />, <InlineMath math="m \neq 1" />, di mana <InlineMath math="a, b, c, m, n" /> adalah bilangan real <InlineMath math="(a, b, c, m, n \in \mathbb{R})" />. Berikut adalah **sifat-sifat logaritma**:

1. <InlineMath math="^a\log a = 1" />
2. <InlineMath math="^a\log 1 = 0" />
3. <InlineMath math="^a\log a^n = n" />
4. <InlineMath math="^a\log (b \times c) = ^a\log b + ^a\log c" />
5. <InlineMath math="^a\log \left(\frac{b}{c}\right) = ^a\log b - ^a\log c" />
6. <InlineMath math="^a\log b^n = n \cdot ^a\log b" />
7. <InlineMath math="^a\log b = \frac{^m\log b}{^m\log a} = \frac{1}{^b\log a}" />
8. <InlineMath math="^a\log b \times ^b\log c = ^a\log c" />

## Pembuktian Sifat Logaritma

### Logaritma Perkalian

**Sifat 4**: <InlineMath math="^a\log (b \times c) = ^a\log b + ^a\log c" />

Bukti: Misalkan <InlineMath math="^a\log b = m" /> dan <InlineMath math="^a\log c = n" />

Ini berarti:

<InlineMath math="b = a^m" /> dan <InlineMath math="c = a^n" />

Menggunakan sifat eksponen:

<BlockMath math="b \times c = a^m \times a^n = a^{m+n}" />

Dengan demikian:

<BlockMath math="^a\log (b \times c) = ^a\log (a^{m+n}) = m + n = ^a\log b + ^a\log c" />

### Logaritma Pembagian

**Sifat 5**: <InlineMath math="^a\log \left(\frac{b}{c}\right) = ^a\log b - ^a\log c" />

Bukti: Misalkan <InlineMath math="^a\log b = m" /> dan <InlineMath math="^a\log c = n" />

Maka <InlineMath math="b = a^m" /> dan <InlineMath math="c = a^n" />

Ingat bahwa <InlineMath math="\frac{a^m}{a^n} = a^{m-n}" />, sehingga:

<BlockMath math="\frac{b}{c} = \frac{a^m}{a^n} = a^{m-n}" />

Oleh karena itu:

<BlockMath math="^a\log \left(\frac{b}{c}\right) = ^a\log (a^{m-n}) = m - n = ^a\log b - ^a\log c" />

### Logaritma Pangkat

**Sifat 6**: <InlineMath math="^a\log b^n = n \cdot ^a\log b" />

Bukti: Misalkan <InlineMath math="^a\log b = m" />

<InlineMath math="^a\log b^n" /> artinya logaritma dari <InlineMath math="b" /> pangkat <InlineMath math="n" />

<InlineMath math="^a\log b^n = ^a\log (\underbrace{b \times b \times b \times \ldots \times b}_{n \text{ faktor}})" />

Menggunakan sifat 4 berulang kali:

<BlockMath math="^a\log b^n = \underbrace{^a\log b + ^a\log b + ^a\log b + \ldots + ^a\log b}_{n \text{ faktor}} = n \cdot ^a\log b" />

### Perubahan Basis Logaritma

**Sifat 7**: <InlineMath math="^a\log b = \frac{^m\log b}{^m\log a} = \frac{1}{^b\log a}" />

Bukti:
Berdasarkan definisi logaritma, <InlineMath math="^a\log b = c" /> jika dan hanya jika <InlineMath math="b = a^c" />

Misalkan kita menggunakan basis <InlineMath math="m" /> untuk logaritma <InlineMath math="b" />:

<BlockMath math="^m\log b = ^m\log a^c" />

Menggunakan sifat 6:

<BlockMath math="^m\log b = c \cdot ^m\log a" />

Karena <InlineMath math="c = ^a\log b" />, maka:

<BlockMath math="^m\log b = ^a\log b \cdot ^m\log a" />

Sehingga:

<BlockMath math="^a\log b = \frac{^m\log b}{^m\log a}" />

Jika <InlineMath math="m = b" />, maka:

<BlockMath math="^a\log b = \frac{^b\log b}{^b\log a} = \frac{1}{^b\log a}" />

### Logaritma Berantai

**Sifat 8**: <InlineMath math="^a\log b \times ^b\log c = ^a\log c" />

Bukti:
Berdasarkan definisi:

<div className="flex flex-col gap-4">
  <BlockMath math="^a\log b = m \Leftrightarrow b = a^m" />
  <BlockMath math="^b\log c = n \Leftrightarrow c = b^n" />
</div>

Substitusi nilai <InlineMath math="b" /> ke persamaan <InlineMath math="c" />:

<BlockMath math="c = b^n = (a^m)^n = a^{mn}" />

Karena <InlineMath math="c = a^{mn}" />, maka:

<BlockMath math="^a\log c = ^a\log (a^{mn}) = mn = ^a\log b \times ^b\log c" />

## Contoh Penerapan

Misalkan kita ingin menghitung <InlineMath math="^5\log 125" />.

Menggunakan **sifat 6**:

<BlockMath math="^5\log 125 = ^5\log 5^3 = 3 \cdot ^5\log 5 = 3 \cdot 1 = 3" />

## Latihan

1. Sederhanakan bentuk akar berikut ini.

   a. <InlineMath math="^9\log 81" />

   b. <InlineMath math="^2\log 64 - ^2\log 16" />

   c. <InlineMath math="^4\log 16^{10}" />

2. Jika <InlineMath math="^5\log 4 = m" />, <InlineMath math="^4\log 3 = n" />, nyatakan <InlineMath math="^{12}\log 100" /> dalam m dan n.

3. Penduduk kota A pada tahun 2010 sebanyak 300.000 jiwa. Pertumbuhan penduduk kota A rata-rata per tahun adalah 6%. Jika diasumsikan pertumbuhan penduduk setiap tahun sama, dalam berapa tahun penduduk kota A menjadi 1 juta jiwa?

4. Berapa waktu yang dibutuhkan sehingga uang Dini yang tadinya Rp2.000.000,00 dapat menjadi Rp6.500.000,00 jika dia menabung di suatu bank yang memberinya bunga sebesar 12%?

### Kunci Jawaban

1. Menentukan nilai logaritma

   a. Jawaban:

   <div className="flex flex-col gap-4">
     <BlockMath math="^9\log 81 = ^9\log 9^2" />
     <BlockMath math="^9\log 81 = 2 \cdot ^9\log 9" />
     <BlockMath math="^9\log 81 = 2 \cdot 1 = 2" />
   </div>

   b. Jawaban:

   <div className="flex flex-col gap-4">
     <BlockMath math="^2\log 64 - ^2\log 16 = ^2\log \frac{64}{16}" />
     <BlockMath math="^2\log 64 - ^2\log 16 = ^2\log 4" />
     <BlockMath math="^2\log 64 - ^2\log 16 = 2" />
   </div>

   c. Jawaban:

   <div className="flex flex-col gap-4">
     <BlockMath math="^4\log 16^{10} = ^4\log (4^2)^{10}" />
     <BlockMath math="^4\log 16^{10} = ^4\log 4^{20}" />
     <BlockMath math="^4\log 16^{10} = 20" />
   </div>

2. Dinyatakan bahwa <InlineMath math="^5\log 4 = m" />, <InlineMath math="^4\log 3 = n" />

   Maka:

   <div className="flex flex-col gap-4">
     <BlockMath math="^{12}\log 100 = \frac{^4\log 100}{^4\log 12}" />
     <BlockMath math="= \frac{^4\log(4 \times 25)}{^4\log(4 \times 3)}" />
     <BlockMath math="= \frac{^4\log 4 + ^4\log 25}{^4\log 4 + ^4\log 3}" />
     <BlockMath math="= \frac{^4\log 4 + 2 \cdot ^4\log 5}{^4\log 4 + ^4\log 3}" />
     <BlockMath math="= \frac{1 + 2 \cdot \frac{1}{m}}{1 + n}" />
     <BlockMath math="= \frac{1 + \frac{2}{m}}{1 + n}" />
   </div>

3. Jumlah penduduk = 300.000 jiwa

   Pertumbuhan penduduk per tahun 6%.

   Fungsi yang tepat untuk menggambarkan pertumbuhan penduduk dalam <InlineMath math="x" /> tahun adalah:

   <BlockMath math="f(x) = 300.000(1 + 0,06)^x" />

   Untuk jumlah penduduk 1.000.000 jiwa:

   <div className="flex flex-col gap-4">
     <BlockMath math="1.000.000 = 300.000(1 + 0,06)^x" />
     <BlockMath math="1.000.000 = 300.000(1,06)^x" />
     <BlockMath math="\frac{1.000.000}{300.000} = (1,06)^x" />
     <BlockMath math="3,33 = (1,06)^x" />
     <BlockMath math="x = ^{1,06}\log 3,33" />
     <BlockMath math="x = 20,645" />
   </div>

   Jadi penduduk akan mencapai 1.000.000 jiwa dalam waktu 20 atau 21 tahun.

4. Tabungan awal = Rp2.000.000,00

   Tabungan akhir = Rp6.500.000,00

   Bunga = 12%

   Fungsi yang tepat untuk menggambarkan tabungan Dini dalam <InlineMath math="x" /> tahun adalah:

   <BlockMath math="f(x) = 2.000.000(1 + 0,12)^x" />

   Untuk tabungan akhir sebesar Rp6.500.000,00:

   <div className="flex flex-col gap-4">
     <BlockMath math="6.500.000 = 2.000.000(1 + 0,12)^x" />
     <BlockMath math="6.500.000 = 2.000.000(1,12)^x" />
     <BlockMath math="\frac{6.500.000}{2.000.000} = (1,12)^x" />
     <BlockMath math="3,25 = (1,12)^x" />
     <BlockMath math="x = ^{1,12}\log 3,25" />
     <BlockMath math="x = 10,4" />
   </div>

   Jadi, tabungan Dini akan mencapai Rp6.500.000,00 dalam waktu 10 tahun.
