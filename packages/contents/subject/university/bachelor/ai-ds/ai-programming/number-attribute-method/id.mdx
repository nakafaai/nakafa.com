## Konsep Dasar Bilangan sebagai Objek

Dalam Python, setiap bilangan adalah sebuah objek yang memiliki atribut dan metode tersendiri. Konsep ini mungkin terdengar asing pada awalnya, tetapi sebenarnya sangat sederhana. Bayangkan bilangan seperti sebuah kotak yang tidak hanya berisi nilai, tetapi juga memiliki kemampuan khusus yang bisa kamu gunakan.

Ketika kamu menulis <InlineMath math="5" /> atau <InlineMath math="3.14" />, Python tidak hanya menyimpan angka tersebut, tetapi juga menyediakan berbagai fungsi yang bisa kamu panggil untuk memanipulasi atau mendapatkan informasi dari bilangan tersebut.

## Mengakses Atribut dan Metode

Untuk mengakses atribut dari sebuah objek bilangan, kamu menggunakan sintaks `objek.atribut`. Sedangkan untuk memanggil metode, kamu menggunakan sintaks `objek.metode()`.

Mari kita lihat contoh praktis dengan bilangan kompleks:

<CodeBlock
  data={[
    {
      language: "python",
      filename: "complex_example.py",
      code: `# Membuat bilangan kompleks
c = 2 + 3j

# Mengakses atribut real (bagian riil)
print(c.real)  # Output: 2.0

# Mengakses atribut imag (bagian imajiner)
print(c.imag)  # Output: 3.0

# Memanggil metode conjugate() untuk mendapatkan konjugat
print(c.conjugate())  # Output: (2-3j)`

## Metode Bilangan Kompleks

### Metode conjugate()

Metode `conjugate()` mengembalikan konjugat kompleks dari bilangan. Konjugat kompleks adalah bilangan dengan bagian riil yang sama tetapi bagian imajiner yang berlawanan tanda.

<CodeBlock
  data={[
    {
      language: "python",
      filename: "conjugate_method.py",
      code: `# Contoh penggunaan metode conjugate()
z1 = 3 + 4j
z2 = -2 - 5j
z3 = 7 + 0j

print(f"Konjugat dari {z1} adalah {z1.conjugate()}")
print(f"Konjugat dari {z2} adalah {z2.conjugate()}")
print(f"Konjugat dari {z3} adalah {z3.conjugate()}")

# Output:
# Konjugat dari (3+4j) adalah (3-4j)
# Konjugat dari (-2-5j) adalah (-2+5j)
# Konjugat dari (7+0j) adalah (7-0j)`
    }
  ]}
/>

### Penggunaan Praktis Konjugat
