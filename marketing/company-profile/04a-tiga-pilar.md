## Pilar 2: Tutor AI (Nina)

**Apa itu Nina?**
Asisten AI yang bisa:
- Menjelaskan konsep dengan bahasa sederhana
- Menjawab pertanyaan spesifik
- Membantu PR (bukan mengerjakan untuk siswa)
- Beradaptasi dengan gaya belajar masing-masing

**Cara Kerja:**
1. Siswa foto soal
2. Tanya ke Nina: "Cara ngerjain ini gimana?"
3. Dapat penjelasan langkah demi langkah
4. Belajar sampai paham

**Teknologi:**
- 15+ AI models (GPT, Claude, Gemini, Kimi, Llama)
- RAG (Retrieval-Augmented Generation)
- Diakses via nakafa.com
- Streaming real-time

## Pilar 3: Open Infrastructure

**Format Terbuka:**
- Materi dalam format Markdown (tambahkan `.md` di URL)
- Developer bisa bangun aplikasi di atas konten kami
- Sekolah bisa integrasi ke LMS mereka

**API Tersedia:**
- `llms.txt` - Untuk training model AI
- REST API - Integrasi aplikasi pihak ketiga
- Webhook - Notifikasi dan update

**Open Source:**
- Kode di GitHub (github.com/nakafaai/nakafa.com)
- Komunitas bisa kontribusi materi
- Sistem review untuk jaga kualitas
